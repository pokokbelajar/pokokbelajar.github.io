4. bagaimana hubungan antara panjang salah satu sisi dengan jumlah panjang seluruh sisi persegi ?


Kelas : IV (4 SD)
Kategori Soal : Bangun Datar
Kata Kunci : persegi, keliling, luas

Pembahasan :
Persegi adalah bangun segi empat memiliki empat buah sisi sama panjang serta empat buah sudut sama besar dan siku-siku.

Perhatikan gambar terlampir.

Sifat-sifat persegi, yaitu :

1. sisi-sisi sama panjang
AB = BC = CD = DA;
2. sudut-sudut sama besar dan siku-siku 

∠DAB = ∠ABC= ∠BCD = ∠CDA = 90°;

3. memiliki empat buah simetri lipat atau sumbu simetri;
4. memiliki empat buah simetri putar;
5. menempati bingkainya dengan delapan cara.

 

Keliling persegi adalah

K = 4 x s,

dengan s merupakan panjang sisi.


Luas persegi adalah

L = s x s

⇔ L = s²

dengan s merupakan panjang sisi.


Mari kita lihat soal tersebut.

Bagaimana hubungan antara panjang salah satu sisi dengan jumlah panjang seluruh sisi persegi?

Jawab:
Diketahui panjang sisi persegi adalah s. Jumlah panjang seluruh sisi atau keliling persegi adalah
K = s + s + s + s
⇔ K = 4s.
Hubungan antara panjang salah satu sisi persegi dengan jumlah panjang seluruh sisi adalah empat kalinya.

13. Jika jumlah kuadrat semua panjangsisi persegi panjang adalah 100, makapanjang diagonal dari persegipanjang tersebut adalah ...​


Jawab:

Panjang diagonal = 5

Penjelasan dengan langkah-langkah:

Panjang diagonal = \sqrt{p^{2} + l^{2}  }

(2p)^{2} + (2l)^{2} = 100

4p^{2} + 4l^{2}  = 100

4 (p^{2} + l^{2})=100

p^{2} + l^{2} =\frac{100}{4}

p^{2} + l^{2} =25

Karena Panjang diagonal = \sqrt{p^{2} + l^{2}  }

maka, \sqrt{p^{2} + l^{2}  } = \sqrt{25} = 5

Jadi, Panjang diagonal persegi panjang = 5

18. jumlah kuadrat dari panjang semua sisi persegi adalah 72. Berapakah panjang diagonal persegi itu?​


Jumlah kuadrat artinya suatu bilangan itu harus dikuadratkan dahulu baru dijumlah (Membaca dari Kanan).

Sisi Persegi = 4.

s^2 + s^2 + s^2 + s^2 = 72.

4s^2 = 72.

s^2 = 72 ÷ 4 = 18.

s = akar dari 18 atau 3 akar 2.

Panjang Diagonal = s akar 2.

Panjang Diagonal = 3 akar 2 x akar 2.

Panjang Diagonal = 3x2 = 6 cm.

Jawab:

6

Penjelasan dengan langkah-langkah:

misal sisi persegi = s

jumlah kuadrat dari panjang semua sisinya = 72

s²+s²+s²+s² = 72

4s² = 72

s² = 18

s = √18

s  3√2

panjang diagonal persegi

d² =s²+s²

d² =2s²

d = √2s²

d = s√2

d = 3√2.√2

d = 3.2

d = 6

20. Jika jumlah kuadrat semua panjang sisi persegi panjang adalah 100, maka panjang diagonal dari persegi panjang tersebut adalah


Jawab:

Panjang diagonal = 5

Penjelasan dengan langkah-langkah:

Panjang diagonal = [tex]\sqrt{p^{2} + l^{2} }[/tex]

[tex](2p)^{2} + (2l)^{2} = 100[/tex]

[tex]4p^{2} + 4l^{2} = 100[/tex]

[tex]4 (p^{2} + l^{2})=100[/tex]

[tex]p^{2} + l^{2} =\frac{100}{4}[/tex]

[tex]p^{2} + l^{2} =25[/tex]

Karena Panjang diagonal = [tex]\sqrt{p^{2} + l^{2} }[/tex]

maka, [tex]\sqrt{p^{2} + l^{2} } = \sqrt{25} = 5[/tex]

Jadi, Panjang diagonal persegi panjang = 5
