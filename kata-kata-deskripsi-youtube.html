Permutasi adalah penyusunan kembali suatu kumpulan objek dalam urutan yang berbeda dari urutan yang semula. Sebagai contoh, kata-kata dalam kalimat sebelumnya dapat disusun kembali sebagai permutasi. Permutasi suatu urutan yang berbeda urutan yang kumpulan semula objek penyusunan kembali dalam dari. yang lebih jelasnya cara menyusun unsur-unsur yang berbeda.

[tex]{\bold {Rumusnya}}[/tex]

n! ( Tanpa Unsur Ganda )

[tex]\frac {n!}{k!}[/tex] ( Dengan Unsur Ganda )

Faktorial adalah perkalian semua bilangan asli yang kurang dari atau sama dengan n. Faktorial dilambangkan dengan tanda !. Jadi n! dibaca sebagai "n faktorial".

n! = 1 x 2 x 3 x ... x n

[tex]{\bold {Contohnya}}[/tex]

2! = 1 × 2

3! = 1 x 2 x 3

4! = 1 x 2 x 3 x 4

5! = 1 x 2 x 3 x 4 x 5

Berikut adalah contoh faktorial yang tidak melibatkan operasi hitung campuran, jadi jika ingin menjumlahkannya kita tinggal menghitungnya seperti bilangan biasa, bilangan faktorial yang hanya perkalian beda dengan bilangan faktrorial yang dijumlahkan dengan pertambahan, penguranga, perkalian, pembagian, sebagai contohnya lihatlah di bawah ini.

6! x 2! - 3! : 4!

Contoh di atas merupakan operasi hitung campuran yang mengandung perkalian, pengurangan, dan pembagian. Ingat bahwa pada operasi hitung campuran perkalian/pembagian harus diselesaikan terlebih dahulu daripada penjumlahan/pengurangan.

Kombinasi adalah susunan berurutan dari elemen suatu himpunan tanpa memperhatikan pola urutan.

[tex]{\bold {Rumusnya}}[/tex]

[tex]\frac {n!}{r! ( k! - r! )}[/tex]

24. Apa arti "Upotsu" di bahasa jepang? Saya sering melihatnya di comment2 youtube, adakah yang bisa menjelaskan arti dari kata tersebut dalam bahasa indonesia ataupun bahasa jepang?​


うぽつは「うｐ乙」の略語です。「うｐ」は動画などのアップロードを、「乙」はお疲れ様を意味する。動画をアップロードしてくれてありがとうというような意味です。うぽつの言葉はネット（ニコニコやYouTubeなど）でよく使います。動画のアップロード者に対して労いと感謝の気持ちを伝える際に使われてます。

Upotsu adalah singkatan dari "Up Otsu". "Up" berarti upload video dan sebagainya, dan "Otsu" berarti Otsukaresama (Terima kasih atas kerja kerasnya). Artinya seperti terima kasih karena sudah mengupload video nya. Kata Upotsu ini sering digunakan di internet (Nikoniko, YouTube, dll). Ini biasa digunakan ketika menyampaikan penghargaan dan rasa terima kasih kepada sang pengupload video.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

日本語のうぽつという言葉は、最近ネットでよく使われてるスラングです。この言葉は「動画をアップロードしてくれてありがとう」というような意味です。このうぽつという言葉に加えて、よく使われ、同じような意味を持つ他の言葉もあります。

Kata Upotsu dalam Bahasa Jepang merupakan kata slang yang baru-baru ini sering digunakan di internet. Arti dari kata ini seperti "Terima kasih sudah mengupload video ini". Selain kata Upotsu ini, ada juga kata-kata lainnya yang biasa digunakan dan memiliki arti yang mirip.

- - - - - - - - - -

1. うぱり (Upari)

「うｐあり（アップロードありがとう）」の略語です。動画サイトなどに動画がアップロードされた時の返信コメントとして、動画のアップロード者に対して感謝の気持ちを伝える際によく使われてます。

Merupakan singkatan dari "Up Ari (Upload Arigatou)". Biasa sering digunakan ketika menyampaikan rasa terima kasih kepada sang pengupload video, sebagai komentar balasan saat sebuah video telah diupload pada situs-situs video.

- - -

2. おまどうま (Omadouma)

「お前の動画を待ってたんだよ」の略語です。お前の動画を待ってたんだよ、とコメントしたくなるほど、その人の投稿した動画に夢中になりましたと言う感謝の言葉が「おまどうま」です。

Merupakan singkatan dari "Omae no douga wo matteta nda yo" (Gw udah lama nungguin video lu nih, gitu artinya). "Omadouma" ini adalah semacam ungkapan terima kasih yang seperti ingin mengatakan "Gw udah lama nungguin video lu nih", dan sudah jadi tergila-gila dengan video yang diupload orang tersebut.
