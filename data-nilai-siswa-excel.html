Dalam Microsoft Excel sudah tersedia berbagai fungsi - fungsi yang akan mempermudah proses pengolahan data atau dalam menyelesaikan sebuah pekerjaan. Secara umum penulisan formula dalam Microsoft Excel bisa dilakukan langsung pada sebuah cell diawali dengan mengetikan simbol "=" (sama dengan) baru kemudian diikuti dengan nama fungsi atau bisa menggunakan fungsi Insert Function dengan memilih icon [tex]fx[/tex] yang berada di sebelah formula bar.

Beberapa contoh fungsi - fungsi yang umum digunakan pada excel antara lain :

=SUM(range), digunakan untuk mendapatkan jumlah angka yang dipilih=AVERAGE(range), digunakan untuk mendapatkan nilai rata-rata dari kumpulan angka yang dipilih=MAX(range), digunakan untuk mencari nilai terbesar dari kumpulan angka=MIN(range), digunakan untuk mencari nilai terendah dari kumpalan angka=STD(range), digunakan untuk mendapatkan nilai standar deviasi atau simpangan baku dari kumpulan data=SQRT(angka), digunakan untuk menghitung akar kuadrat sebuah angka=ROUND(angka,n), digunakan untuk membulatkan angka desimal hingga n angka dibelakang

20. Fungsi dalam Excel yang digunakan menghitung banyak data dari sekelompok nilai …


Pembahasan :

pengertian M.excel

M.excel adalah sebuah program aplikasi kerja(spedsheet) yang berfungsi sebagai pengolah angka

operator matematik

^ : pemangkatan

* : perkalian

/ : pembagian

+ : penjumlahan

- : pengurangan

Rumus excel beserta fungsi :

Sum ==> mencari penjumlahan atau menjumlahkan.

Max ==> mencari nilai terbesar.

Min ==> mencari nilai terkecil.

Average ==> mencari nilai rata rata suatu variebel.

And ==> menghasilakan nilai true jika semua argumen yang diuji bernilai benar dan false jika semua atau salah satu argumen bernilai salah.

Not ==> kebalikan dari And yaitu menghadilkan nilai true jika kondisi yang diuji salah atau false jika yang diuji benar.

Or ==> menghasilkan nilai true jika argumen benar dan false jika semua argumen salah.

If ==> mengembalikan sebuah nilai jika kondisi yang di cek benar dan mengembalikan sebuah nilai lainnya jika kondisi yang di cek salah.

25. jenis data di excel ada berapa?


Terdapat 4 jenis data di Microsoft Excel
1) Data label ( huruf )
2) Data angka ( value )
3) Data waktu ( time )
4) Data tanggal ( date )


Semoga BermanfaatJenis Data Pada Excel
Ada empat jenis data yang dapat dimasukkan ke dalam sel pada program pengolah angka Microsoft Excel yaitu data teks (label), angka (value),waktu (time),dan tanggal (date).

1. Data Teks (Label)
Data ini terdiri atas kumpulan huruf. Data. teks
dapat dimasukkan bilangan, namun harus
didahului tanda petik ( ‘ ). Misalnya ’45. Berikut
adalah karakter-karakter yang dapat dimasukkan
ke dalam sel Excel.
a. Huruf : A, a, B, b, ….
b. Angka : 1, 2, 3, 4, ….
c. Simbol : !, @, #, $, ….
d. Spasi

2. Data Angka (Value)
Data angka yang dapat dimasukkan ke dalam sel
pada Microsoft Excel adalah sebagai berikut.
a. Bilangan bulat : 1001
b. Bilangan pecahan desimal : 0,35
c. Bilangan pecahan bulat : 18
d. Bilangan dengan notasi ilmiah: 10.2E +3 artinya: 0,2 × 103.
e. Bilangan negatif yang di awali tanda minus (-), misalnya – 12.
Data waktu (time)
Contoh data waktu adalah 07:30, 15 am

3. Data Tanggal (Date)
Format data tanggal yang dapat dimasukkan ke
dalam sel adalah sebagai berikut.
a. Feb 16, 2009
b. 2 / 16 / 2009
c. 16 – Feb – 2009
d. 16 – 02 – 2009
