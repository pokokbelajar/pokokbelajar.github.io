19. latihan komposisi fungsi​


Jawab:
(fog)(x) = 4x²-22x+27

Penjelasan dengan langkah-langkah:

Diketahui:
f(x) = x²-5x+3
g(x) = 2x-3
Dicari (fog)(x)
f(g(x)) = x²-5x+3
f(g(x)) = (2x-3)²-5(2x-3)+3
(fog)(x) = (2x-3)(2x-3) -10x+15+3
(fog)(x) = 4x²-6x-6x+9-10x+18
(fog)(x) = 4x²-6x-6x-10x+9+18
(fog)(x) = 4x²-22x+27

<(7o7)>

Penjelasan dengan langkah-langkah:

f(x) = x² - 5x + 3

g(x) = 2x - 3

( f o g )(x) = f(g(x))

= (2x - 3)² - 5(2x - 3) + 3

= 4x² - 12x + 9 - 10x + 15 + 3

= 4x² - 22x + 24 + 3

= 4x² - 22x + 27
