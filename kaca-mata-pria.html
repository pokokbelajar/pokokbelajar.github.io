6. seorang pria yang menggunakan lensa dengan kekuatan 3 dioptri harus memegang surat kabar paling dekat 25 cm didepan matanya agar dapat membaca dengan jelas. jika pria itu melepas kacamata dan tetap ingin membaca surat kabar dengan jelas, berapa jauh surat kabar itu paling dekat kematanya?


diketahui :
P = 3 Dioptri
s = 25 cm = 0,25 m

ditanyakan :
titik dekat penderita (s')

pembahasan :
P = 1/f = 3
sedangkan
1/f = 1/s + 1/s'
3 = 1/0,25 + 1/s'
1/s' = 3 - 1/0,25
1/s' = -1
s' = -1 meter
s' = -100 cm

jadi titik terdekat pria tsb. adalah 100 cm

Diketahui :
P = 3 Dioptri
S' = 25 cm
Ditanyakan : f ?

Dijawab :
P = 100/f - 100/S'
3 = 100/f - 100/25
3 = 100/f - 4
100/f = 3 + 4
100/f = 7
f        = 100/7
f        = 14,28 cm

jadi jarak paling dekat = 14,28 cm

mata sipit = resesif = l

maka dapat dibuat diagram persilangannya sebagai berikut:

P1 = keriting, lebar  x lurus, sipit
             KKLL              x    kkll
gamet=  KL                        kl
F1      = KkLl = keriting, lebar

P2       =  F1  x lurus, lebar
             KkLl  x  kkLL
gamet= KL      kL
               Kl
             kL
           kl
F2=
KkLL=keriting lebar
KkLl = keriting lebar
kkLL = lurus lebar
kkLl = lurus lebar

jika mempunyai F2 sejumlah 8 anak, maka anak yang lurus dan sipit = tidak ada = 0%

wanita berambut lurus bermata sipit homozigot = kkll

pria berambut lurus bermata lebar homozigot = kkLL

KKLL >< kkll

KL >< kl

KkLl (ketiga anak berambut keriting bermata lebar heterozigot)

KkLl >< kkLL

KL Kl kL kl >< kL

KkLL KkLl kkLL kkLl = 50% berambut keriting bermata lebar (K-L-) dan 50% berambut lurus bermata lebar (kkL-)

8 anak = 4 berambut keriting bermata lebar dan 4 berambut lurus bermata lebar
