Gunakan program yang berbeda

Format file tunggal bisa dibuka oleh banyak program, baik built-in maupun pihak ketiga. Selain itu, setiap program memiliki algoritma dan persyaratan untuk membuka file. 13. perbedaan cara menyimpan file dengan save dan save astolong dijawab secepatnya


Save berfungsi untuk menyimpan dokumen yang telah kamu buat dan belum pernah disimpan sebelumnya. Save juga dapat digunakan untuk menyimpan file yang sudah pernah tersimpan dengan menggunakan nama file yang sama. Misal, kamu ingin melanjutkan ketika yang pernah dibuat sebelumnya. Kamu cukup membuka dokumen tersebut, lalu menambahkan ketikanmu. Jika sudah selesai, pilih Save untuk menyimpannya kembali.

Save As berguna untuk menyimpan file dokumen pertama kali. Save As biasanya digunakan untuk menyimpan file cadangan. Cadangan atau back up ini berfungsi untuk mengamankan file. Dengan Save As, kamu dapat menyimpan file yang telah tersimpan sebelumnya dengan menggunakan nama file yang berbedasave: menyimpan dokumen yg telah di edit atau di revisi bisa juga untuk menyimpan dokumen baru

save as: menyimpan dokumen yang sama tapi dengan nama file yang berbeda atau dengan format yang berbeda, dengan save juga dapat memberi beberapa options seperti password
