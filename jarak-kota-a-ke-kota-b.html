8. Jarak kota A ke kota B adalah 2 kali jarak kota C ke kota D. Kota E ditengah-tengah kota C dan D. Jarak kota C ke kota A, sama dengan jarak kota D ke kota B, yaitu setengah jarak kota A ke kota E. Jarak kota A ke C dibilangan A Ke B adalah


Jawaban:

A ke B  = 2 x ( C ke D)

E ditengah C dan D

AC = DB

sketsa

A           C            E           D           B

Misalkan

A ke C adalah 1

maka DB = 1

CD = 2

AB = 1 + 2 + 1 = 4

Jadi jarak paling jauh adalah A ke B

Jawaban:

AC = 1/4 AB

Penjelasan:

AB = 2CD

CE = ED

AC = DB = 1/2 AE ⇒ AE = 2AC

CE = AE - AC = 2AC - AC = AC

AB = AC + CB = AC + (CD + DB) = AC + ((CE + ED) + DB)

AB = AC + ((AC + AC) + AC)

AB = 4AC

AC = 1/4 AB
