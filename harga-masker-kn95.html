Jawaban:

Rp 150.000

Penjelasan dengan langkah-langkah:

x = dus masker medis

y = dus masker KN95

5x + 3y = 570.000

3x + 2y = 360.000

1x + 1y =?

cara eliminasi

5x + 3y = 570.000 | dikali 2

3x + 2y = 360.000 | dikali 3

jadi

10x + 6y = 1.140.000

9x + 6y = 1.080.000

——————————- —

1x + 0y = 60.000

1x = 60.000

lanjut di substitusi

3x + 2y = 360.000

(3 X 60.000) + 2y = 360.000

180.000 + 2y = 360.000

2y = 360.000 - 180.000

2y = 180.000

1y = 90.000

kembali ke soal

1 masker medis + 1 KN95

→ 1x + 1y = 60.000 + 90.000

1x + 1y = 150.000

jadi harga yang harus dibayar adalah Rp 150.000

12. harga masker adalah RP 20.000,00 per box,karena pademi maka ada kenaikan sebesar 14%,maka harga masker tersebut menjadi


Jawab:

Harga masker setelah kenaikan sebesar Rp. 22.800

Penjelasan dengan langkah-langkah:

Materi : Matematika - Harga dan Diskon

Ada dua cara penyelesaian :

1. (100% +  kenaikan%) x harga semula, jika penurunan (100% - penurunan%) x harga semula

2. Harga semula + (kenaikan% x harga semula)

Jawaban :

Cara 1

(100% + 14%) x 20.000 = 114% x 20.000 = 22.800

Cara 2

20.000 + (20.000 x 14%) = 20.000 + 2.800 = 22.800

Sehingga, harga masker setelah kenaikan sebesar Rp. 22.800

Rujukan : Matematika, SMP Kelas 8 KTSP 2006 (2008), Kemendikbud.

2. Paman membeli sebuah baju dengan harga Rp50.000,00 dan mendapat diskon 40 persen. Paman juga membeli baju untuk Meni seharga Rp30.000,00 dan mendapatkan diskon 15 persen. Berapa rupiah yang harus dibayar?

Jawab

Besar diskon untuk baju seharga Rp50.000,00

= 40% dari Rp50.000,00

=  × Rp50.000,00

= 40 × Rp500,00

= Rp20.000,00

Besar diskon untuk baju seharga Rp30.000,00

= 15% dari Rp30.000,00

=  × Rp30.000,00

= 15 × Rp300,00

= Rp4.500,00

Jadi jumlah uang yang harus dibayar adalah

= (Rp50.000,00 – Rp20.000,00) + (Rp30.000,00 – Rp4.500,00)

= Rp30.000,00 + Rp25.500,00

= Rp55.500,00
