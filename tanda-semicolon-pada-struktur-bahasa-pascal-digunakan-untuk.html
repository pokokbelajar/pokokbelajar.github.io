17. Sebutkan struktur program pascal?


Struktur program pascal meliputi tiga bagian besar antara lain:

Headings / Judul ProgramDeklarasi dan pernyataanInti program

Pembahasan

Heading / Judul program

Merupakan bagian yang diisikan dengan nama program dimana penulisannya tepat sebelum Begin namun, bagian judul bukanlah bagian yang wajib diisikan. Penulisan judul tetap harus diakhiri dengan ;

Penulisan judul yang tepat:

Program nama_program

Deklarasi dan pernyataan

Bagian deklarasi merupakan bagian yang digunakan untuk mengenalkan semua variabel yang terlibat dalam algoritma pada inti program termasuk tipe datanya.  Hal-hal yang dideklarasikan dalam bagian ini adalah tipe, konstanta, variabel, label, fungsi dan prosedur. Sedangkan, tipe data dapat diisi seperti integer, string,  char dan real.

Inti program

Inti program berisikan implementasi dari algoritma-algoritma yang akan digunakan dalam bentuk perintah menggunakan bahasa pascal untuk menghasilkan suatu output. Inti program dapat memerlukan bagian deklarasi bila dalam prosesnya menggunakan variabel-variabel. Bagian ini diakhiri dengan End.

24. Mengapa java dalam perkembangannya lebih cepat diimplementasikan penggunaannya dari bahasa pascal


kenapa bahasa pemrograman java populer dibanding bahasa pemrograman yang lain? Inilah alasannya.

Object Oriented

Bahasa pemrograman java menggunakan pemrograman berorientasi objek (Oriented Object) sehingga java dapat membuat program secara modular dan dapat digunakan kembali. Pemrograman berorientasi objek, memodelkan dunia nyata ke dalam suatu objek dan dapat melakukan interaksi antara objek tersebut.

Gratis

Gratis merupakan salah satu yang paling diincar oleh orang-orang, bukan? Biaya merupakan suatu hal yang penting untuk dipertimbangkan oleh para programmer untuk menciptakan programnya. Java merupakan bahasa pemrograman yang gratis dari awal. Karena gratis itulah salah satu penyebab java menjadi bahasa populer dikalangan programmer individual dan organisasi besar

Aman

Java memliki mekanisme tingkat kemanan yang tinggi untuk menjaga aplikasi sehingga tidak akan merusak sistem komputer yang menjalankan aplikasi tersebut. Hal itu dikarenakan java merupakan bahasa pemrograman yang dipakai untuk aplikasi internet dan terdistribusi.

Ada dimana-mana

Java ada dimana-mana, di web, di ponsel dan di desktop. Begitu juga dengan programmer java hampir juga ada dimana-mana. Dengan tersedianya programmer java yang banyak ini adalah alasan sebuah organisasi memilih bahasa pemrograman java untuk mengembangkan sesuatu yang baru dibanding bahasa pemrograman yang lainnya

Platform iendentndep

Ide independensi dari bahasa pemrograman java tersebut sangat hebat serta tagline “write once run anywhere” merupakan sesatu yang cukup menarik untuk perkembangan baru dalam bahasa pemrograman java. Sebagian besar, bahasa pemrograman java dikembangkan di lingkungan Windows dan dijalankan pada platform UNIX

Dari beberapa alasan itulah menjadikan java sebagai bahasa pemrograman yang populer dikalangan programmer dibanding bahasa yang lainnya. Selain itu, java juga memiliki fasilitas penggunaan memori sehingga pengaturan memori tidak perlu lagi diatur secara langsung.

26. Apa kegunaan simbol semicolon dalam membuat program menggunakan C++ ? Berikan contoh penggunaannya!​


Pembahasan

Penggunaan simbol semicolon (;) pada program bahasa C++ adalah untuk memberi tau komputer bahwa garis perintah pada pemrograman sudah mencapai titik akhir perintahnya.

Tanpa simbol semicolon, program tidak dapat dijalankan (karena komputer tidak akan bisa membaca pemrogramanya baik - baik, cara kerja komputer membaca program adalah untuk menjalankan satu - satu perintahnya, tapi kalau tidak ada simbol semicolon, maka perintahnya tidak dapat diterima dan dijalankan satu - satu)

Contoh penggunaan

printf("Halo dunia");

❃ Penjelasan program

Output akan menjadi - Halo duniaprintf() berfungsi untuk mengeluarkan output."Halo dunia" merupakan string isinya; semikolon untuk memberi tau komputer agar perinta tersebut terbaca.

29. sebutkan aturan penulisan masing² bagian pada struktur bahasa pascal?​


Jawaban:Judul Program

Nama  program diawali dengan kata Program, sebagai kata baku dalam Pascal, diikuti dengan nama  program atau identifier.

Aturan pemberian nama identifer, dalam hal ini termasuk judul program adalah sebagai berikut.

1. Nama identifier harus dimulai dengan karakter huruf alfabet a-z atau A-Z.

2. Setelah karakter pertama, dapat dilanjutkan dengan karakter afanumerik (angka maupun huruf)

dan underscore (_).

3. Dalam penulisan identifier tidak boleh menggunakan karakter-karakter berikut ini.

~ ! @ # $ % ^ & * ( ) + ` - = { } [ ] : " ; ' <> ? , . / |

4. Tidak boleh menuliskan identifier dengan nama yang sama pada kata kunci dari Pascal, seperti  

and, array, begin, case, const, div, do, downto, else, end, file, for, forward, function,

goto, if, in, label, mod, nil, not, of, or, packed, procedure, program, record, repeat, set,

then, to, type, until, var, while, with

5. Jika ingin tetap digunakan, maka kata tersebut harus dirangkai dengan kata/karakter yang lain.

Misal program_if, begin1.

Deklarasi Program

Yang termasuk dari  bagian deklarasi adalah sebagai berikut.

1. Unit, yaitu fungsi-fungsi tambahan dari Pascal, cara memanggilnya adalah dengan

menggunakan kata kunci Uses.

Sebagai contoh pada program di atas adalah penggunaan Uses crt.Uses crt diperlukan

karena kode program menggunakan perintah clrscr untuk membersihkan layar. Perintah ini

disediakan oleh unit crt.

2. Konstanta, yaitu suatu nilai yang besarnya tetap (konstan) selama program dijalankan

3. Deklarasi Variabel, yaitu identifer atau nama-nama untuk menyimpan nilai beserta denan

tipe datanya.

4. Deklarasi Tipe data baru yang didefinisikan oleh programmer

5. Prosedur dan fungsi

Bagian Pernyataan/Statement

Pada bagian ini dituliskan urutan-urutan instruksi yang perlu dieksekusi oleh program. Aturan  penulisan;

1. Bagian program inti diawali dengan Begin(tanpa diikuti tanda baca) dan diakhiri dengan End  (diikuti tanda titik (.) )

2. Dalam Pascal penggunaan huruf besar dan kecil tidak dipermasalahkan (tidak sensitive-case)

3. Untuk setiap perintahatau pernyataan, diakhiri dengan titik koma (;), boleh ada lebih dari 1  pernyataan dalam tiap baris.

4. Jika ada struktur percabangan (kondisi bersyarat) atau perulangan dengan lebih dari satu  pernyataan, diawali dengan begin dan diakhiri dengan end dan titik koma (;)

5. Komentar dituliskan di antara tanda kurung {}
