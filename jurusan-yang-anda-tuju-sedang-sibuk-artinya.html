13. Apa artinya BTS? dalam jurusan TKJ


Jawaban:

BTS dalam jurusan TKJ artinya Base Transceiver Station.

Pembahasan

BTS dalam jurusan TKJ (Teknik Komputer dan Jarinhan) mempunyai kepanjangan Base Transceiver Station yakni merupakan prasarana telekomunikasi yang dimana di dalamnya menyediakan perangkat nirkabel dan jaringan operator yang fungsinya ialah untuk menerima maupun mengirimkan signal radio ke sarana ataupun prasarana yang lainnya yang jangkauannya mencapai 10 km jauhnya.

21. Apabila kamu perhatikan kegiatan yang dilakukan manusia yang ada di sekitar tempat kamu tinggal dari pagi sampai sore hari, mereka sibuk bekerja, berdagang, ke sekolah, ke sawah, atau ke laut. Kesibukan itu tentu saja mempunyai tujuan yang ingin dicapai. Tahukah kamu apa tujuannya?


Tujuan manusia yang tergambar dari kegiatan kesehariannya seperti sekolah, bekerja, dan lainnya merupakan tindakan manusia untuk memenuhi kebutuhan hidupnya, dari nafkah batin hingga ilmu pengetahuan

Pembahasan:

Manusia, selain dikenal makhluk sosial, juga dikenal sebagai makhluk ekonomi atau homo economicus. Hal ini dikarenakan manusia selalu ingin memenuhi kebutuhannya untuk mendapatkan kesejahteraan. Kebutuhan manusia sangatlah beragam. Kebutuhan tersebut selalu ada setiap hari. Contohnya adalah kebutuhan manusia akan keturunan sehingga perlu adanya suatu keluarga, kebutuhan akan pendidikan sehingga perlu adanya sekolah, kebutuhan akan makan sehingga perlu adanya pasar untuk mendapatkan makanan, dan sebagainya.

Untuk memenuhi kebutuhannya, manusia harus melakukan interaksi dengan manusia lainnya, baik secara individu maupun secara kelompok. Hal ini tercermin dari tindakan keseharian manusia, dari sekolah, belanja, bekerja, bertani, dan lain sebagainya.

24. Apa Arti C CC++ dalam jurusan IT


Jawaban:

Sejarah C CC++

Tahun 1978, Brian W. Kerningham dan Dennis W. Ritchie dari AT&T Laboratories mengembangkan bahasa C dari bahasa B.

Pada tahun 1980, bahasa C digunakan oleh kalangan orang Eropa. Bahasa C menjadi populer pada masa itu dengan bahasa lainnya seperti pascal, basic dan fortran.

Tahun 1989, dikeluarkannya standard bahasa C oleh American National Standards Institute.

Pada tahun 1985, Bahasa C dikembangkan lagi menjadi bahasa C++. Tanda ++ berarti kenaikan. Dalam bahasa C, ++ merupakan operasi kenaikan 1.

Kelebihan bahasa C CC++ dari bahasa C adalah bahasa C CC++ mendukung Object Oriented Programming/ Pemrograman Berorientasi Objek.

Compiler yang tersedia adalah Borland C++, Turbo C++, Dev C++ yang dapat digunakan oleh pengguna C CC++. Pada Borland C CC++ dan Turbo C CC++, masih menggunakan sistem DOS. Sedangkan Dev C CC++ sudah memiliki tampilan GUI (Graphical User Interface).

Contoh program sederhana dari C CC++ adalah sebagai berikut:

#include <iostream>

using namespace std;
