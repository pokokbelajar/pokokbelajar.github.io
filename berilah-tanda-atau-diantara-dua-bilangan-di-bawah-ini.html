b. -402 dan - 420

Jawaban :<>PENDAHULUAN :

Macam-macam bilangan :

Bilangan AsliBilangan cacahBilangan primaBilangan bulat

Pengertian :

1.Bilangan Asli :

Bilangan Asli adalahbilangan yang dimulai dari angka 1 dan seterusnya.

2.Bilangan cacah :

Bilangan cacah adalah bilangan yang di mulai dari angak nol dan seterusnya .

3.Bilangan prima :

Bilangan prima adalah bilangan yang mempunyai tepat 2 faktor,yaitu bilangan itu sendiri dan bilangan 1.

4.Bilangan Bulat :

Bilangan bulat adalah bilangan yang terdiri

dari bilangan positif,negatif dan netral (0)

Ketentuan Perkalian Bilangan Bulat :

Positif×Positif=Positif

Positif×Negatif=Negatif

Negatif×Positif=Negatif

Negatif×Negatif=Positif

Ketentuan Pembagian Bilangan Bulat :

Positif÷Positif=Positif

Positif÷Negatif=Negatif

Negatif÷Positif=Negatif

Negatif÷Negatif=Positif

Sifat-Sifat Operasi hitung :

1.Sifat Komutatif :[tex]a + b = b + a[/tex]

[tex]a \times b = b \times a[/tex]

2.Sifat Asosiatif :

[tex]a + (b + c) = (a + b) + c[/tex]

[tex]a \times (b \times c) =( a \times b) \times c[/tex]

3.Sifat Distributif :

[tex]a \times (b + c) = (a \times b) + (a \times c)[/tex]

[tex]a \times (b - c) = (a \times b) - (a \times c)[/tex]

4.Sifat Identitas :

[tex]a + 0 = a (0 = usur\: netral\:penjumlahan)[/tex]

[tex]a \times 1 = a(a = unsur \: netral \:perkalian)[/tex]

PEMBAHASAN :

Sifat bilangan Minus (-) :

Bilangan Minus(-) akan semakin kecil nilainya jika angka yang di keluarkan semakin besar begitu juga sebaliknya bilangan Minus(-) akan semakin besar jika nilainya terus berkurang(kecil)

Sifat Bilangan Positif (+) :

Bilangan Positif(+) akan semakin besar nilainya jika angkanya semakin besar begitu juga sebaliknya bilangan positif(+) akan kecil nilainya jika bilangannya semakin kecil

[tex]\bf {Ingat:}[/tex]

Tanda > adalah tanda besar dari

Tanda < adalah tanda Kurang dari

Tanda = Berat bilangan tersebut nilainya sama atau sama besarnya

Tanda [tex]\geq[/tex] lebih besar sama dengan

Tanda [tex]\leq[/tex] lebih kecil sama dengan

Penyelesaian Soal :Bagian A :[tex]-211\boxed{<}-121[/tex]

[tex]\bf{Keterangan:}[/tex]

-211 lebih kecil dari -121 ,karena bilangan Minus(-) akan lebih kecil nilainya jika bilangan nya semakin besar.l,Sebaliknya Bilangan Minus(-) akan semakin besar nilainya jika bilangannya semakin kecil.

Bagian B :[tex]-402\boxed{>}-420[/tex]

[tex]\bf{Keterangan:}[/tex]

-402 lebih besar dari -420,karena bilangan Minus(-) akan lebih kecil nilainya jika bilangan nya semakin besar.l,Sebaliknya Bilangan Minus(-) akan semakin besar nilainya jika bilangannya semakin kecil.

♣ Soal:

Beri tanda > atau <

a) 3 … –6

b) –2 … 2

c) –8 … –10

d) –7 … –5

♣ Penyelesaian:

a) 3 > –6

b) –2 < 2

c) –8 > –10

d) –7 < –5

♣ Pembahasan:

Contoh baris bilangan:

… , –4, –3, –2, –1, 0, 1, 2, 3, 4, …

• Semakin ke kiri bilangannya akan semakin kecil.

• Semakin ke kanan bilangannya akan semakin besar.

• Bilangan bulat positif lebih besar daripada bilangan bulat negatif.

• Bilangan bulat negatif lebih kecil daripada bilangan bulat positif.
