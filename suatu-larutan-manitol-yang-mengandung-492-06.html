suatu larutan manitol yang mengandung 18,04 gram manitol dalam 100 gram air pada suhu 20°C mempunyai tekanan uap sebesar 17,23 mmHg. jika tekanan uap jenuh air pada suhu tersebut adalah 17,54 mmHg. Maka massa molekul relatif (Mr) manitol tersebut adalah 180,4 g/mol.

Penyelesaian Soal :

Diketahui : massa manitol = 18,04 gram

                  massa air = 100 gram

                  suhu (T) = 20°C

                  tekanan uap (P) = 17,23 mmHg

                  Tekanan uap pelarut (P°) = 17,54 mmHg

Ditanya : massa molekul relatif (Mr) manitol ?

Jawab :

LANGKAH PERTAMA (I)

Hitung fraksi mol zat pelarut (Xp) dengan menggunakan rumus : P = P° × Xp

Keterangan : P = tekanan uap larutan

                     P° = tekanan uap pelarut

                     Xp = fraksi mol pelrut

Caranya :

P = P° × Xp

Xp = P / P°

     = 17,23 mmHg / 17,54 mmHg

     = 0,98

LANGKAH KEDUA (II)

Hitung fraksi mol massa molekul relatif (Mr) H₂O dengan cara sebagai berikut :

Mr H₂O = (2 × Ar H) + Ar O

             = (2 × 1 g/mol) + 16 g/mol

             = 2 g/mol + 16 g/mol

             = 18 g/mol

LANGKAH KETIGA (III)

Hitung mol H₂O dengan cara sebagai berikut :

mol H₂O = massa/ Mr

               = 100 gram / 18 g/mol

               = 5,56 mol

LANGKAH KEEMPAT (IV)

Hitung mol manitol dengan cara sebagai berikut :

misalkan mol manitol = x

Xp = [tex]\frac{mol~air}{mol~manitol~+~mol~air}[/tex]

0,982 = [tex]\frac{5,56~mol}{x~+~5,56~mol}[/tex]

x + 5,56 mol = 5,56 mol / 0,982

x + 5,56 mol = 5,66 mol

x = 5,66 mol - 5,56 mol

x = 0,1 mol

LANGKAH KELIMA (V)

Hitung massa molekul relatif (Mr) manitol dengan cara sebagai berikut :

Mr = massa / mol

    = 18,04 gram / 0,1 mol

    = 180,4 g/mol

∴ Kesimpulan massa molekul relatif (Mr) manitol adalah 180,4 g/mol.

13. Suatu larutan manitol yang mengandung 492,06 gram manitol dalam 50 gram air pada 25derajatC mempunyai tekanan uap 8,77 mmHg. jika tekanan uap jenuh air pada suhu tersebut adalah 17,54 mmHg, hitunglah massa molekul relatif manitol tersebut..


Sebanyak 492,06 gram manitol dalam 50 gram air pada suhu 25°C mempunyai tekanan uap 8,77 mmHg. Apabila diketahui tekanan uap jenuh air pada suhu tersebut adalah 17,54 mmHg, Maka massa molekul relatif (Mr) larutan manitol tersebut adalah 180,4 g/mol.

Penyelesaian Soal :

Diketahui : massa manitol = 18,04 gram

                  massa air = 100 gram

                  suhu (T) = 20°C

                  tekanan uap (P) = 17,23 mmHg

                  Tekanan uap pelarut (P°) = 17,54 mmHg

Ditanya : massa molekul relatif (Mr) manitol ?

Jawab :

LANGKAH PERTAMA (I)

Hitung massa molekul relatif (Mr) H₂O dengan menggunakan cara sebagai berikut :

Mr H₂O = (2 × Ar H) + Ar O

             = (2 × 1 g/mol) + 16 g/mol

             = 2 g/mol + 16 g/mol

             = 18 g/mol

LANGKAH KEDUA (II)

Kemudian setelah itu hitung mol H₂O dengan cara sebagai berikut :

mol H₂O = massa/ Mr

               = 100 gram / 18 g/mol

               = 5,56 mol

LANGKAH KETIGA (III)

Gunakan rumus tekanan uap jenuh larutan untuk dapat dihitung terlebih dahulu fraksi mol zat pelarut (Xp) dengan menggunakan rumus berikut :

P = P° × Xp

Keterangan : P = tekanan uap larutan (mmHg)

                     P° = tekanan uap pelarut (mmHg)

                     Xp = fraksi mol pelarut

Caranya :

P = P° × Xp    (Kemudian turunkan rumus tersebut menjadi)

Xp = P / P°

     = 17,23 mmHg / 17,54 mmHg

     = 0,98

LANGKAH KEEMPAT (IV)

Hitung mol manitol dengan mengguanakan cara sebagai berikut :

misalkan mol manitol = nt

Xp = np / (np + nt)

0,982 = 5,56 mol / (5,56 mol + nt)

nt + 5,56 mol = 5,56 mol / 0,982

nt + 5,56 mol = 5,66 mol

nt = 5,66 mol - 5,56 mol

nt = 0,1 mol

LANGKAH KELIMA (V)

Kemudian langkah terakhir hitung massa molekul relatif (Mr) manitol dengan cara sebagai berikut :

Mr = massa / mol

    = 18,04 gram / 0,1 mol

    = 180,4 g/mol

∴ Kesimpulan massa molekul relatif (Mr) manitol adalah 180,4 g/mol.

Penjelasan:

Diketahui: Massa = 18,04 gram

                 Massa H₂O = 100 gram

                 P = 17,23 mmHg

                 P° = 17,54 mmHg

Ditanya: Mr montimol adalah?

Jawab:

Tahap 1.

Tentukan fraksi mol terlebih dahulu (Xp) dengan cara berikut:

P = P° × Xp

Xp = P / P°

      = 17,23 mmHg / 17,54 mmHg

      = 0,98

Tahap 2.

Tentukan mol H₂O (zat pelarut) dengan cara:

n H₂O = massa/Mr

          = 100 gram/ 18 gram/mol

          = 5,56 mol

Tahap 3.

selanjutnya mentukan mol larutan montimol dengan cara:

Xp = np / np + nt

Ket : Xp = Fraksi mol

       np = mol pelarut (Air)

       nt  = mol zat terlarut (montimol)

Xp = np / np + nt

0,98 = 5,56 mol / 5,56 mol + nt

nt + 5,56 mol = 5,56 mol / 0,98

nt + 5,56 mol = 5,67

nt = 5,67 - 5,56

nt = 0,11 mol

Tahap 4.

Setelah diketahui mol montimol selanjutnya, menentukan massa molekul relatif dengan cara:

mol = massa/Mr

Mr = massa/mol

Mr = 18,04 gram / 0,1 mol

Mr = 164 gram/mol

Maka,  massa molekul relatif montimol adalah sebesar 164 gram/mol

22. Suatu larutan manitol yang mengandung 18,04 gram manitol dalam 100 gram air pada suhu 20˚c mempunyai tekanan uap 17,23 mmhg. jika tekanan uap jenuh air pada suhu tersebut adalah 17,58 mmhg, hitunglah massa molekul relatif manitol!


Jawaban:

suatu larutan manitol yang mengandung 18,04 gram manitol dalam 100 gram air pada suhu 20°C mempunyai tekanan uap sebesar 17,23 mmHg. jika tekanan uap jenuh air pada suhu tersebut adalah 17,54 mmHg. Maka massa molekul relatif (Mr) manitol tersebut adalah 180,4 g/mol.

Penyelesaian Soal :

Diketahui : massa manitol = 18,04 gram

                  massa air = 100 gram

                  suhu (T) = 20°C

                  tekanan uap (P) = 17,23 mmHg

                  Tekanan uap pelarut (P°) = 17,54 mmHg

Ditanya : massa molekul relatif (Mr) manitol ?

Jawab :

LANGKAH PERTAMA (I)

Hitung fraksi mol zat pelarut (Xp) dengan menggunakan rumus : P = P° × Xp

Keterangan : P = tekanan uap larutan

                     P° = tekanan uap pelarut

                     Xp = fraksi mol pelrut

Caranya :

P = P° × Xp

Xp = P / P°

     = 17,23 mmHg / 17,54 mmHg

     = 0,98

LANGKAH KEDUA (II)

Hitung fraksi mol massa molekul relatif (Mr) H₂O dengan cara sebagai berikut :

Mr H₂O = (2 × Ar H) + Ar O

             = (2 × 1 g/mol) + 16 g/mol

             = 2 g/mol + 16 g/mol

             = 18 g/mol

LANGKAH KETIGA (III)

Hitung mol H₂O dengan cara sebagai berikut :

mol H₂O = massa/ Mr

               = 100 gram / 18 g/mol

               = 5,56 mol

LANGKAH KEEMPAT (IV)

Hitung mol manitol dengan cara sebagai berikut :

misalkan mol manitol = x

Xp =

0,982 =

x + 5,56 mol = 5,56 mol / 0,982

x + 5,56 mol = 5,66 mol

x = 5,66 mol - 5,56 mol

x = 0,1 mol

LANGKAH KELIMA (V)

Hitung massa molekul relatif (Mr) manitol dengan cara sebagai berikut :

Mr = massa / mol

    = 18,04 gram / 0,1 mol

    = 180,4 g/mol

Penjelasan:

Kesimpulan massa molekul relatif (Mr) manitol adalah 180,4 g/mol.
