N (untuk berapa hari), M (untuk berat cabai), dan X (untuk harga yang akan didapatkan).

Variabel K, gunanya untuk variabel perulangan. Variabel k untuk flowchart sudah dideklarasikan untuk perulangan.

Variabel "Hasil", gunanya untuk menunjukkan harga dari X * M.

5 x 2 = 10 kg.

Perulangan ke-3

1280 * 2 = 2560 kg.

X * M (2560) = 75 * 2560 = 192.000

Pembahasan

Algoritma digunakan dalam kehidupan manusia berupa masalah-masalah maupun secara penyelesaian urusan komputer dengan pemikiran logis yang jelas, dari hal tersebutlah tentu saja algoritma memiliki ciri-ciri yang penting. Berikut beberapa ciri-ciri tersebut:

Dari pemikiran manusia yang mana menjadi inti program untuk dilaksanakan pada komputer. Pada program yang dibuat oleh manusia secara umum memiliki sebuah pernyataan program instruksi yang ada stabil dan bisa juga berkembang, hal-hal tersebut tergantung sesuai dengan pembuatan yang ada. Maka dari hal tersebut pentingnya peranan manusia dalam menghasilkan program yang berdampak besar terhadap komputer yang dijalankan.Dapat diterjermahkan dalam bahasa program. Dalam bahasa program sendiri algoritma harus jelas dan bisa diterjemahkan agar bisa berjalan dengan baik, hal tersebut dapat terjadi jika bahasa pemrograman sudah ditulis dengan relatif baik dan benar. Maka dari hal tersebut pentingnya membuat bahasa program yang benar sehingga instruksi bisa dijalankan dengan baik.Keluaran algoritma menghasilkan program yang sama. Program secara umum berupa cerminan dari algoritma yang telah dibuat untuk banyak kepentingan sesuai kebutuhan komputer, dari hal tersebut banyak orang yang menjadi programmer untuk membuat program yang penting untuk komputer. Pengertian Algoritma

Algoritma adalah susunan logis dan sistematis dari langkah-langkah untuk memecahkan atau menyelesaikan suatu permasalahan.

Dalam dunia pemrograman, algoritma digunakan untuk membuat software dengan berbagai kegunaan yang digunakan pada perangkat komputer. Algoritma dalam pembuatan software diimplementasikan dengan beberapa bahasa pemrograman seperti C/C++, C#, Visual Basic, dan bahasa pemrograman lainnya.Tanpa algoritma yang tepat, baris demi baris kode tidak akan menghasilkan output yang diinginkan. Oleh karena itu algortima sangat penting dalam membuat program berfungsi sebagaimana mestinya.

Pengertian dan Pengenalan Program

Apakah program menurut kalian? Program adalah file teks yang berisi pernyataan-pernyataan dalam bahasa pemrograman.  Kemudian bagaimanakah cara membuat program? Ada beberapa langkah yang dilakukan, yaitu:

Merancang program. Disini kita mendefinisikan masalah dan menulis algoritmanya untuk penyelesaian masalah tersebut. Memasukkan program ke komputer (Implementasi).  Kompilasi program ke “excecutable”. Excecutable: membuat program dapat dijalankan. Jalankan program. Ulangi hingga benar. Ini disebut proses debugging.

Struktur program, disini kita gunakan patokan bahasa C/C++

Bahasa C/C++ secara umum memiliki struktur program sbb:

Daftar Header file Deklarasi variabel global dan fungsi-fungsi tambahan  Kepala fungsi utama /main Definisi fungsi utama / main Definisi definisi tambahan (subroutine)

Subroutine dapat ditulis setelah fungsi utama atau didalam Library. Namun bila diletakan pada library, maka file library harus disertakan dengan menggunakan preprocessor directive yang diletakan pada daftar header file.

1. untuk menghitung keliling dan luas lingkaran digunakan rumus:

Keliling (K) = 2 x pi x r

Luas (L) = pi x r x r

Pseudocode menghitung keliling dan luas lingkaran adalah sebagai berikut:

ALGORITMA menghitung keliling dan luas lingkaran

Deklarasi:

     luas, keliling, jarijari : float

     pi = 3.14

Deskripsi:

     read( jarijari )

     keliling = 2 * pi * jarijari

     luas = pi * jarijari * jarijari

     write( keliling )

     write( luas )


Flowchart ada di lampiran


Kode program akan dituliskan dalam C/C++

#include<stdio.h>

int main()

{

     float luas, keliling, jarijari;

     float pi = 3.14;

     printf("masukan jari-jari : ");

     scanf("%f", &jarijari);

     luas = pi * jarijari * jarijari;

     keliling = 2 * pi * jarijari;

     printf("keliling : %f\n", keliling);

     printf("luas : %f\n", luas);

     return 0;

}


2. Algoritma dan program konversi suhu celcius ke reamur ke fahrenheit ke kelvin.

Rumus konversi celcius ke reamur = 4 / 5 x suhu dalam celcius

Rumus konversi celcius ke fahrenheit = 9 / 5 x suhu dalam celcius + 32

Rumus konversi celcius ke kelvin = suhu dalam celcius + 273

Pseudocode konversi suhu celcius ke reamur ke fahrenheit ke kelvin adalah sebagai berikut:

ALGORITMA konversi suhu celcius ke reamur ke fahrenheit ke kelvin

Deklarasi:

     c, f, r, k : float

Deskripsi:

     read( c )

     r = 4 / 5 * c

     f = 9 / 5 * c + 32

     k = c + 273

     write(r)

     write(f)

     write(k)

Note: celcius disimbolkan dengan c, reamur = r, fahrenheit = f, kelvin = k


Kode program akan dituliskan dalam C/C++

#include<stdio.h>

int main()

{

     float c, r, f, k;

     printf("masukan suhu dalam celcius : ");

     scanf("%f", &c);

     r = 4 / 5 * c;

     f = 9 / 5 * c + 32;

     k = c + 273;

     printf("suhu dalam reamur : %f\n", r);

     printf("suhu dalam fahrenheit : %f\n", f);

     printf("suhu dalam kelvin : %f", k);

     return 0;

}
