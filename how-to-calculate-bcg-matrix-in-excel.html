2. how to calculate this?


[tex]P(X) = 0.27 \\
P(Y) = 0.48 \\
P(X|Y') = 0.36[/tex]
The last one called conditional probability.

If [tex]P(X) = 0.27[/tex] then
[tex]P(X') = 1-P(X)
\\P(X')=1-0.27
\\P(X')=0.73[/tex]

If [tex]P(Y) = 0.48[/tex] then
[tex]P(Y') = 1-P(Y)
\\P(Y') =1-0.48
\\P(Y') =0.52[/tex]

a. [tex] P(X \cap Y')[/tex]?
We can use [tex]P(X|Y') [/tex] to help us find [tex] P(X \cap Y')[/tex].

[tex]P(X|Y') = \frac{P(X \cap Y') } {P(Y')} \\ 0.36 = \frac{P(X \cap Y')}{0.52} \\
P(X \cap Y')= (0.36)(0.52)= 0.1872 \\ [/tex]

b. [tex]P(Y|X') [/tex]?
[tex]P(Y|X') = \frac{P(Y \cap X') } {P(X')} \\ [/tex]
First we need to calculate [tex]P(Y \cap X')[/tex], but we need to know that
[tex]P(X) = P(X \cap Y) +P(X \cap Y') \\
0.27 = P(X \cap Y) + 0.1872 \\
P(X \cap Y) = 0.27 - 0.1872 = 0.0828[/tex]

And,
[tex]P(Y) = P(Y \cap X) +P(Y \cap X') \\
0.48 = 0.0828 + P(Y \cap X') \\
P(Y \cap X') = 0.48 - 0.0828 = 0.3972[/tex]

Then,
[tex]P(Y|X') = \frac{P(Y \cap X') } {P(X')} \\
= \frac{0.3972} {0.73} \\
= 0.5441[/tex]

c. Two events X and Y are independent if and only if [tex]P(X \cap Y) = P(X) P(Y)[/tex]

Now lets check the value of both side,
[tex]P(X \cap Y) = 0.0828[/tex]
[tex]P(X)P(Y)=0.27×0.48 =0.1296[/tex]

Both values are different then we can conclude that both events X and Y are not independent because [tex]P(X \cap Y) ≠ P(X) P(Y)[/tex].

11. how to calculate that mathematic questions ?


Itu Merupakan soal perbandingan dan cukup gampang.

Penyelesaian :
1. Lihat Gambar Pensil
Pensil Hitam = 5
Pensil Putih = 3
Semua Pensil = 5 + 3 = 8

Perbandingan :
Pensil Hitam : Semua Pensil = 5 : 8

Anda telah mendapatkan sebuah perbandingan, cukup mudah kan?

2. Lihat Gambar Kelinci
Kelinci Hitam = 10
Kelinci Putih = 4
Semua Kelinci = 10+4 =14

Perbandingan :
Kelinci Hitam : Semua Kelinci = 10 : 14 = 5 : 7

Jadi, anda dapat mengecilkan perbandingan seperti mengecilkan pecahan.

3. Lihat Gambar Ikan
Ikan Merah = 7
Ikan Putih = 3
Semua Ikan = 7+3 = 10

a. Ikan Merah : Ikan Putih = 7 : 3
b. Ikan Merah : Semua Ikan = 7 : 10

29. How do scientists calculate the magnitude of an earthquake (help)​


The Richter scale measures the greatest vibration (amplitude) in record, but another measure measures a different part of an earthquake. The usgs currently report the scale of earthquakes using their present size, although many other magnates are calculated for research and comparison.

English:

Seismographs have sensitive instruments that can receive seismic waves generated by earthquakes. The seismic waves that occur during the earthquake are depicted with wavy lines on the seismogram. Seismologists measure these lines and calculate the magnitude of the earthquake.

30. convert decimal time to hours and minutes in excel


1. Select a blank cell which you want to output the converted result, and enter this formula =A2/24, drag fill handle over the cells you need. See screenshot:


2. Keep the formula cells highlighted, and right click to show context menu and select Format Cells. And in the Format Cells dialog, select Time from the Category list, and choose one time type as you need from right section. See screenshot:




3. Click OK. Now the decimal hours have been converted to time format.
