Jawab: dari ilustrasi soal dapat kita rangkum

yang diketahui:

modal = 8.000.000

harga sepatu pria = 20.000

harga sepatu wanita = 16.000

keuntungan sepatu pria = 6.000

keuntungan sepatu wanita = 5.000

kapasitas kios = 450 pasang

pertanyaan :

a) model matematika

b) keuntungan maksimum

penyelesaian

jika sepatu pria = x dan sepatu wanita = y

model matematika yang dapat dibuat

1) 20.000x + 16.000y ≤ 8.000.000 kita kecilkan dengan membagi semua ruas dengan angka 4.000, sehingga model matematikanya

5x + 4y ≤ 2.000

2) x + y ≤ 450

3) x ≥ 0

4) y ≥ 0

5) nilai obyektif → f(x,y) = 6.000x + 5.000y

penyelesaian agar mengetahui keuntungan maksimum

untuk pertidaksamaan 1

5x + 4y ≤ 2000

kita ambil tanda = untuk mengetahui titik potong terhadap sumbu x dan sumbu y nya

5x + 4y = 2000

memotong sumbu x, jika y = 0

5x + 4y = 2000

5x + 4(0) = 2000

5x + 0 = 2000

5x = 2000

x = 2000/5

x = 400

memotong sumbu y, jika x = 0

5x + 4y = 2000

5(0) + 4y = 2000

0 + 4y = 2000

4y = 2000

y = 2000/4

y = 500

untuk pertidaksamaan 2

x + y ≤ 450

kita ambil tanda = untuk mengetahui titik potong terhadap masing" sumbu

x + y = 450

memotong sumbu x, jika y = 0

x + y = 450

x + 0 = 450

x = 450

memotong sumbu y, jika x = 0

x + y = 450

0 + y = 450

y = 450

untuk mengetahui titik potong kedua persamaan kita eliminasi kedua persamaan

5x + 4y = 2000 → kali 1 → 5x + 4y = 2000

x + y = 450        → kali 4 → 4x + 4y = 1800  -

                                           x  = 200

subsitusikan ke persamaan x + y = 450

200 + y = 450

y = 450 - 200

y = 250

dari hasil hitungan tadi kita buat gambarnya , perhatikan lampiran

dari gambar kita ketajui ada 3 titik uji,

kita subsitusikan 3 titik uji pada nilai obyektif f(x,y)

f(x,y) = 6.000x + 5.000y

f(0,450) → 6000(0) + 5000(450) = 2.250.000

f(200,250) → 6000(200) + 5000(250) = 2.450.000

f(400,0) → 6000(400) + 5000(0) = 2.400.000

jadi keuntungan maksimum didapat ketika pedagang membeli 200 sepatu pria dan 250 sepatu wanita, dengan besar keuntungan 2.450.000

Untuk mengungkapkan ekspresi yang menyatakan kemampuan, kita dapat menggunakan modal verb ‘can’. Sebelum menuliskan ekspresi kemampuan berdasarka teks, mari kita bahas cara membuat kalimatnya dan terjemahkan percakapan kedalam Bahasa Inggris.

Pembahasan

Rumus yang dapat kita gunakan untuk membuat kalimat yang mengekspresikan kemampuan adalah:

(+) Subject + can + base verb(-) Subject + can’t + base verb(?) Can + subject + base verb?

Contoh:

I can play piano.I can bake a cake.She can paint very well.My father can drive fast.My mother can cook traditional Indonesian food.

Terjemahan percakapan

Andi: Hi Ando, how are you today?

Ando: Oh hi Andi, I’m fine, thanks.

Andi: Do you have any plan for tomorrow?

Ando: No, I don’t have any plan. Why?

Andi: Would you like to come to my uncle’s house with me?

Ando: Of course I would like to.

Andi: But, can you drive a car?

Ando: Don’t worry, it’s easy. I have taken a driving course and got a sim, but I can’t drive a big car.

Andi: It’s not a problem. My car is small so you don’t have to drive a big car.

Ando: Okay. So, when are we going to go?

Andi: Wait at your house, I’ll come to your house.

Ando: Okay!

Jawaban

Can you drive a car? (Apakah kamu mengendarai mobil?)Don’t worry, it’s easy. (Jangan khawatir, itu mudah = saya bisa mengendarai mobil)I can’t drive a big car. (Aku tidak bisa mengendarai mobil yang besar)

18. Ando naik motor ke rumah Gino dengan kecepatan 60 km/jam. Jika Ando berangkat pukul 13.30, dan jarak rumah Ando dengan Gino adalah 30 km, maka pukul berapakah Ando tiba di rumah Gino​


PEMBAHASAN

Diketahui:

K = 60 km/jam

berangkat = 13.30

J = 30 km

Ditanya:

tiba = ?

Jawaban:

Menentukan lama

[tex]w = \frac{j}{k} [/tex]

[tex]w = \frac{30 \: km}{60 \: km/jam} [/tex]

[tex]w = \frac{1}{2} \: jam[/tex]

[tex]w = \frac{1}{2} \times 60 \: menit[/tex]

[tex]w = 30 \: menit[/tex]

Menentukan tiba

tiba = berangkat + lama

tiba = 13.30 + 30 menit

tiba = 14.00

Kesimpulan:

Jadi, Ando tiba di rumah Gino pada pukul 14.00.

24. Ando dan nanda sedang bermain jungkat-jungkit.nanda lebih gemuk daripada ando.pada saat jarak nanda ke titik tumpu dan jarak ando ke titik tumpu sama, ando tidak dapat mengangkat nanda.mengapa demikian? Apa yg harus di lakukan ando agar dia dapat mengangkat nanda sehingga mereka dapat bermain jungkat-jungkit


Ando tidak dapat mengangkat Nanda dikarenakan terjadi ketidak seimbangan antara beban dan gaya kuasa yang diberikan. Adapun besarnya beban berbanding terbalik dengan besar lengan beban. Sehingga agar Ando bisa mengangkat Nanda, Ando perlu mendekat ke titik tumpu (memperkecil jarak antara lengan beban dengan kedudukannya di awal).

Pembahasan:

Jungkat-jungkit merupakan salah satu alat yang digunakan untuk memindahkan beban. Jungkat-jungkit terdiri dari komponen beban, lengan beban, titik tumpu, kuasa dan lengan kuasa. Adapun secara matematis dapat dilihat sebagai berikut

W x Lb = F x Lk

atau dapat juga ditulis dengan

[tex]\frac{W}{F} = \frac{Lk}{Lb}[/tex]

Dari persamaan di atas dapat disimpulkan bahwa beban berbanding terbalik dengan lengan beban dan besarnya gaya kuasa berbanding terbalik dengan lengan kuasa.
