= 48%banyak topi 30buah

Penyelesaian

Presentase Keuntungan

Harga Jual = (Rp.15.000,- x 8) + (Rp.12.000,- x 32)

Harga Jual = Rp.120.000,- + Rp.384.000,-

Harga Jual = Rp.504.000,-

Modal = Harga Jual - Keuntungan

Modal = Rp.504.000,- - Rp.116.000,-

Modal = Rp.388.000,-

Keuntungan = (Rp.116.000,-/Rp.388.000,-) x 100%

Keuntungan = (29/97) x 100%

Keuntungan = (2900/97)%

Keuntungan = 29,8969072%

Keuntungan = 30%

Jadi, keuntungan dari penjualan topi sebesar 30%.
