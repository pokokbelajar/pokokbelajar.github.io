13. Apa yang anda ketahui tentang menu Ribbon pada Microsoft Excel ?


Kelas         : XI
Pelajaran   :  Information Technology
Kategori     :  I T dan sistem informasi
Kata Kunci : Information Technology, Ekonomi, Excel,microsoft,menu, ribbon
Kode          :  -,-,-
   

          halo adik adik pada kesempatan kali ini kakak akan sedikit berbagi mengenai soal "Apa yang anda ketahui tentang menu Ribbon pada Microsoft Excel ?" Di dunia TI  yang populer saai ini yaitu microsoft Excel, seperti kita ketahui penggunaaan aplikasi microsoft sudah tidak asing di kalangan kita. ada Microsoft Windows, Words, ada microsoft Excel dan lain lain. kali ini kita akan lebih membahas sedikit mengenai menu dalam aplikasi Microsoft Excel yaitu menu ribbon. karena yang ditanyakan adalah itu.
     
          pada tgl 4 Aprli 1975 dua pemuda yaitu Paul Alen dan bill Gates membuat sebuah perusahaan yang bernama microsoft. Pusat nya di Redmond, Washington Amerika. Perusahaan ini bergerak di bidang produk dan jasa yang berhubungan dengan komputer. Salah satu produk nya yang sering kita pakai selain ms word adalah microsoft excel. ini adalah aplikasi pengolah data yang sangat membantu sekali untuk kita dalam menganalisa data , mengolah nya hingga menyajikan nya menjadi sebuah laporan yang menarik. Untuk menyusun Laporan Keuangan ekonomi dan lain lain. 

          Pada Microsoft Office 2002 yang di gunakan adalah menu bar dan toolbar lalu pada Microsoft Office 2007 menu itu di ganti dengan mennu ribbon. fungsi nya adalah sebagai penyempurna mempermudah kita dalam mengolah data melakukan pengeditan karena dalam menu ribbon berisi tombol tombol yang lebih komplit dan tambahan fasilitas sehingga kita sebagai pengguna dalam melakukan pengeditan data/ dokumen jadi lebih mudah dan baik.

Di dalam menu ribbon di situ terdapat , antara lain : 
   
1. Ribbon Home

Clipboard            : Copy, Paste, Cut, Format 
Font                    : Jenis font, ukuran font, warna huruf, Bold, Italic, Underline, warna background cell,Border
Cell Alignment     : Top alignment, Middle alignment, Bottom alignment, Align text left, Align text center, Align text right, Decrease indent, Increase indent, Warp text, Merge & Center 
Number              : Combo box format cell, Accounting number format, Percent style, Comma style, Increase decimal, Decrease decimal 
Styles                 : Conditional formatting, Format as table, Cell styles 
Cell                     : Insert, Delete dan Format cell, selected cell, format cell. 
Editing                : Sum, Sort, Find & Select, Fill
   
2. Ribbon Insert

Table                   : Pivot Table, TableIllustrations: Picture, Clip Art, Shapes, SmartArt 
Charts                 : Column, Line, Pie, Bar, Area, Scatter, Other Chart 
Link                     : Hyperlink 
Text                     : Text box, Header & Footer, Wordart, Sugnature Line, Object, Symbol

3. Ribbon Page Layout

Themes               : Themes, Colors, Fonts,Effects
Page Setup         : Margin, Orientation, Size, Print Area, Breaks, Background, Print Tittles
Scale to Fit          : Width, Height, ScaleSheet
Options               : Gridlies, Heading
Arrange               : Bring to Front, Send to Back, Selection Pane, Align, Group ,Rotate

4. Ribbon Formulas

Function Library   : Insert Function, Auto Sum, Recently Used, Financial, Logical, Text, Date & Time, Lookup and References, Math and Trig, More Functions
Defined Names    : Name Manager, Define Name, Use in Formula, Create from Selection
Formula Auditing  : Trace Precedents, Trace Dependents, Remove Arrows, Show Fromula, Error Checking, dan Evaluate Formula
Calculation           : Calculation Options

5. Ribbon Data

Get External Data : From Access, From Web, From Text, From Other Sources, Existing Connections.
Connections        : Refresh All, Connections, Properties, Edit Links.
Sort Filter            : Sort, Filter
Data Tools           : Text to Columns, Remove Duplicates, Data Validation, Consolidate, What If Analysis
Outline                 : Group, Ungroup dan Subtotal

6. Ribbon Review

Comments           : New Comment,
Delete                  : Previous, Next, Show/Hide Comment, Show All Comments ,Show Link
Changes              : Protect Sheet, Protect Workbook, Share Workbook, Protect, Share Workbook, Allow Users

Window                  : New Window, Arrange All, Freeze Panes, Slip, Hide, Unhide, Save Workspace, Switch

29. Apa yang anda ketahui tentang Microsoft Office, Microsoft Excel, dan Microsoft Power Point......​


1. Microsoft Office merupakan suatu perangkat lunak yang didalamnya terdapat software seperti Microsoft Excel, Microsoft Team, Microsoft Word, dan Microsoft PowerPoint. Microsoft Office ini dibuat oleh Perusahaan Microsoft Corporation untuk mengembangkan software yang dibuatnya agar banyak yang menggunakan dan menjadi populer.

2. Microsoft Excel merupakan suatu perangkat lunak pengolah angka, seperti membuat suatu rumus, jadwal pelajaran, atau daftar absensi siswa.

3. Microsoft PowerPoint merupakan suatu perangkat lunak yang digunakan untuk membuat presentasi atau desain yang berbentuk file, agar menjadi lebih menarik. biasanya digunakan untuk guru mengajar siswa pada pembelajaran daring.

Pembahasan

✏ Microsoft Office dibuat dan dikembangkan oleh tim Microsoft Corporation yang berasal dari Amerika Serikat. Mereka membuat software/program yang berhubungan dengan komputer/laptop. Ada beberapa saja software yang akan kita bahas diantaranya adalah Microsoft Word, Microsoft Excel, dan Microsoft PowerPoint.

Microsoft Word dibuat oleh Bill Gates di Washington, Pendiri Microsoft Word ini dikenal orang terkaya di dunia, karena penghasilan yang sangat besar diperolehnya dari Microsoft Word hingga sekarang. Microsoft Excel dibuat oleh Perusahaan Microsoft Corporation yang digunakan untuk pengolahan angkaMicrosoft PowerPoint dibuat oleh Robert Gaskins dan Dennis Austin yang dahulunya hanya bernama PowerPoint dan dibayar oleh Perusahaan Microsoft.
