17. untuk membuat kue ubi jalar,menggunakan tepung terigu dan ubi jalar kukus dengan perbandingan 1:2 jika adit akan membuat kue ubi jalar dengan 500gram ubi jalar berapah kah tepung terigu yg adit butuhkan


Untuk mengerjakan soal diatas bisa langsung diperhatikan pembahasan soal sebagai berikut :

Pembahasan

Cara pertama :

tepung terigu : ubi jalar  

=           1           :     2

=           x           :    500 gram

Artinya : 2 unit = 500 gram, sehingga 1 unit = 500 gram : 2 = 250 gram.

Maka tepung terigu yg dibutuhkan Adit adalah 250 gram.

Cara kedua :

Untuk mencari jumlah terigu yg dibutuhkan Adit yaitu :

= [tex] \frac {1} {2}[/tex] x 500 gram

= [tex] \frac {1} {2 : 2}[/tex] x ( 500 : 2 ) gram

= 250 gram.

Maka tepung terigu yg dibutuhkan Adit adalah 250 gram.
