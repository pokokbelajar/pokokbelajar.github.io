3. Akar-akar persamaan kuadrat 5x2 – 3x + 1 = 0 adalah …​


Jawaban:

Akar-akar persamaan kuadrat 5x² - 3x + 1 = 0 adalah akar-akar imajiner.

Penjelasan dengan langkah-langkah:

Diketahui

Persamaan kuadrat 5x² - 3x + 1 = 0

Ditanya

Jenis akar-akar persamaan kuadrat tersebut apakah termasuk imajiner, kompleks, atau nyata, rasional, dan sama.

Proses

Bentuk umum persamaan kuadrat ax² + bx + c = 0.

Dengan a, b, c ∈ R (bilangan real), dan a ≠ 0.

a sebagai koefisien x²

b sebagai koefisien x

c adalah konstanta

Diskriminan adalah D = b² - 4ac, digunakan untuk mengamati sifat-sifat akar yang dimiliki oleh suatu persamaan kuadrat.

D ≥ 0 ⇒ memiliki akar-akar real

D > 0 ⇒ memiliki akar-akar real yang berbeda

D = 0 ⇒ memiiki akar tunggal atau akar kembar

D < 0 ⇒ tidak memliki akar-akar real (imajiner/khayal)

Mari kita amati sifat diskriminan dari persamaan kuadrat 5x² - 3x + 1 = 0,

a = 5

b = -3

c = 1

D = (-3)² - 4(5)(1)

D = 9 - 20

D = -11

Karena D < 0, maka persamaan kuadrat tersebut tidak memiliki akar-akar real atau dengan kata lain akar-akarnya imajiner.
