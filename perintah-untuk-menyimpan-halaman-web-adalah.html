15. langkah langkah untuk menyimpan sebuah halaman web yang ditampilkan


Menyimpan sebuah halaman web

Pada bagian atas jendela Firefox, klik pada menuFile, lalu pilihSave Page As.... JendelaSave Asakan muncul.Gunakan jendelaSave Asuntuk memilih lokasi penyimpanan halaman Web sesuai dengan yang Anda kehendaki.Pada paneldropdownSave As Type, pilih format penyimpanan halaman yang diinginkan:


Web page, complete: Jika Anda memilih opsi ini maka seluruh halaman web akan disimpan bersama dengan seluruh gambar yang Anda di halaman tersebut. Opsi ini dapat digunakan agar Anda untuk melihat halaman web yang telah Anda simpan sebagaimana aslinya, walaupun struktur tautan HTML akan berbeda dibandingkan halaman aslinya. Firefox akan membuat direktori baru tempat seluruh gambar dan berkas yang diperlukan untuk membuka halaman yang akan disimpan.Web page, HTML only: Opsi ini memungkinkan Anda menyimpan halaman asli namun tanpa disertai dengan gambar. Seluruh struktur tautan HTML akan disimpan pada satu berkas sebagaimana halaman aslinya.Text files: Jika opsi ini dipilih, halaman asli akan disimpan sebagai sebuah berkas teks. Struktur tautan HTML tidak akan tersimpan. Akan tetapi dengan opsi ini, Anda dapat membuka hasil simpanan halaman web dengan menggunakan program editor teks.All files: Pada dasarnya sama, opsi ini sama halnya dengan dengan opsi "Web page, HTML only", hanya saja Anda dapat menentukan sendiri ekstensi berkasnya (misal: ".htm" atau ".shtml").KlikSave. Salinan halaman web akan tersimpan sesuai dengan tipe yang telah Anda pilih di lokasi yang telah Anda tentukan.
