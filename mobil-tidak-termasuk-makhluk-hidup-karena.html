26. mobil dapat bergerak,akan tetapi tidak termasuk kelompok makhluk hidup, mengapa demikian?


Ans:
Ada syarat lain agar suatu benda dikatakan hidup:
// Mampu tumbuh dan berkembang.
// Mampu berkembang biak.
// Melakukan ekskresi.
// Merespon iritasi dan stimulan eksternal.
// Membutuhkan nutrisi dan suplemen.
// Mampu melakukan respirasi atau pernapasan.
// Mampu bergerak (entah pasif atau aktif).

Mobil hanya memnuhi satu dari tujuh syarat makhluk hidup, maka mobil bukanlah makhluk hidup.
