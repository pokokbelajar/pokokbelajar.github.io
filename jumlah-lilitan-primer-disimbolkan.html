4. kuat arus kumparan primer = 1,5 A jumlah lilitan primer = 500 lilitan. jumlah lilitan sekunder = 125 lilitan. berapa kuat arus sekunder???


Diketahui :

Ns = 125 lilitan

Np = 500 lilitan

Ip = 1,5 A

Ditanya :

Is ?

Rumus :

Np/ Ns x Is/Ip

Is = Np . Ip/ Ns

= 500. 1,5 /125

= 750 /125

= 6 ... A

Keterangan :

1.Untuk huruf awal huruf besar { N, I, dan V }

2.Dan untuk sisanya menggunakan huruf kecil { p dan s}.

3. Apabila rumus masih tidak jelas maka bisa lihat di gambar atas pada aplikasi dan di bawah untuk website

-----------------------

Transformator

------------------------

Transformator step up berfungsi untuk menaikkan tegangan

Transformator step down berfungsi untuk menurunkan tegangan

keterangan rumus :

Is = kuat arus sekunder

Ip = Kuat arus primer

Vp = tegangan primer

Vs =tegangan sekunder

Np = lilitan primer

Ns = lilitan sekunder

Jika sebuah transformator bertipe step up maka

-.Vs lebih besar dari Vp

-. Ns lebih besar dari Np

Jika sebuah transformator bertipe step down maka

-. Vp lebih besar dari Vs

-. Np lebih besar dari Ns

Tambahan rumus :

Perhitungan efesiensi trafo (π)

π = Pout /Pin x 100%

Keterangan rumus :

Pout = daya listrik pada kumparan sekunder

Pin = daya listrik pada kumparan primer

5. apabila jumlah lilitan primer sebanyak 200 lilitan dan lilitan sekunder sebanyak 1000 lilitan tegangan primer 110 Volt maka tegangan sekundernya adalah


diketahui:
Np = 200 lilitan
Ns = 1000 lilitan
Vp = 110 V

ditanya:
Vs?

jawab:
Vs / Vp = Ns / Np
Vs = Vp Ns / Np
Vs = 110 . 1000 / 200
Vs = 110.000 / 200
Vs = 550 V

jadi tegangan sekundernya adalah 550 V
