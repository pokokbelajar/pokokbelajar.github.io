3. cara cara memblokir YouTube dengan mikrotik ​


BISMILLAH

Jawaban:

Berikut cara memblokir akses ke situs YouTube menggunakan Mikrotik :

Akses Mikrotik RouterOS melalui winbox atau terminal console.Pilih menu "IP" -> "Firewall" -> "Filter Rules".Klik tombol "+" untuk membuat aturan baru.Pada bagian "Chain", pilih "Forward".Pada bagian "Protocol", pilih "tcp".Pada bagian "Dst. Port", masukkan nomor port untuk YouTube (80, 443, atau 8080).Pada bagian "Action", pilih "Drop".Pada bagian "Comment", masukkan keterangan misalnya "Blokir Akses ke YouTube".Klik tombol "Apply" untuk menyimpan perubahan.

Penjelasan:
Dengan mengikuti langkah-langkah di atas, akses ke situs YouTube akan diblokir oleh Mikrotik. Namun, pastikan bahwa setting firewall dikonfigurasi dengan benar dan sesuai dengan kebutuhan jaringan Kamu.

13. Cara Membuka Situs Yang Diblokir Mudah Tanpa Ribet gmna?


Cara paling mudah yaitu dengan merubah DNS di laptop/ PC, caranya:
Bukalah menu "Control Panel" pada PC atau laptop.
Kemudian pilih menu "Network and Internet".
Selanjutnya klik menu "Network and Sharing Center".
Langkah seterusnya yaitu "Local Area Connection".
Pilih "Properties".
Klik menu "Internet Protocol Version 4/ TCP IPv4".
Cari menu "Properties".
Ganti DNSnya dengan DNS gratis dari Google: (Preferred DNS Server : 8.8.8.8 - Alternate DNS Server : 8.8.4.4).
