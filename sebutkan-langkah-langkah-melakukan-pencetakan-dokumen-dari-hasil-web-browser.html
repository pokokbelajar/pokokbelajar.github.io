3. sebutkan langkah-langkah menyimpan sebuah halaman web yang ditampilkan di browser


Laman web adalah halaman utama dari suatu situs web yang diakses oleh pengguna pada awal masuk ke situs tersebut. Kata laman web berasal dari bahasa asing website, yaitu halaman web yang dapat diakses melalui halaman depan menggunakan sebuah browser dan juga jaringan internet.

Pembahasan

Langkah-langkah menyimpan sebuah laman web yang ditampilan di browser yaitu

Buka halaman web yang akan disimpan.Pada bagian atas jendela Firefox, klik pada menu FILE, lalu pilih SAVE PAGE AS. Kemudian Jendela Save As akan muncul.Gunakan jendela Save As untuk memilih lokasi penyimpanan halaman Web sesuai dengan yang dikehendaki. Pada panel dropdown Save As Type, pilih format penyimpanan halaman yang diinginkan.Klik SAVE.

Cara lainnya dengan adalah

Buka halaman web yang akan disimpan.Tekan kombinasi tombol Ctrl+S pada keyboard.Muncul jendela Save, pilih direktori untuk menyimpan halaman Web.Ketik nama file halaman WEB tersebut.Pilih format penyimpanan halaman.Klik SAVE.Pelajari lebih lanjut

4. sebutkan langkah langkah mencetak sebuah dokumen


Buka file yang akan diprint Klik Ctrl + P atau Office Button – Print untuk membuka kota dialog print Lakukan pengaturan : Name ~ memilih printer yang akan digunakan Number of Copies ~ menentukan jumlah salinan (copian) Page Range : All – mencetak semua halaman dokumen Current Page – mencetak halaman yang sedang aktif Selection – mencetak halaman yang sudah dipilih Pages – mencetak halaman tertentu (1,2,5,7 atau 1-5) Print : Print what – menentukan bagian dokumen yang akan dicetak All pages in range – mencetak semua halaman dokumen Odd pages – mencetak halaman ganjil dokumen (1,3,5 …) Even pages – mencetak halaman genap dokumen (2,4,6 …) Collate ~ jika dicentang akan mencetak keseluruhan dokumen kemudian rangkapannya (berdasar number of copies), dan jika tidak dicentang akan mencetak setiap halaman beserta rangkapannya langsung Properties ~ setting ukuran kertas, bentuk pencetakan dan pewarnaan Zoom ~ Page per sheet digunakan untuk memilih jumlah halaman dokumen yang akan dicetak dalam satu lembar dan Scale to paper size digunakan untuk memilih ukuran kertas Klik OK untuk memulai pencetakan.

22. Langkah-langkah yang benar untuk menyimpan hasil pencarian informasi pada program web browser adalah


Menyimpan hasil informasi dalam bentuk offline, dapat dilakukan dengan mendownload file html pada halaman website yang dikehendaki.
Tekan tombol kombinasi Control+S (Ctrl+S) pada tab halaman website tsb, kemudian simpan berkas dalam bentuk html.

Menyimpan dalam bentuk online, dalam artian dibutuhkan koneksi untuk membuka kembali informasi pada website tujuan, dapat dilakukan dengan mem-bookmarks halaman web yang dikehendaki.
Tekan tombol kombinasi Control+D (Ctrl+D) pada tab halaman website, kemudian simpan bookmarks untuk menyimpan halaman untuk dibuka di lain waktu.

28. Sebutkan langkah langkah menjalankan web browser mozila firefox menggunakan menu start


Langkah-langkah menjalankan web browser Mozilla Firefox menggunakan menu start yakni:

Mengklik gambar Windows yang berada di sisi kiri taskbar atau dapat dilakukan dengan menekan gambar Windows yang ada di keyboard (diantara Ctrl dan Alt atau Fn dan Alt pada beberapa versi keyboard).Jika sebelumnya saat melakukan instalasi atau beberapa pengaturan Firefox sudah dipin di start, maka ketika mengakses menu Start, akan langsung terlihat Firefox.Jika belum, maka dapat dilakukan pencarian menu start dengan cara : (1) Pada Windows 7 dapat dilakukan dengan melakukan dengan mengakses menu All Programs - Cari pada Folder Firefox / Shortcut bila tersedia dan dapat pula melalui pencarian di kolom bawah. (2) Pada Windows 8 dan 10,dapat dilakukan dengan kolom search yang tersedia di menu start maupun taskbar atau dengan mengakses menu Start - All Apps lalu cari Firefox.

Pembahasan

Menjalankan web browser Mozilla Firefox sebenarnya dapat dilakukan dengan beragam cara sesuai dengan kebutuhan pengguna. Ada beberapa cara yang dapat membuat pengguna menjalankan Mozilla Firefox lebih efisien yakni dengan membuat shortcut pada Desktop ketika instalasi maupun langsung dari master filenya, selain itu dapat juga dilakukan dengan melakukan 'Pin to Taskbar' maupun 'Pin to Start'. Selain terpasang di Windows, Mozilla Firefox juga tersedia di sistem operasi lain yakni Macintosh/Apple.
