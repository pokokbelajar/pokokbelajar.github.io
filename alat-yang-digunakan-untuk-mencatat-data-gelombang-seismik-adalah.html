4. sebuah gelombang primer tercatat di seismik pd pukul 18.15 dan gelombang sekunder tercatat pada pukul 18.19 Berapakah kisaran jarak antara stasium seismik dengan episentrum gempa


Diketahui :
tp = 18.15 ⇔ waktu gelombang primer
ts = 18.19 ⇔ waktu gelombang sekunder

Ditanyakan : Jarak episentrum (Δ) ?

Jawab :
Δ = ((ts - tp)-1) x 1000 km
= (18.19 - 18.15) -1) x 1000 km
= (4-1 dalam menit) x 1000 km
= (3dalam menit) x 1000 km
⇒ 3 menit = 180 sekon
1 menit = 60 sekon
= (180 ) x 1000 km : 60 = 3.000 km

Jadi, jarak ke lokasi permukaan (episentrum) adalah 3.000 km.

Δ = { (s-p)-I} M
s = gelombang sekunder
p = gelombang primer
I = konstanta (menit)
M = megameter (1000 km)

Δ = { (18.19 - 18.15) - 1 } 1000 km
= { 4 - 1 } 1000 km
= 3 . 1000
= 3000 detik/km

Jadi, 3000 detik/km :)

Jawaban:

Kisaran jarak stasiun seismik dengan Episentrum gempa adalah 3000 km.

_______________________

Pembahasan

Jarak stasiun seismik dengan Episentrum gempa Menggunakan Rumus:

[tex]\boxed{\texttt ∆ = ( ( S - P ) -1' ) \times 1000 \: km }[/tex]

Keterangan:

∆ = Jarak Episentrum Gempa dengan Stasiun pencatatS = Waktu terjadi gelombang SekunderP = Waktu terjadi gelombang Primer1000 km = Konstanta

++ Penyelesaian soal ++

Dik:

Waktu terjadi gelombang primer => 18.15Waktu terjadi gelombang Sekunder => 18.19

Dit;

Jarak episentrum gempa dengan stasiun seismik

Jawab:

∆ = [ ( S - P ) - 1' ] × 1.000 km

∆ = [ 18.19 - 18.15 ) - 1' ] × 1.000 km

∆ = [ 4' - 1' ] × 1.000 km

∆ = 3' × 1.000 km

∆ = 3.000 km

Jadi, Kisaran jarak stasiun seismik dengan Episentrum gempa adalah 3.000 km.

25. Sebuah gelombang primer tercatat di seismik pada pukul 18.15 dan gelombang sekunder tercatat pada pukul 18.19 berapakah kisaran jarak antara statiun seismik dengan episentrum gempa?


Mapel : IPS - GEOGRAFI
Kelas. : X SMP
Kategori : TENAGA ENDOGEN
Kata Kunci : Epicentrum gempa, menghitung jarak epicentrum gempa
Kode : -
============================


JAWABAN:

Diketahui:
gelombang primer tercatat pukul 18.15
gelombang sekunder tercatat pukul 18.19
Ditanya:
berapakah kisaran jarak antara statiun seismik dengan episentrum gempa?
Jawab:
Delta = ((S-P) – 1’) X 1.000 Kilometer)
= ((18.19 - 18.15) - 1') x 1000 km
= ((4') - 1') x 1000 km
= 3' x 1000 km
= 3.000 km

Jadi jarak epicentrum gempa tersebut dari stasiun pencatat gempa adalah 3.000 km.

------------------------------------------------------


xxx


PEMBAHASAN LEBIH LANJUT

Episentrum gempa adalah gelombang gempa yang berada di permukaan bumi dan tegak lurus dengan hiposentrum kemudian menyebar ke berbagai arah. Untuk menghitung jarak episentrum gempa dapat menggunakan metode episentral berikut:

METODE EPISENTRAL
Episentral ialah jarak episentrum atau pusat gempa di stasiun pencatat gempa. Untuk menentukan episentrum dengan menggunakan metode episentral diperlukan minimal tiga stasiun pengamat yang mencatat kejadian gempa, sehingga dapat dihitung jarak episentral masing-masing stasiun. Untuk menghitung jarak episentral digunakan rumus LASKA, sebagai berikut:

Delta = ((S-P) – 1’) X 1.000 Kilometer)

Ket:
Delta = jarak episentral dari stasiun pengamat dalam satuan kilometer
S = gelombang sekunder (dalam menit)
P = gelombang primer (dalam menit)
1’ = 1 menit = 60 detik
