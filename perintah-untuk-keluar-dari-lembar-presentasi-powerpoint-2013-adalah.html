13. Jelaskan perintah apa saja yang digunakan untuk menempatkan teks Microsoft Word ke slide presentasi Microsoft Powerpoint dan kegunaan dari masing-masing perintah.​


Ada kalanya di dalam mengoperasikan microsoft office kita akan mengkombinasikan kedua aplikasi agar bisa mendapatkan apa yang kita inginkan. Salah satunya adalah dengan memindahkan data dari Word ke Powerpoint. Berikut ini perintah yang digunakan untuk menempatkan teks Microsoft Word ke dalam slide presentasi Microsoft Powerpoint beserta kegunaan dari masing-masing perintah:

Kita dapat menggunakan perintah Salin dan Tempel untuk memindahkan teks dari Microsoft Word ke Microsoft Powerpoint. Pertama, salin teks yang ingin Anda pindahkan ke ms.powerpoint dengan mengklik kanan pada blok teks yang ingin Anda pindahkan dan pilih Salin. Kemudian buka Microsoft Powerpoint, pilih slide dan letakkan teks yang disalin, klik kanan dan pilih Tempel. Menggunakan perintah salin: salin teks untuk menempel.Gunakan Perintah Tempel: tempel teks yang disalin.

Pembahasan

Dalam beberapa kasus, kita mungkin perlu menggunakan materi di file Word untuk memberikan presentasi di PowerPoint. Ini biasanya dapat dilakukan dengan menggunakan alat online yang tersebar di seluruh web. Namun tahukah kita bahwa Microsoft sendiri memiliki tool online yang dapat digunakan untuk mengubah dokumen Word menjadi slide PPT. Microsoft telah meluncurkannya sejak awal tahun 2021.  Langkahnya adalah sebagai berikut:

Buka situs web Microsoft Office dan masuk dengan akun Microsoft kita. Buka dokumen Word yang ingin kita konversi dan pastikan file diunggah ke OneDrive.  Klik menu File di pojok kiri atas.  Klik opsi Ekspor. Klik opsi Ekspor ke Presentasi PowerPower. Klik Ekspor dan pilih template PPT yang diinginkan. Setelah mengklik opsi Open Presentation, tunggu beberapa saat.Selesai

Setelah proses selesai, tab PowerPoint akan terbuka dengan dokumen di Word. Kemudian kita dapat menyimpan file PPT ke penyimpanan lokal kita sesuai kebutuhan.
