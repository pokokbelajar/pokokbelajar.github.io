5. telnet merupakan fasilitas transfer data tertutup.arti dari tertutup pada telnet adalah


Telnet merupakan fasilitas transfer data tertutup.arti dari tertutup pada telnet adalah hanya bisa dilakukan antar sesama anggota yang sudah terdaftar. Jadi tidak semua orang bisa menggunakan telnet. Telnet sendiri merupakan salah satu aplikasi berbasis teks yang terdapat hampir pada semua sistem operasi. Untuk menggunakan telnet kita dapat mengaksesnya melalui command promt atau melalui terminal apabila kita menggunakan sistem operasi Linux.

Telnet memiliki tampilan mode teks jika yang terbiasa menggunakan terminal atau command prompt pasti tidak akan mengalami kendala, namun sebaliknya apabila sudah terbiasa menggunakan aplikasi dengan tampilan dengan mode grafis akan mengalami kesulitan ketika menggunakan telnet. Untuk sistem operasi windows, kita perlu menambahkan terlebih dahulu aplikasi Telnet Client melalui menu control panel agar bisa menggunakan telnet pada komputer kita.

Demikian pembahasan mengenai apa itu telnet dan maksud dari tertutup pada telnet, maksud tertutup adalah telnet dapat digunakan apabila sudah didaftarkan terlebih dahulu. Untuk membaca materi mengenai jaringan internet, jaringan komputer dan materi mengenai telnet dapat dibaca pada link berikut

7. antar muka yang di gunakan pada aplikasi telnet/ssh aksess jarK JAUH ADALAH


Antar muka yang digunakan pada aplikasi telnet atau ssh jarak jauh adalah menggunakan antar muka teks. Telnet merupakan kepanjangan dari Tellecomunication Network. Telnet merupakan salah satu komunikasi antar komputer. Antar muka teks adalah antar muka yang hanya berisikan teks dan biasanya hanya menggunakan keyboard sebagai inputannya. Tampilannya mirip dengan command prompt pada windows atau terminal pada lInux.

Seiring dengan perkembangan teknologi, sekarang apabila kita ingin mengakses komputer lain menggunakan telnet atau ssh kita dapat menggunakan aplikasi yang bernama putty. Aplikasi ini merupakan aplikasi yang cukup populer. Namun sebenarnya akses menggunakan telnet masih dapat menggunakan command prompt pada windows atau untuk pengguna linux dapat menggunakan terminal untuk mengakses aplikasi telnet.

Dengan menggunakan telnet seseorang dapat mengakses komputer lain tanpa harus berhadapan fisik dengan komputer yang diakses, atau bisa disebut dengan remote. Namun apabila menggunakan telnet harap berhati-hati karena keamanan masih kurang. Solusi dari permasalahan tersebut adalah menggunakan SSH. Dengan SSH keamanan akses remote komputer  menjadi lebih terjamin keamanannya.

8. Sebutkan kegunaan telnet, ftp, finger, dan ping ?


Telnet yaitu sebuah perintah yg digunakan untuk meremote suatu network atau juga komputer.
FTP atau File Transfer Protocol adalah suatu protocol yang berfungsi untuk saling berbagi atau sharing data maupun file.
Ping berfungsi untuk mengetahui apakah suatu komputer sudah saling terhubung, atau dengan kata lain yaitu untuk menguji koneksi antar komputer maulun network. Ping bekerja dengan cara menirim paket ICMP kepada alamat tujuan.
