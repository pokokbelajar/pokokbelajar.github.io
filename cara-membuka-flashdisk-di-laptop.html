3. Ada yang tau gimana caranya agar flashdisk bisa kebaca di laptop ?


Uh... langsung ditancep ke usb port aja (slot usb yg ada di pinggiran laptop).
Kalau belum kebaca, coba digerak-gerakin, atau agak ditarik sedikit flashdisknya (beberapa komputer punya port yg agak rewel)
Kalau masih belum kebaca, coba deh cek port usb nya support usb 3.0 atau nggak, karena beberapa keluaran flashdisk terbaru suka ga kebaca di laptop lama. Kalau udah gitu, kamu bisa buka di komputer atau laptop lain.Sambungkan Flashdisk ke port USB di bagian samping laptop kamu/anda/engkau. Biasanya akan berlangsung lama dalam penginstallan kalau memang itu Flashdisk baru. Kalau Flashdisk tidak bermasalah, maka akan langsung terbaca oleh komputer secara otomatis.
Apabila tidak terbaca/tidak terdeteksi, bisa ikuti langkah berikut;
1. Pasangkan Flashdisk yang tak terbaca ke port USB laptop
2. Arahkan mouse ke Start - Run - ketik "cmd.exe" tanpa kutip - enter
3. ketik "CHKDSK/fH:" tanpa kutip - enter
catatan : "CHKDSK/"f"H f disini adalah format flashdisk. Jika flashdisk berformat G, maka rubah menjadi "CHKDSK/gH. Jika E, ubah jadi "CHKDSK/eH" dan seterusnya.

jika tidak bisa menggunakan Command Prompt, bisa menggunakan cara format;
1. Pasangkan Flashdisk yang tak terbaca ke port USB laptop
2. Klik Start - Run - "compmgmt.msc" tanpa kutip - enter
3. Klik program tersebut
4. Pilih storage yang ada di tengah kemudian pilih disk management.
5. Setelah Flashdisk terbaca, klik kanan, pilih create partition.
6. Lalu format flashdisk
(source: wantekno.com)

6. ronald membeli laptop dan flashdisk di sebuah toko yang sedang mengadakan promo. laptop tersebut diberi diskon 18% dan flashdisk 10%.jika harga laptop RP. 13.700.000,00 dan flashdisk RP150.000,00,maka uang yang harus dibayarkan Tania adalah..​


Jawaban:

Rp11.369.000

Penjelasan dengan langkah-langkah:

LAPTOP

Harga laptop Rp13.700.000

Diskon laptop 18%

Harga setelah diskon = harga awal × (100% - diskon)

= Rp13.700.000 × (100% - 18%)

= Rp13.700.000 x 82%

= Rp11.234.000

FLASHDISK

Harga flashdisk Rp150.000

Diskon flashdisk 10%

Harga setelah diskon = harga awal × (100% - diskon)

= Rp150.000 × (100% - 10%)

= Rp150.000 × 90%

= Rp135.000

Uang yang harus dibayarkan Tania

= harga laptop sesudah diskon + harga flashdisk sesudah diskon

= Rp11.234.000 + Rp135.000

= Rp11.369.000

Jadi, uang yang harus dibayarkan Tania sebesar Rp11.369.000

13. Bagaimana cara mengembalikan file yang tidak bisa dibuka pada flashdisk?


Tahan tombol SHIFT di komputer/laptop dan masukkan flashdisk. Menekan dan menahan tombol SHIFT adalah untuk pengamanan saja, memastikan agar tidak ada program di flashdisk yang langsung dijalankan tanpa sepengetahuan pengguna (autorun.inf)Setelah itu hapus semua file dan folder yang merupakan shortcut (silahkan melihat kolom TYPE di windows Explorer). Pastikan yang dihapus adalah shortcut, dan jangan sampai salah melakukan klik dua kali (membuka shortcut) tersebut.Untuk mengembalikan semua data yang disembunyikan virus, bisa menggunakan program buatan saya Hidden File Tool atau langsung melalui Command Prompt. Silahkan memilih salah satu atau jika dengan Hidden File Tool ada yang tidak berhasil ditampilkan (misalnya nama file menggunakan huruf unicode: Arabic, China, Jepang dll) bisa ditambahkan menggunakan Command Prompt. Menggunakan Hidden File Tool, klik tombol Browse dan pilih drive flashdisk. Selanjutnya klik tombol -=SEARCH=-. Setelah hasilnya ditampilkan, pilih semua semuanya (klik file yang paling atas, tekan dan tahan SHIFT dan klik file yang terakhir). Selanjutnya klik tombol APPLY, sehingga semua file dan folder di flashdisk akan muncul semuaMenggunakan Command Prompt. Buka Command Prompt melalui Start Menu > Run dan tulis atau ketikkan:CMD lalu Enter. Atau melalui Start Menu > All Programs > Accessories > Command Prompt. Selanjutnya ketikkan huruf yang merupakan drive flashdisknya. Misalnya flashdisk ada di drive E, maka ketikkan E: (E diikuti tanda titik dua) dan tekan Enter. Setelah berhasil tampil E:\> di layar Command prompt, ketikkan perintah berikut: ATTRIB -S -H -R *.* /S /D kemudian tekan Enter.

16. Mengapa setelah dimasukkan ke dalam flashdisk video tidak bisa dibuka?


Jawaban:

Kemungkinan pertama: flashdisk tersebut ada virusnya. Sehingga file video mengalami kerusakan dan tidak bisa dibuka (diputar).

Kemungkinan kedua: file video tidak dikenali formatnya. Misalnya format videonya adalah FLV, sedangkan di komputer hanya ada program AVI player dan MKV player.

Kemungkinan ketiga: tidak ada program pemutar video sama sekali di komputer yang akan digunakan untuk membuka file video dalam flashdisk.

Penjelasan:

Solusi kemungkinan pertama adalah melakukan scan pada flashdisk dengan antivirus.

Solusi kemungkinan kedua yaitu mengkonversi format video dengan format yang dikenali (misalnya file video FLV dikonversi menjadi file video AVI).

Solusi kemungkinan ketiga ialah menginstal program pemutar video di komputer yang akan dipakai untuk membuka file video dalam flashdisk (contohnya program ALL Player).

20. Edi mempunyai flashdisk dengan kapasitas 1024 MB (1 Gigabyte) dengan beberapa data seperti di Gambar.••Jika Edi ingin menambahkan 400 MB file gambar. Maka salah satu file paling besar dalam flashdisk tersebut ingin dipindahkan ke laptop. File manakah yang harus dipindahkan ke laptop itu?​


File Yang Harus Dipindahkan Ke Laptop Adalah Musik

=> Penyelesaian:

Jumlah File Edi Didalam Flashdisk =

= 350 + 150 + 500

= 500 + 500

= 1.000 MB

Jumlah Kapasitas Yang Diperlukan Agar File Gambar Dapat Ditambah =

= 1.024 - 1.000 + 400

= 24 + 400

= 424

Jumlah Yang Lebih Besar Dari 424 MB =

424 > 350

424 > 150

424 < 500

Jadi, File Yang Perlu Dipindahkan Ke Laptop Adalah File Musik

30. Tania membeli laptop dan flashdisk di sebuah toko yang sedang mengadakan promo, Laptop tersebut diberi diskon 18% dan flashdisk 10%. Jika harga laptop Rp3.400,000,00 dan flashdisk Rp150.000,00, maka uang yang harus dibayarkan Tania adalahTOLONG YA JAWABNYA PAKE CARA DAN YG SERIUS​


Jawaban singkat:

Uang yg harus dibayarkan Tania adalah Rp 2.923.000

Penjelasan dg langkah²:

Harga Diskon laptop = 18% x Rp3.400.000,00

Harga Diskon laptop = [tex]\frac{18}{100}[/tex] x Rp3.400.000,00

Harga Diskon laptop = Rp 612.000,00

Harga Diskon flashdisk = 10% x Rp 150.000,00

Harga Diskon flashdisk = [tex]\frac{10}{100}[/tex] x Rp 150.000,00

Harga Diskon flashdisk = Rp 15.000,00

Uang yg harus dibayarkan Tania untuk beli laptop adalah

(Rp3.400.000,00 - Rp 612.000,00)

= Rp 2.788.000,00

Uang yg harus dibayarkan Tania untuk beli flashdisk adalah

(Rp 150.000,00 - Rp 15.000,00)

= Rp 135.000,00

Total uang yg harus dibayarkan Tania ialah sebagai berikut

Rp 2.788.000,00 + Rp 135.000,00

= Rp 2.923.000
