5. jelaskan langkah-langkah untuk menyimpan dokumen microsoft excel!


1. Klik Office Button, lalu pilih menu Save atau Save As.
– Save : menyimpan dokumen baru atau dokumen lama dengan nama yang sama.
– Save As : menyimpan dokumen baru yang belum pernah disimpan atau dokumen lama dengan nama baru.
2. Jika kamu menyorot ikon Save As maka akan muncul kotak pilihan format penyimpanan data. Pilih Excel Workbook atau kamu dapat langsung mengklik ikon Save As, maka akan muncul kotak dialog Save As.
3. Tentukan folder tempat penyimpanan dokumen kamu pada kotak isian Save In dengan acara klik tombol panah kecil di ujung lalu pilih folder yang diinginkan.
4. Ketikkan nama dokumen kamu pada kotak File Name.
5. Isilah kotak isian Save as type dengan Excel Workbook.
6. Klik tombol Save

12. Jelaskan langkah langkah menyimpan dokumen ms power point


Jelaskan langkah langkah menyimpan dokumen ms power point

Jawab:

Untuk menyimpan dokumen di program komputer untuk membuat presentasi microsoft power point, kita perlu untuk menggunakan fungsi save. Untuk menjalankan fungsi save di program microsoft power point ada beberapa cara yang bisa kita lakukan.

1. Pilih menu file, lalu klik tombol save  yang memiliki ikon berbentuk disket.

2. Pada dokumen Anda, tekan CTRL + S untuk menyimpan dokumen Anda.

3. Pada menu toolbar, cari dan klik ikon berbentuk disket berwarna biru.

JIka Anda belum menyimpan dokumen Anda sebelumnya, program akan memunculkan dialog save as, disini Anda hanya perlu memilih lokasi file disimpan lalu isikan nama file dan klik tombol save.
