Penyelesaian dari SPLDV [tex]\rm 5x-4y=33 [/tex] dan [tex]\rm 3x+5y=-5 [/tex] adalah Hp = { [tex]\rm \dfrac{ 145 }{ 37 } [/tex] , [tex]\rm -\dfrac{ 124 }{ 37} [/tex] }

↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓

PENDAHULUAN

Sistem Persamaan Linear Dua Variabel atau SPLDV adalah sistem persamaan yang memuat dua variabel dan pangkat tertinggi dari variabelnya adalah satu. Variabel digunakan untuk memisalkan suatu bentuk yang terlalu panjang.

Bentuk umum SPLDV

[tex] \begin{gathered}\left\{\begin{matrix} ax+by=c\\\\px+qy=r\end{matrix}\right.\end{gathered} [/tex]

x dan y = variabela,b,p,q = koefisienc,r = konstanta.

Ada juga beberapa metode yang dapat kita gunakan untuk menyelesaikan SPLDV yaitu

1. Metode Grafik

Dengan menggambarkan kedua persamaan dalam bentuk grafik pada bidang kartesius, kemudian mencari letak titik potong kedua garis.

2. Metode Eliminasi

Dengan mengeliminasi atau menghilangkan satu variabel, sehingga variabel lainnya dapat ditemukan. Biasanya dilakukan dengan mengurangi atau menjumlahkan kedua persamaan.

3. Metode Subsitusi

Dengan mengganti atau memasukkan nilai suatu variabel pada suatu persamaan dari persamaan lainnya.

4. Metode Campuran

Dengan menggunakan kedua cara eliminasi dan subsitusi. Caranya dengan menggunakan metode eliminasi untuk menentukan nilai dari x kemudian x nya disubsitusikan pada persamaan lainnya untuk menentukan nilai y, atau sebaliknya.

PEMBAHASAN

Persamaan 1

[tex]\rm 5x-4y=33 [/tex]

[tex]\rm 5x=4y+33 [/tex]

[tex]\rm x=\dfrac{ 4y+33 }{5 } [/tex]

Subsitusi ke persamaan 2

[tex]\rm 3x+5y=-5 [/tex]

[tex]\rm 3(\dfrac{ 4y+33 }{5 } )+5y=-5 [/tex]

[tex]\rm \dfrac{ 3(4y+33 )}{5 } +5y=-5 [/tex]

[tex]\rm \dfrac{ 12y+99}{5 } +5y=-5 [/tex]

Kedua ruas dikalikan 5 menjadi

[tex]\rm 12y+99+25y=-25 [/tex]

[tex]\rm 37y=-25-99 [/tex]

[tex]\rm 37y=-124 [/tex]

[tex]\rm y=-\dfrac{ 124 }{ 37} [/tex]

Subsitusi kembali ke Persamaan 1

[tex]\rm x=\dfrac{ 4y+33 }{5 } [/tex]

[tex]\rm x=\dfrac{ 4(-\frac{ 124 }{ 37} )+33 }{5 } [/tex]

[tex]\rm x=\dfrac{ -\frac{ 496 }{ 37} +33 }{5 } [/tex]

[tex]\rm x=\dfrac{ 145 }{ 37 } [/tex]

Kesimpulan :

Jadi, Penyelesaian dari SPLDV [tex]\rm 5x-4y=33 [/tex] dan [tex]\rm 3x+5y=-5 [/tex] adalah Hp = { [tex]\rm \dfrac{ 145 }{ 37 } [/tex] , [tex]\rm -\dfrac{ 124 }{ 37} [/tex] }
