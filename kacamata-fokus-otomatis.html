7. seorang penderita miopi menggunakan kacamata berkekuatan -0,5 dioptri. berapa fokus lensa kacamata tersebut?tolong bantu jawab ​


Kuat Lensa ( P ) : - 0,5 Dioptri

Fokus Lensa ( PR ) : ?

Fokus Lensa ( cm ) = - 100 / PR

- 0,5 Dioptri = - 100 ( cm )/ PR

PR = - 100 ( cm ) / - 0,5 Dioptri

PR = - 100 ( cm ) ÷ 5 / 10 Dioptri

PR = - 100 ( cm ) × 10 / 5 Dioptri

PR = 200 cm

PR = 2 m

Yap, mungkin aja jawabannya emang betul C.

Kita coba pake Rumus ya,
Rumus kekuatan Lensa => 100/F
F itu titik fokus

Di opsi C, dikatakan bertitik fokus -200 cm

Gimana ? Benar kan?
