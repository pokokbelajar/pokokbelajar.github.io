→ r = 2√3 cm

s = 2√3 cm

21. kubus mempunyai sisi=25cm,maka diagonal ruang =...cm


bangun ruang kubus

diagonal sisi ( d ) :
d² = s² + s²
d² = 2s²
d² = 2(25)²
d² = 2(625)
d² = 1250

diagonal ruang ( D ) :
D² = d² + s²
D² = 1250 + (25)²
D² = 1250 + 625
D² = 1875
D = √1875
D = √(625 × 3)
D = 25√3 cm

semoga membantu
diagonal ruang = sisi√3
diagonal ruang = 25√3 cm

semoga membantu ya
