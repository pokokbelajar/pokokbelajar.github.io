1. 1. bagaimana suhu pada kedua beker gelas, sebelum bunsen (pembakar spiritus) dinyalakan?​


Jawaban:

dingin atau sama dengan suhu ruang

Penjelasan:

Bunsen (pembakar spiritus) adalah salah satu alat laboratorium yang sangat penting kegunaanya. Biasanya bunsen digunakan untuk menguji zat kimia yang membutuhkan proses pemanasan/pembakaran. Bunsen terdiri dari tabung logam vertikal terhubung ke sumber bahan bakar gas (burner tube), dengan lubang pemasukan udara (collar). Udara akan masuk ke dalam tabung dan bercampur dengan gas untuk membuat api yang sangat panas.

Bunsen digunakan untuk menghasilkan tiga jenis api, yaitu:

Api kuning/ oranye. Dikenal sebagai api keselamatan dan diperkirakan suhunya mencapai 300 derajat Celcius. Api biru. Suhu sekitar 500 derajat celcius. Api biru menderu. Merupakan api yang paling umum digunakan dalam pembakar bunsen karena api yang dihasilkan sangat panas. Suhu yang didapat mencapai 700 drajat celsius.
