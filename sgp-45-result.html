6. berapa resultan vektor A=12n dan b =6 n,jika sudut apitnya 45°


Rumus untuk mencari resultan adalah

R = √(A² + B² + 2. A. B. cos θ), masukkan nilai-nilai yang diketahui

R = √((12 N)² + (6 N)² + 2. (12 N). (6 N). cos 45°)

R = √(144 N² + 36 N² + 2. 72 N². 0,5√2)

R = √(180 N² + 72√2 N²)

R = √(180 N² + 101,8234 N²)

R = √(281,8234 N²)

R = 16,7876 N

Jadi, besar resultan vektornya adalah 16,7876 N.

18. dua buah vektor yang besarnya 8 dan 15 satuan saling mengapit dengan sudut 45. hitung besar resultannya dan sudut antara resultan dengan vektor pertama


PENDAHULUAN
• Vektor
→ resultan

V₁ = 15 satuan
V₂ = 8 satuan
α = 45°
R = __?
α₁ = __?

lihat gambaran permasalahan (terlampir)

Besar resultan
R² = V₁² + V₂² + 2 V₁ V₂ cos α
R² = 15² + 8² + 2 • 15 • 8 • ½√2
R² = 225 + 64 + 120√2
R = √(289 + 120√2) ✓
R ≈ 21,4 satuan

Arah resultan
V₂ / sin α₁ = R / sin α
8 / sin α₁ = 21,4 / sin 45°
sin α₁ = 8 sin 45° / 21,4
α₁ ≈ 15,3° ✓
(sudut antara R dengan V₁)

19. jika F=10N dan sudut apit sebesar 45° hitung resultan dari vektor tersebut​


Diketahui:

F = 10N

θ = 45°

Ditanya:

Resultan (R)

Jawaban:

Fx = F.cosθ

= 10.½√2

= 5√2 N

Fy = F.sinθ

= 10.½√2

= 5√2 N

R = √Fx² + Fy²

= √(5√2)²+(5√2)²)N

= √50N+50N

= √100N

= 10 N

Maka,resultan dari vektor tersebut adalah 10N.
