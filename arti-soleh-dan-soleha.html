Penjelasan dengan langkah-langkah:

a = besar tabungan mula-mula

b = besar bunga selama setahun

b = 6% x a

c = besar bunga selama 10 bulan

c = 10/12 x b

c = 10/12 x 6/100 x a

c = 60a/1200

c = a/20

a + c = Rp2.100.000

a + a/20 = Rp2.100.000

20a + a = Rp2.100.000 x 20

21a = Rp42.000.000

a = Rp42.000.000/21

a = Rp2.000.000

10. soleha menyimpan yang Di bank dressed 8000.000 bang memberikan bungalow 3% pertahaun setelah beberapa Bulan soleha mengambil uang tersebut jika tabungan akhir soleha menjadi 8.720.000 maka berapa lama soleha menyimpan uangnya .berikan Cara Dan jawabannya? ​


Jawaban:

##

uang awal = 8.000.000

bunga = 3% pertahun = 3/12 % per bulan

uang akhir = 8.720.000

#

berapa lama menabung = n?

#

n = jumlah bulan

uang akhir = uang awal + (uang awal x n x persen per bulan)

8.720.000 = 8.000.000 + (8.000.000 x n x 3/12 %)

8.720.000 - 8.000.000 = 8.000.000 x n x 3/1200

720.000 = n x 8.000.000 x 1/400

n = 720.000/20.000

n = 16 bulan

##
