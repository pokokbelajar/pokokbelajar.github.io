27. berapa kg SO3 dari pengalahan yqng diberikan SO2 berlebih dengan 50gr O2


massa O2 = 50 gr
Mr O2 = 32
Mr SO3 = 80
massa SO3 = ?

---

SO2 + 1/2 O2 --> SO3

mol O2 = massa / Mr = 50 gr / 32 = 1,5625

mol SO3 = 2 x mol O2 = 2 x 1,5625 = 3,125 mol

massa SO3 = mol x Mr = 3,125 x 80 = 250 gram = 0,25 kg
