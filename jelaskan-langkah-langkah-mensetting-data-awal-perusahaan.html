1. jelaskan langkah langkah men setting data awal perusahaan


1. Pilih Create, kemudian akan tampil gambar sebagai berikut.


2. Klik next saja, karena itu adalah informasi awal program Myob.

3. Langkah berikutnya adalah isi data perusahaan, termasuk nama Perusahaan, alamat, No. Telepon, Fax dam Email, lalu klik Next.


4. Langkah berikutnya adalah setting periode Akuntansi.


Untuk setting diatas, anda harus pastikan settingan tahun benar yaitu 2008 dan settingan bulan juga harus tepat yaitu di akhiri bulan desemberpada Last month of financial (bulan akhir periode akuntansi) serta bulanJanuari pada Conversion Month (bulan awal periode akuntansi). Karena kalau anda sala, makapada waktu menjurnal nantinya akan berpengaruh. Kemudian klik Next.

5. Maka akan tampil informasi yag telah di set tadi. Pada tahap ini cukup klikNext saja.

6. Pada langkah ini anda diperlihatkan 3 pilihan untuk memilih Perkiraan.
Pilihan pertama = bila menggunakan perkiraan dari program Myob
Pilihan kedua = bila ingin mengimpor perkiraan dari perusahan lain / file lain
Pilihan ketiga = bila ingin membuat sendiri perkiraan tersebut.

Anda boleh memilih apa saja, karena nantinya akan anda set juga perkiraannya, standarnya pilih saja No. 1 lalu klik Next.

7. Kemudian silahkan jenis Perusahaan anda, dan type bisnis dari perusahaan tersebut. Ini pun boleh anda abaikan, lalu klik aja Next.

8. Pada langkah ini tentukan tempat penyimpanan file anda.


Setelah itu, silahkan klik Next. maka tampil proses dari settingan awal file anda.

9. Kemudian pada pilihan terakhir , pilihCommand Center.

10.Setelah anda klik Command Center maka akan muncul tampilan awal file anda seperti gambar dibawah ini :



Demikian settingan awal dalam membuat file data Myob.

10. jelaskan langkah langkah untuk mensetting akun pajak perusahaan pada myob accounting plus versi 18​


Langkah langkah untuk mensetting akun pajak perusahaan pada myob accounting plus versi 18:

1. Tekan List yang ada di toolbar MYOB Accounting+ Versi 18, lalu pilih Tax Codes.2. Delete tax code FRE GST Free di jendela Tax Code List dengan klik kanan pada tax codes FRE GST Free, lalu klik Delete Tax Code.3. Lakukan editing pada tax code GST dengan mengklik edit, lalu tulis seperti ini:

( Tax code : PPN.

( Description : Pajak Pertambahan Nilai.

( Tax type : Goods & Service Tax.

( Rate : 10%.

( Linked Account for Tax Collected : Vat Outcome.

( Linked Account for Tax Paid : Vat Income.

Klik OK.

4. Lakukan editing pada tax code N-T dengan mengklik Edit, lalu tulis seperti ini:

( Description : Tidak kena pajak.

( Linked Account for Tax Collected : Vat Outcome.

( Linked Account for Tax Paid : Vat Income.

Klik OK.

5. Klik Close.

Pembahasan:

MYOB ialah suatu tecno-acccunting yang berkepanjangan Mind Your Own Bussines yang difungsikan untuk mempermudah menulis data keuangan suatu perusahaan. Dalam MYOB ini, terkandung banyak fitur seperti halnya fitur membuat data pajak. Berikut langkah-langkahnya:

1. Tekan List yang ada di toolbar MYOB Accounting+ Versi 18, lalu pilih Tax Codes.

2. Delete tax code FRE GST Free di jendela Tax Code List dengan klik kanan pada tax codes FRE GST Free, lalu klik Delete Tax Code.

3. Lakukan editing pada tax code GST dengan mengklik edit, lalu tulis seperti ini:

( Tax code : PPN.( Description : Pajak Pertambahan Nilai.( Tax type : Goods & Service Tax.( Rate : 10%.( Linked Account for Tax Collected : Vat Outcome.( Linked Account for Tax Paid : Vat Income.

Klik OK.

4. Lakukan editing pada tax code N-T dengan mengklik Edit, lalu tulis seperti ini:

( Description : Tidak kena pajak.( Linked Account for Tax Collected : Vat Outcome.( Linked Account for Tax Paid : Vat Income.

Klik OK.

5. Klik Close.

14. 3. Jelaskan langkah-langkah dalam setting environment!


Caranya pertama-tama kita buka File Explorer di Windows, kemudian klik-kanan pada My Computer untuk pengguna Windows 7 kebawah atau untuk OS Windows 8 keatas namanya This PC dan pilih Properties.


Selanjutnya Pilih Advanced System Settings.


Pada System Properties, pilih tab Advanced klik tombol Environment Variables


Pada Environment Variables, kita akan memilih/klik System Variable Path, dan klik Edit


System Variable pada Path dapat berbeda pada masing-masing OS kita dikarenakan kebutuhan aplikasi yang terinstal juga berbeda-beda. Namun perlu diperhatikan, pada saat melakukan Edit Jangan Pernah Menghapus Value Yang Sudah Ada karena akan mempengaruhi system anda. Kita hanya perlu menambahkan value yang dengan cara menambahakan tanda titik-koma kemudian direktori yang kita inginkan yaitu:

;D:\xampp\php



Masih pada Edit System Variable, klik OK. Maka dengan ini setting path php.exe sudah selesai.


Cara menggunakannya hanya tinggal menggetikkan

php <perintah_anda> 



dan aplikasi php.exe anda sudah bisa jalan

Dalam hal ini extensi php adalah .exe dan valuenya sudah terdaftar pada Environment Variable PATHEXT.


Sama halnya ketika anda ingin menggunakan aplikasi Composer. Ketika aplikasi Composer sudah terinstal maka path direktorinya terletak di dalam Web Directory Project anda. Dalam hal ini direktori Project saya terletak di D:\xampp\htdocs dan nama aplikasi Composer itu sendiri adalah composer.phar berarti extensinya adalah .phar


Lakukan pendaftaran variable dan jangan lupa menambahkan tanda titik-koma ";" sebelum menuliskan value yang diinginkan.

Untuk value Path ditambahkan dengan:


;D:\xampp\htdocs



kemudian untuk value PATHEXT untuk extensi ditambahkan dengan:



Sampai disini settingnya sudah selesai. Dan ketika kita akan menggunakan aplikasi PHP dan Composer secara bersamaan hanya semudah mengetikkan:
php composer <perintah_anda>

27. Tuliskan langkah-langkah setting IP​


Telah terdapat langkah-langkah untuk setting IP yaitu ialah sebagai berikut:

Masuk ke dalam Control Panel lalu setelah itu pada menu Network and Internet pilihlah view network stataus and task. Kemudian, dapat diarahkan menuju ke halaman network and sharing center. Pada panel menu sebelah kiri pilihlah change adapter setting. Setelah itu, pilihlah local area connection klik kanan dan pilihlah properties. Setelah itu, pilihlah internet protocol version 4 TCP/IPV4 setelah itu klik pada properties dan berilah ceklist pada opsi use the following IP address. Kemudian form ini telah diisi dengan menggunakan IP address yang telah diinginkan dan klik OK.Pembahasan

Pengertian dari IP address yaitu ialah sebuah alamat protokol internet ataupun alamat IP yang telah mengidentifikasikan segala perangkat yang telah terhubung ke jaringan baik pada jaringan internet pada lokal maupun umum. Selanjutnya, apabila diibaratkan IP address ini sendiri merupakan nomor rumah yang di mana telah memiliki fungsi untuk dapat memastikan bahwa paket data telah dikirimkan ke rumah ataupun perangkat yang tepat.

Dengan kata lain, fungsi dari IP address ini sendiri telah menjadi media komunikasi atas sebuah perangkat agar permintaan untuknya tersebut diarahkan kepada tujuan yang tepat dengan melalui jaringan. IP address konsumen juga telah dibagi menjadi empat macam jenis yaitu ialah statis, dinamis, publik, dan juga privat. Kemudian, juga terdapat jenis IP address website yaitu dedicated dan juga shared.
