Jawaban:

Seorang pedagang akan berjualan kaos katun dan kaos nylon. Modal yang tersedia hanya Rp6.000.000,00, harga beli kaos katun Rp40.000,00 per potong dan kaos nylon Rp80.000,00 per potong. Toko tersebut hanya mampu menampung tidak lebih dari 100 potong kaos. Keuntungan untuk setiap penjualan satu potong kaos katun dan 1 potong kaos nylon berturut-turut adalah Rp12.000,00 dan Rp16.000,00. Keuntungan akan maksimum jika kaos nylon terjual sebanyak 50 potong. Soal ini merupakan penerapan materi program linear.

Pembahasan

Misal

x = kaos katun

y = kaos nylon

Katun (x) Nylon (y)

Harga beli 40.000x 80.000y 6.000.000

Daya tampung x y 100

Keuntungan 12.000x 16.000y …. ?

Model matematika

40.000x + 80.000y ≤ 6.000.000 ⇒ x + 2y ≤ 150

x + y ≤ 100

x ≥ 0, y ≥ 0

Fungsi sasaran:

F(x, y) = 12.000x + 16.000y

Menggambar grafik

1) x + 2y ≤ 150

Jika x = 0 maka y = 75 ⇒ (0, 75)

Jika y = 0 maka x = 150 ⇒ (150, 0)

Hubungkan titik (0, 75) dan (150, 0) sehingga diperoleh garis x + 2y ≤ 150 dan arsir ke bawah

2) x + y ≤ 100

Jika x = 0 maka y = 100 ⇒ (0, 100)

Jika y = 0 maka x = 100 ⇒ (100, 0)

Hubungkan titik (0, 100) dan (100, 0) sehingga diperoleh garis x + y ≤ 100 dan arsir ke bawah

Titik potong kedua garis

x + 2y = 150

x + y = 100

-------------- –

y = 50

Substitusikan ke persamaan (2)

x + y = 100

x + 50 = 100

x = 50

Jadi titik potong kedua garis tersebut adalah (50, 50)

Setelah kita gambar, diperoleh titik-titik sudutnya adalah (0, 75), (100, 0) dan (50, 50)

Substitusikan ketiga titik tersebut ke fungsi sasaran yaitu

F(x, y) = 12.000x + 16.000y

F(0, 75) = 12.000(0) + 16.000(75) = 0 + 1.200.000 = 1.200.000

F(100, 0) = 12.000(100) + 16.000(0) = 1.200.000 + 0 = 1.200.000

F(50, 50) = 12.000(50) + 16.000(50) = 600.000 + 800.000 = 1.400.000

Jadi keuntungan maksimum yang diperoleh adalah Rp1.400.000,00 di titik (50, 50) artinya terjual 50 potong kain katun dan 50 potong kain nylon.

Sehingga keuntungan akan maksimum jika kaos nylon terjual sebanyak 50 potong
