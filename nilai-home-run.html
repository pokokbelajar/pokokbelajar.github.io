24. Bagaimana cara mendapatkan nilai home run dalam permainan rounders


Tiap pemain berhak melakukan pukulan sebanyak 3 kali.
Setelah berhasil memukul, pemukul harus berlari menuju tiang hinggap berikutnya dengan membawa serta tongkat pemukulnya.
Jika pukulan ketiga tidak berhasil, pemukul harus berlari menuju base berikutnya.
Tiap pemukul akan mendapat nilai satu untuk atas setiap base yang dilewatinya.
Setiap pemukul yang dapat kembali ke ruang bebas tanpa pernah dimatikan oleh tim penjaga akan mendapat nilai 5.
Jika berhasil memukul bola dengan baik, kemudian melewati semua base kembali ke ruang bebas atas pukulannya sendiri, maka pemukul akan mendapat nilai 6. Peristiwa ini disebut homerun.

30. Change into negative the children run home. a.the children run not home. b.the children doesn't run home. c.the children did not run home. d.the children did not runned home.


Jawaban:

C. the children did not run home

karna itu past tense dan V2 dari run itu tetap run

jadinya

the children didn't run home

Jawaban:

Present tense

Penjelasan:

(+) The children run home

(-) The children don't run home

(?) Do the children run home ?

Jawaban yang tepat: ⤵️

The children don't run home

Alasan :

1. Noun merupakan bentuk plural dari child.

2. Noun yang bersifat plural atau jamak tidak mengalami penambahan s/es pada kata kerja (verb)

3. Verb pada awal kalimat merupakan verb 1. Yaitu RUN

4. Run merupakan irregular verb ( Run, Ran, Run)

5. Kalimat tersebut tidak termasuk pada past Tense sebab, pada awal soal tidak terjadi perubahan verb ke bentuk ke-2.

Jawaban a tidak tepat karena not harus diiringi dengan kata bantu . seperti do, will etc

Jawaban b kurang tepat karena children adalah bentuk jamak dan Dan jamak menggunakan DO bukan DOES

Jawaban c dan d tidak tepat karena pada soal tidak terjadi perubahan kata kerja ke bentuk ke - 2. Sehingga kalimatnya termasuk present tense
