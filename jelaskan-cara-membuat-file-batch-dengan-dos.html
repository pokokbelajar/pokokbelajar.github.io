1. Jelaskan cara membuat file batch dengan dos​


Jawaban:

Ketika sebuah batch file dijalankan,  programshell yang berupa command.com akan membaca file batch dan kemudian mengeksekusi perintahnya, secara baris demi baris. Batch files berguna untuk menjalankan rangkaian dari dari beberapa perintah yang bisa dieksekusi (executables) secara otomatis dan  sering digunakan oleh sistem administrator untuk mengotomatisasi banyak proses secara bersamaan. Perintah-perintah batch file bersifat case insensitive jadi tidak ada perbedaan maksud  antara huruf besar dengan huruf kecil.

Dalam DOS batch command, perintah-perintah seperti Copy, Dir, Delete disebut dengan perintah internal sedangkan perintah semisal Format,Undelete,Backup merupakan bagian dari perintah eksternal. Setiap kita menjalankan perintah DOS batch file baik internal ataupun eksternal, command.com akan langsung menjalankan peritah internal atau memanggil program eksternal yang terpisah dan menjalankan perintah tersebut untuk kemudian mengembalikan hasilnya dalam bentuk perintah eksternal.

7. berikut file dalam dos yang berfungsi sebagai boot loader adalah


file dalam dos yang berfungsi sebagai boot loader adalah mbldr.exe (CMIIW) File tersebut berfungsi sebagai boot loader membawa OS untuk memasuki boot yang ada di dalam sistem operasi.

MBLDR merupakan kependekan atau singkatan dari Master Boot Loader. Mbldr dibuat dengan base OS Boot Select (OS-BS) by Thomas wolfRAM


Pembahasan

Sebagai intermezzo, Boot loader atau dalam bahasa indonesia pemuat boot. Bootloader merupakan program komputer yang memuat sistem operasi (OS) untuk komputer secara sederhananya digunakan untuk memanggil boot sistem operasi dan memilih akan menggunakan Ssistem operasi yang mana.

Bootloader dapat digunakan untuk mengkonfigurasi  dan sarana untuk memilih darimana memulai kernel. Hal ini memungkinkan untuk melakukan aktivitas boot yang memiliki 2 Sistem operasi seperti contoh untuk mengunakan Linux alternatif, atau Windows. Bootloader adalah komponen penting dari proses booting yang harus dipastikan dalam keadaan baik, memori bootloader tersebut tersimpan dalam memori non-volatile contohnya seperti memori flash.

Aturan penamaan file dalam DOS adalah:

Case-Insensitive alias tidak sensitif dengan huruf. Maksudnya, penggunaan huruf kapitall dan huruf kecil tidak akan berpengaruh nantinya saat dimasukkan perintah.Nama ekstensi dibatasi maksimal sebanyak 3 karakter. 11. Jelaskan file utama pendukung berjalannya sistem MS - DOS​


Disk Operating System atau disingkat dengan DOS adalah sistem operasi yang menggunakan interface command-line yang digunakan para pengguna komputer pada dekade tahun 1980-an. Sekarang DOS menjadi istilah generik bagi setiap sistem operasi yang dimuat dari perangkat penyimpanan berupa disk saat sistem komputer dinyalakan.

Fungsi Disk Operating System (DOS)

Berikut ini adalah fungsi dari Disk Operating System (DOS).

> Mengorganisasikan atau mengendalikan kegiatan komputer

> Mengatur memori

> Mengatur proses input dan output data

> Management file

> Management directory

Bagian-bagian Disk Operating System (DOS)

Terdapat tiga bagian utama yang membangun disk operating system yaitu :

File boot, yaitu : File ini digunakan pada saat proses boot pada DOS atau proses startup pada sistem. Manajemen file, yaitu : Bagian ini memungkinkan sistem untuk mengatur datanya dalam suatu sistem file dan folder. File utility, yaitu : Bagian ini memungkinkan pengguna mengatur resources dari sistem, melakukan troubleshooting sistem, dan mengkonfigurasi seting sistem.

12. bagai mana cara membuat file di sistem operasi DOS.


1. buka cmd.exe atau command prompt
2. pada prompt yang sedang aktif (misal C:\Documents>) ketikkan :
copy con “test.txt” kemudian tekan enter
3. silakan isi file tersebut deng tulisan sesuai yang diinginkan
4. setelah selesai, untuk menyimpan dan keluar dari editor :
– tekan Ctrl + Z kemudian enter

Maka pembuatan file test.txt telah selesai.
Dan apabila ingin melihat kembali isi file tersebut melalui command prompt silakan ketikkan :

– type test.txt kemudian enter

Maka akan muncul isi dari file tersebut pada command prompt.
Kemudian bila ingin mengeditnya lagi silakan ketik pada command prompt :

– edit test.txt kemudian enter
– muncul editor bawaan dari Cmd
– untuk menyimpan : file -> save
– untuk keluar : file -> exit

25. menjelaskan fungsi file konfigurasi pada sistem komputer(batch,confi file)


Batch file memiliki 2 jenis fungsi yaitu :
1. Fungsi positif :
Bagi sebagian orang yang cinta kedamaian mereka akan memanfaatkan batch file untuk hal-hal yang baik yang diantaranya : untuk membuat program tuneup windows, program administrator windows dll.
2. Fungsi Negatif :
Bagi sebagian para cracker batch file bisa mereka manfaatkan untuk membuat sebuah aplikasi yang di antaranya : virus, program perusak dll.
Batch file biasanya kita buat menggunakan notepad dan untuk mempelajari pemrogramman batch file tidaklah sesulit kita mempelajari bahasa pemrogramman yang lain nya seperti : c, c++, java, vb dll. Pemrogramman batch file merupakan pemrogramman yang klasik dimana semua orang pun bisa mempelajarinya dengan mudah hanya dengan beberapa hari saja. bahkan ada yang bilang mereka bisa menjadi programmer batch file expert hanya membutuhkan 7 hari saja( sudah bisa buat virus, dan program untuk tuneup windows)

26. Mengapa Batch File di dalam operating system sangat diperlukan


Berkas instruksi atau batch file adalah sebuah berkas teks yang berisi beberapa seri perintah untuk dijalankan oleh interpreter perintah. Atau dapat didefinisikan juga batch merupakan kumpulan perintah-perintah command line yang dapat dijalankan pada Microsoft Windows.
Ketika sebuah batch file dijalankan, program shell yang berupa command.com akan membaca file batch dan kemudian mengeksekusi perintahnya, secara baris demi baris. Batch files berguna untuk menjalankan rangkaian dari dari beberapa perintah yang bisa dieksekusi (executables) secara otomatis dan sering digunakan oleh sistem administrator untuk mengotomatisasi banyak proses secara bersamaan. Perintah-perintah batch file bersifat case insensitive jadi tidak ada perbedaan maksud antara huruf besar dengan huruf kecil.

Dalam DOS batch command, perintah-perintah seperti Copy, Dir, Delete disebut dengan perintah internal sedangkan perintah semisal Format,Undelete,Backup merupakan bagian dari perintah eksternal. Setiap kita menjalankan perintah DOS batch file baik internal ataupun eksternal, command.com akan langsung menjalankan peritah internal atau memanggil program eksternal yang terpisah dan menjalankan perintah tersebut untuk kemudian mengembalikan hasilnya dalam bentuk perintah eksternal
