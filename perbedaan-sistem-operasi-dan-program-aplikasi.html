1. Apa perbedaan antara sistem operasi dan program aplikasi?


Sistem operasi adalah program yang berinteraksi langsung dengan mesin. dalam sistem operasi terdapat kernel yang menghubungkan aplikasi dengan hardware.
sedangkan aplikasi adalah program yang berjalan dibawah sistem operasi, aplikasi memanfaatkan kernel dari sistem operasi untuk menjalankan fungsinya.Program aplikasi adalah software/ perangkat lunak komputer yang dibuat untuk melakukan tugas tertentu.

Sistem operasi adalah komponen pengolah peranti lunak dasar tersistem sebagai pengelola sumber daya perangkat keras komputer (hardware), dan menyediakan layanan umum untuk aplikasi perangkat lunak.

Jawaban:

Type Jaringan

Type Jaringan terkait erat dengan sistem operasi jaringan. Ada dua type jaringan, yaitu client-server dan type jaringan peer to peer.

1. Jaringan Client-Server

Server adalah komputer yang menyediakan fasilitas bagi komputer-komputer lain di dalam jaringan dan client adalah komputer-komputer yang menerima atau menggunakan fasilitas yang disediakan oleh server

keunggulan

Kecepatan akses lebih tinggi karena penyediaan fasilitas jaringan dan pengelolaannya dilakukan secara khusus oleh satu komputer (server) yang tidak dibebani dengan tugas lain seperti sebagai workstation.

kekurangan

Kelangsungan jaringan sangat tergantung pada server. Bila server mengalami gangguan maka secara keseluruhan jaringan akan terganggu.

2.Jaringan Peer To Peer

Bila ditinjau dari peran server di kedua tipe jaringan tersebut, maka server di jaringan tipe peer to peer diistilahkan non-dedicated server, karena server tidak berperan sebagai server murni melainkan sekaligus dapat berperan sebagai workstation.

keunggulan

· Antar komputer dalam jaringan dapat saling berbagi-pakai fasilitas yang dimilikinya seperti: harddisk, drive, fax/modem, printer

kelemahan

Karena data jaringan tersebar di masing-masing komputer dalam jaringan, maka backup harus dilakukan oleh masing-masing komputer tersebut.

20. Jelaskan perbedaan sistem operasi dengan program aplikasi dalam sebuah sistem komputer ? Jelaskan dan berikan contoh-contohnya !.


Perbedaan sistem operasi dan program aplikasi

Segi Tugas

Tugas sistem operasi sendiri adalah mengatur manajemen resources dan operasi yang terjadi antar komponennya.

Program aplikasi bertugas untuk memenuhi kebutuhan penggunanya melalui kegunaannya baik secara umum maupun spesifik

Segi Keterkaitan

Sistem operasi dapat berjalan walaupun tidak ada program aplikasi. Akan tetapi karena tidak ada program aplikasi, sistem operasi tidak memiliki manfaat untuk penggunanya

Program aplikasi hanya dapat berjalan jika ada sistem operasi yang aktif di komputer tersebut

Contoh

Sistem operasi yang populer saat ini,Microsoft Windows, Macintosh dan Linux

Program aplikasi seperti program aplikasi pengolah grafis ( Adobe photoshop, CorelDrwa, AutoCad), pengolah kata ( Microsoft Word, WPS ) dan masih banyak lagi.

Pembahasan

Sistem operasi adalah sekumpulan program yang memiliki tugas dalam penagturan resources sekaligus menjadi jembatan perangkat keras dengan perangkat lunak. Keberadaan sistem operasi dalam suatu sistem komputer sangat diperlukan.

Program aplikasi atau Sistem Aplikasi merupakan program yang sudah berbentuk jadi yang memiliki fitur dan tujuan yang berbeda - beda sesuai kebutuhan pengguna dan kemampuan komputer. Program aplikasi secara umum contohnya pengolah kata, pengolah angka, pengolah database, pengolah grafis dan lain - lain. Sedangkan secara spesifik seperti Microsoft Word, Microsoft Access, Adobe After Effect, Open Office Calc dan masih banyak lagi
