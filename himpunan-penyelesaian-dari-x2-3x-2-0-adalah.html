2. Himpunan penyelesaian persamaan x2 - 3x 2 = 0 adalah


Terdapat sebuah persamaan: x²-3x+2 = 0. Himpunan penyelesaiannya (HP) adalah {1, 2}. Himpunan ini diperoleh dengan konsep persamaan kuadrat.

Penjelasan dengan langkah-langkah

Soal tidak mencatumkan tanda persamaan dengan jelas, khususnya untuk konstanta. Karena tanda negatif tercantum mendampingi koefisien x, asumsikan tanda yang tidak tercantum adalah tanda positif.

Diketahui: x²-3x+2 = 0

Ditanya: HP

Jawab:

Pemfaktoran persamaan kuadrat

Dua angka yang dijumlahkan menghasilkan -3 (koefisien x) dan dikalikan menghasilkan 2 (konstanta) adalah -2 dan -1. Dengan demikian, pemfaktorannya menjadi sebagai berikut:

x²-3x+2 = 0

(x-2)(x-1) = 0

x = 2 atau x = 1

HP

Tuliskan solusi persamaan kuadrat ke dalam notasi himpunan sebagai berikut:

{1, 2}

Jadi, HP persamaan kuadrat tersebut adalah {1, 2}.

27. Tentukan himpunan penyelesaian dari persamaan kuadrat berikut x2-3x+2=0 menggunakan rumus abc


Himpunan penyelesaian dari persamaan kuadrat [tex]\rm x^2 - 3x + 2 = 0[/tex] adalah {-1, -2}

Pembahasan

Rumus abc :

[tex]\boxed{\rm x =\dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}}[/tex]

Diketahui :

Persamaan kuadrat : [tex]\rm x^2 - 3x + 2 = 0[/tex]a = 1b = 3c = 2

Ditanya :

Penyelesaian dengan rumus abc ?

Jawaban :

Gunakan rumus diatas

[tex]\rm x =\dfrac{-3 \pm \sqrt{3^2 - 4(1\times 2)}}{2(1)}\\\rm x =\dfrac{-3 \pm \sqrt{9 - 8}}{2}\\\rm x =\dfrac{-3 \pm \sqrt{1}}{2}\\\rm x =\dfrac{-3 \pm 1}{2}[/tex]

Untuk [tex]\rm x =\dfrac{-3 + 1}{2}[/tex]

[tex]\rm x =\dfrac{-3 + 1}{2}\\\rm x =\dfrac{-2}{2}\\\rm x = -1[/tex]

Untuk [tex]\rm x =\dfrac{-3 - 1}{2}[/tex]

[tex]\rm x =\dfrac{-3 - 1}{2}\\\rm x =\dfrac{-4}{2}\\\rm x = -2[/tex]

Maka, himpunan penyelesaiannya adalah {-1, -2}.

28. Himpunan penyelesaian dari persamaan kuadrat x2 - 3x + 2 = 0 adalah . . . A. {1,2}B. {-2,-1}C. {-2,1}D. {-1,2}​


Jawaban:

A. {1,2}

Penjelasan dengan langkah-langkah:

x² - 3x + 2 = 0 ======> Di faktorkan

( -1 × (-2))= 2

(-1 + (-2)) = -3

jadi.......

(x - 1) dan (x - 2)

x - 1 = 0 x - 2 = 0

x = 1 x = 2

jadi himpunan penyelesaian dari x² - 3x + 2 = 0 adalah { 1, 2 }

maaf kalau salah
