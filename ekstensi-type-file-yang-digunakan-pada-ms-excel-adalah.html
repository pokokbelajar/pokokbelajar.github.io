12. Ekstensi dari file dokumen microsoft excel 2016 adalah


بسم الله الرحمن الرحيم

1. *.XLS merupakan format default Microsoft Excel sebelum Excel 12. Masih berbasis format BIFF dan dapat menyimpan macro VBA.

2. *.XML atau yang disebut sebagai XML Spreadsheet. Tidak mampu menampung macro VBA.

3. *.XLA merupakan format untuk Excel Add-in sebelum Excel 12. Masih berbasis format BIFF dan dapat menyimpan macro VBA, mengingat tujuannya adalah untuk menambahkan kemampuan Microsoft Excel.

Jawaban: C

Penjelasan:

Extensi file dari dokumen Microsoft Excel yang menggunakan Macro VBA adalah xlsm, dengan xls menunjukkan bahwa file tersebut adalah dokumen Microsoft Excel dan m menunjukkan bahwa dokumen Microsoft Excel tersebut menggunakan Macro VBA.

A. xls -> Merupakan file dokumen Microsoft Excel biasa

B. ppt -> Merupakan file dokumen Microsoft Powerpoint

29. Sebutkan ekstensi file hasil penyimpanan Ms. Excel!(Buat yang tau tolong dijawab ya!)​


Untuk satu file

1. Di Mac Anda, pilih file, lalu pilih File > Dapatkan Info, atau tekan Command-I.

2. Klik segitiga di samping Nama & Ekstensi untuk memperluas bagian tersebut.

3. Untuk menampilkan atau menyembunyikan ekstensi nama file, pilih atau batal pilih “Sembunyikan ekstensi”.

Untuk semua file

1. Di Finder di Mac Anda, pilih Finder > Preferensi, lalu klik Lanjutan.

2. Pilih atau batal pilih “Tampilkan semua ekstensi nama file”.
