7. termometer celcius dan reamur menunjukkan suhu yang sama pada suhu


• SUHU DAN KALOR

Kata Kunci : termometer.


Masalah :

Termometer celcius dan reamur menunjukkan suhu yang sama pada suhu berapa ?

Penyelesaian :

Apabila kedua termometer menunjukkan angka yang sama, maka nilai C dan R adalah sama ( C = R ). Karena sama, saya simbolkan X.


(C - 0)/5 = (R-0)/4

(X - 0)/5 = (X - 0)/4

4X - 0 = 5X - 0

0 = X


Maka, dapat disimpulkan bahwa termometer celcius dan reamur akan menunjukkan suhu yang sama pada suhu 0°.

Kelas : XI
Materi : Suhu dan Kalor
Kata Kunci : termometer.

Kode Kategorisasi : 11.6.5

8. Suhu pada termometer fahrenhait 86°. Berapakah suhu termometer reamur dan celcius


Penjelasan dengan langkah-langkah:

Fahrenheit = 86°

Celcius = 5/9 x ( 86 - 32 )

5/9 x 54 = 54 : 9 = 6 x 5 = 30 °C

Reamur = 4/9 x ( 86 - 32 )

4/9 x 54 = 54 : 9 = 6 x 4 = 24 °R

Kelvin = 5/9 x ( 86 - 32 ) + 273

5/9 x 54 + 273

5/9 x 54 = 54 : 9 = 6 x 5 = 30 + 273 = 303 K

12. Termometer skala reamur menunjukan suhu 40 celcius,maka termometer reamur menunjukan


Jawaban :

 32°R

Pertanyaan :

Termometer skala reamur menunjukan suhu 40 celcius,maka termometer reamur menunjukan..

Penjelasan :

Suhu (Termometer) Adalah Ukuran derajat panas atau dinginnya suatu benda. untuk dapat mengetahui besar suhu suatu benda dapat digunakan alat ukur suhu yaitu termometer. Pada setiap termometer memiliki 2 titik tetap yaitu titik tetap bawah dan titik tetap atas. Syarat dari kedua titik itu yang harus dipenuhi yaitu diperoleh dan tidak berubah ubah. Ada 4 jenis termometer berdasarkan pembuatannya, antara lain: Celsius , Reamur , Fahrenheit , Dan Kelvin.

Berikut beberapa rumus rumusnya :

✨ Reamur ( °R )

[tex]\boxed {\boxed { R= \frac{4}{5} \times C }}[/tex]

✨ Fahrenheit ( °F )

[tex]\boxed {\boxed { F = \frac{9}{5} \times (C + 32) }}[/tex]

✨ Kelvin ( K )

[tex]\boxed {\boxed { K = C + 273}}[/tex]

Penyelesaian :

Diketahui :

✨ C = 40°C

Ditanya :

✨ Skala Reamur

Jawab :

[tex]R = \frac{4}{5} \times C[/tex]

[tex]R =\frac{4}{5}\times 40^{o}[/tex]

[tex]R = 4 \times 8^{o}[/tex]

[tex]R = 32^{o}R [/tex]

Kesimpulan :

Jadi skala reamur menunjukan 32°R
