16. Bagaimana cara menggabungkan beberapa file excel menjadi 1 file dalam microsoft excel 2010?


Cara menggabungkan beberapa file excel menjadi 1 file yaitu :

1. Buka semua file yang akan anda gabung

2. Habis itu klik di view lalu cari yang ada tulisan save workspace

3. Beri nama kepada semua file yang mau digabung dan kemudian save

4. Lalu kita buka file excel yang kita save tadi

5. Maka akan terbuka seluruh file excel tadi


Tujuannya adalah untuk mempermudah seseorang untuk membuka beberapa file yang mudah dan biasanya ini digunakan pada orang yang bekerja di kantor

25. cara menyimpan dokumen excel dengan menggunakan nama baru dan cara mengirimkan data excel melalui email


Misalkan dokumen excel 2007:
- bukalah dokumen escel yang ingin diganti dengan nama baru
- klik tombol office button dan klik tombol Save As
- isikan File Name dengan nama baru dan klik tombol Save

Misalkan mengirim data excel melalui email yahoo:
- akses situs www.ymail.com dan login dengan username serta passwordmu
- klik tombol Compose mail (tulis email baru)
- isikan To (tujuan email) dan Subject (judul email)
- klik tombol lampiran dan pilih Lampirkan file dari komputer
- saat tampil jendela pemilihan file, silakan dobel klik file excel yang ingin dikirim
- klik tombol Kirim (yg berwarna biru)

# untuk lebih jelas dalam melampirkan file di email, silakan perhatikan gambar berikut ini
