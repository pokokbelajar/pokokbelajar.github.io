7. Tentukan suku ke-n pada barisan bilanganberikut: 4, 11, 18, 25, ​


Penjelasan:

= Un = a + ( n - 1 ) b

• a = 4 ( 4 ini suku pertama )

• b = 7 ( karena dari 4 ke 11 lompat 7 angka, begitu juga dengan seterusnya )

= Un = 4 + ( n - 1 ) 7

= 4 + ( 7 × n ) + ( 7 × (-1))

= 4 + 7n - 7

= 7n + 4 - 7

= 7n - 3

Jadi, suku ke-n adalah 7n - 3

Suku ke-n adalah (c.) Uₙ = 7n - 3.

__________________________

Pendahuluan

Barisan Aritmatika adalah barisan dari bilangan-bilangan yang memiliki beda atau selisih yang sama di antara 2 suku yang saling berdekatan.

Rumus umum menentukan Suku ke-n:

Uₙ = a + ( n - 1 )b

Rumus menentukan Beda:

b = Uₙ - Uₙ₋₁

Keterangan:

Uₙ = Suku ke-n

n = Bilangan asli

a = Suku pertama (U₁)

b = Beda

__________________________

PenyelesaianDiketahui:

Barisan = 4, 11, 18, 25, ...

Suku pertama (a) = 4

Ditanya:

Suku ke-n (Uₙ)?

Dijawab:⏺ Beda (b)

b = Uₙ - Uₙ₋₁

ᵇⁱᵃˢᵃⁿʸᵃ ˢᵃʸᵃ ᵐᵉⁿᵍᵍᵘⁿᵃᵏᵃⁿ U₂

b = U₂ - U₂₋₁

b = U₂ - U₁

b = 11 - 4

b = 7

----------------------------

⏺ Suku ke-n (Uₙ)

Uₙ = a + ( n - 1 )b

Uₙ = 4 + ( n - 1 )7

ᵍᵘⁿᵃᵏᵃⁿ ˢⁱᶠᵃᵗ ᵈⁱˢᵗʳⁱᵇᵘᵗⁱᶠ a × ( b - c ) = ( a × b ) - ( a × c )

Uₙ = 4 + ( 7 × n - 7 × 1 )

Uₙ = 4 + 7n - 7

Uₙ = 7n - 7 + 4

Uₙ = 7n - 3 (c.) ✔

Kesimpulan:

Rumus suku ke-n pada barisan 4, 11, 18, 25, ... adalah Uₙ = 7n - 3 (c.).
