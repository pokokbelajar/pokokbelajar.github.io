15. 21. Harga 1 pulpen dan 2 buku Rp10.000,00 sedangkan harga 2 pulpen dan 1 buku Rp11.000,00. Harga 1 pulpen adalah ....


■ Mapel : Math
■ Materi : Eliminasi dan Substitusi
■ Bab : SPLDV
———————————————— [tex] [/tex]
Pulpen = p
Buku = b

Eliminasi
p + 2b = 10.000 ( kalikan 2 )
2p + b = 11.000 ( kalikan 1 )

2p + 4b = 20.000
2p + b = 11.000
------------------------- -
       3b = 9.000
         b = 3.000

Substitusikan
2p + b = 11.000
2p = 11.000 - 3.000
2p = 8.000
p = 4.000

Jadi, harga 1 pulpen adalah Rp 4.000,00Pulpen = X
Buku = Y
===========
Sekarang mencari harga Pulpen (X)
 x  + 2y = 10.000  (dikali 2 )
2x + y   = 11.000  ( dikali 1 )

2x + 4y = 20.000
2x + y   = 11.000
----------------------- - (dikurang)
x + 3 = 9.000
x = 9.000 : 3 = 3.000 (X)

Mencari buku ( Y )
x   + 2y = 10.000  (dikali 1)
2x + y   = 11.000  (dikali 2 )

x   + 2y = 10.000
4x + 2y = 22.000
------------------------ - (dikurangi)
-3  -y = - 12.000
y = - 12.000 / -3 = 4.000


Jadi harga 1 pulpen adalah Rp. 3.000

Harga 1 pulpen dan 2 buku adalah Rp. 10.000. Sedangkan harga 2 pulpen dan 1 buku adalah Rp. 11.000. Maka harga 1 pulpen adalah Rp. 4.000.

Penyelesaian Soal :

Misalkan Harga pulpen = x

               Harga buku = y

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Harga 1 pulpen dan 2 buku adalah Rp. 10.000" yaitu :

x + 2y = 10.000    ... (Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "harga 2 pulpen dan 1 buku adalah Rp. 11.000" yaitu :

2x + y = 11.000    ... (Persamaan 2)

LANGKAH KETIGA (III)

Eliminasikan persamaan 1 dan persamaan 2 sehingga diperoleh nilai x dengan menggunakan cara sebagai berikut :

x + 2y = 10.000    ║×1 ║    x + 2y = 10.000

2x + y = 11.000     ║×2║   4x + 2y = 22.000

___________________________________ -

                                                -3x = -12.000

                                                   x = [tex]\frac{12.000}{3}[/tex]

                                                   x = 4.000

∴ Kesimpulan harga 1 pulpen adalah Rp. 4.000.
