Tentukanlah besar : a. Harga pembelian tiket untuk 1 orang dewasa b. Harga pembelian tiket untuk 1 orang anak-anak


Jawab:

Penjelasan dengan langkah :

misal

orang dewasa = x

anak-anak = y

jawab :

2x + 3y = 130.000 / dikali 5

5x + 2y = 215.000 / dikali 2

hilangkan x nya, menjadi = 10x + 15y = 650.000

                                              10x + 4y = 430.000   -

                                                          11y = 220.000

                                                              y = 20.000 ( anak anak )

subtitusikan

2x + 3y = 130.000

2x + 3(20.000) = 130.000

2x + 60.000 = 130.000

                2x = 130.000 - 60.000

                2x = 70.000

                  x = 35.000 ( orang dewasa )

jadi harga a. 1 tiket orang dewasa = 35.000

                  b. 1 tiket anak anak = 20.000

Disini saya jelaskan teknisnya. Kita pecahkan ada berapa variable yang harus dibuat. Variable

bonus, minimal pembelian, maximal pembelian, harga, total harga,

Kondisi 1

Apabila user membeli tiket kurang dari min pemb maka akan menampilkan output bahwa minimal pembelian.

Kondisi 2

Apabila user membeli tiket lebih dari max pemb maka akan menampilkan output bahwa minimal pembelian max pemb.

Kondisi 3

Apabila user membeli lebih dari 1 tiket maka akan langsung diproses yaitu total harga=harga*tiket.

Kondisi 4

✄ Kita perlu mengetahui berapa harga per km nya atau harga 1 km nya. Maka dapat dijelaskan bahwa :

Ditanya :

Harga tiket kereta untuk 60 km?

⌖ Mencari harga tiket untuk 1 km / per km

[tex]\red{\boxed{\boxed{ = \frac{\sf harga \: tiket \: yang \: diketahui}{\sf satuan \: km \: yang \: diketahui}}}}[/tex]

⌖ Mencari harga tiket untuk 60 km

[tex]\red{\boxed{\boxed{\sf =Harga \: tiket \: 1 \: km × Harga \: dari \: satuan \: yang \: ditanyakan}}}[/tex]

[tex]\tt = RP. \: 300 × 60 \: km[/tex]

[tex]\green{\boxed{\boxed{\boxed{\bf{= \: RP. \: 18.000}}}}}[/tex]

Jadi, kesimpulan dari harga tiket kereta untuk 60 km adalah Rp. 18.000
