9. Jelaskan langkah langkah untuk mengisi neraca saldo awal pada myob accounting plus versi 18


Langkah-langkah mengisi neraca saldo awal pada MYOB accounting+ versi 18 adalah sebagai berikut :

Pada toolbar MYOB, klik SetupSetelah itu, klik BalanceKlik Account Opening BalanceMasukan saldo semua akunnya dengan syarat ketik angka tanpa mata uang, koma, atau titik (ex : Rp 100.000 menjadi 100000)Pada akun yang merupakan akun pengurangan saldo akun sebelumnya di tambahkan tanda minus (-) sebelum saldo akun tersebut

PEMBAHASAN

MYOB alias Mind Your Own Bussines adalah salah satu aplikasi akuntansi yang berbasis komputer. Dalam tampilan awal MYOB terdapat empat menu yaitu :

Open - Membuka dokumen yang sebelumnya telah dibuat. Create - Membuat dokumen baru untuk laporan keuangan akuntansi.Explore - Membuat dokumen berdasarkan sample/contoh yang telah disediakan dalam MYOB. What's New - melihat ada atau tidaknya pembaharuan pada sistem MYOBExit - Keluar dari sistem MYOB.

Untuk mengisi neraca saldo pada MYOB accounting+ versi 18 dapat mengikuti langkah berikut :

Pada toolbar MYOB, klik menu SetupSetelah itu, klik Balance. Klik Account Opening Balance. Masukan saldo semua akunnya dengan syarat ketik angka tanpa mata uang, koma, atau titik (ex : Rp 100.000 menjadi 100000)Pada akun yang merupakan akun pengurangan saldo akun sebelumnya di tambahkan tanda minus (-) sebelum saldo akun tersebut.

14. jelaskan langkah langkah untuk mensetting akun pajak perusahaan pada myob accounting plus versi 18​


Langkah langkah untuk mensetting akun pajak perusahaan pada myob accounting plus versi 18:

1. Tekan List yang ada di toolbar MYOB Accounting+ Versi 18, lalu pilih Tax Codes.2. Delete tax code FRE GST Free di jendela Tax Code List dengan klik kanan pada tax codes FRE GST Free, lalu klik Delete Tax Code.3. Lakukan editing pada tax code GST dengan mengklik edit, lalu tulis seperti ini:

( Tax code : PPN.

( Description : Pajak Pertambahan Nilai.

( Tax type : Goods & Service Tax.

( Rate : 10%.

( Linked Account for Tax Collected : Vat Outcome.

( Linked Account for Tax Paid : Vat Income.

Klik OK.

4. Lakukan editing pada tax code N-T dengan mengklik Edit, lalu tulis seperti ini:

( Description : Tidak kena pajak.

( Linked Account for Tax Collected : Vat Outcome.

( Linked Account for Tax Paid : Vat Income.

Klik OK.

5. Klik Close.

Pembahasan:

MYOB ialah suatu tecno-acccunting yang berkepanjangan Mind Your Own Bussines yang difungsikan untuk mempermudah menulis data keuangan suatu perusahaan. Dalam MYOB ini, terkandung banyak fitur seperti halnya fitur membuat data pajak. Berikut langkah-langkahnya:

1. Tekan List yang ada di toolbar MYOB Accounting+ Versi 18, lalu pilih Tax Codes.

2. Delete tax code FRE GST Free di jendela Tax Code List dengan klik kanan pada tax codes FRE GST Free, lalu klik Delete Tax Code.

3. Lakukan editing pada tax code GST dengan mengklik edit, lalu tulis seperti ini:

( Tax code : PPN.( Description : Pajak Pertambahan Nilai.( Tax type : Goods & Service Tax.( Rate : 10%.( Linked Account for Tax Collected : Vat Outcome.( Linked Account for Tax Paid : Vat Income.

Klik OK.

4. Lakukan editing pada tax code N-T dengan mengklik Edit, lalu tulis seperti ini:

( Description : Tidak kena pajak.( Linked Account for Tax Collected : Vat Outcome.( Linked Account for Tax Paid : Vat Income.

Klik OK.

5. Klik Close.

15. jelaskan komponen komponen yg di butuh kan dalam menginstal myob accounting versi 18​


Jawaban:

MYOB ( Minding Your Own Business) adalah paket program komputer akuntansi yang dibuat secara terpadu. Program ini dibuat oleh perusahaan pembuat software : Data Tech Software Pty. Ltd. Austrlia. MYOB ini dapat digunakan untuk pembukuan berbagai jenis usaha baik usaha besar maupun usaha kecil. Program ini digunakan untuk pembukuan secara lengkap, cepat dan akurat seperti pemasukan daftar akun, pengaturan (setup), mengelola bank, pelanggan, pemasok, produk sampai pada pembuatan laporan keuangan ; neraca, laba rugi dan sebagainya.

Nah kalau kita sudah tahu pengertian dan kegunaan aplikasi MYOB sekarang tinggal bagaimana cara menginstall program tersebut pada komputer atau laptop kita. Berikut langkah langkah install MYOB :

Silahkan download file aplikasi MYOB disini. Kemudian ekstrak filenya di folder yang sudah Anda tentukan pada komputer Anda.

Klik dua kali pada aplikasi MYOB_SG_Accounting_V18.

3. Selanjutnya akan muncul kotak dialog Preparing to Install… Tunggu sampai selesai.

4. Kemudian klik tombol Next pada kotak dialog yang muncul berikutnya.

5. Selanjutnya pada kotak dialog Licence Agreement, pilih opsi yang pertama “I accept the terms of the licence agreement“. Lalu klik tombol Next.

6. Pada kotak dialog Install Type, pilih opsi “Typical” dan kemudian klik tombol Next.

7. Klik tombol Next pada kotak dialog selanjutnya ini.

8. Klik tombol Next kembali pada kotak dialog selanjutnya ini.

9. Kemudian proses instalasi aplikasi akan berjalan dan tunggu sampai proses instalasi selesai.

10. Setelah semua proses selesai, maka pilih “Yes, I want to restart my computer now” untuk me-restart komputer. Dan kemudian klik tombol Finish. Proses instalasi MYOB v18 telah selesai dan aplikasi sudah dapat digunakan.

Jawaban:

1) Langkah-langkah memeriksa kebenaran neraca saldo pada aplikasi MYOB Accounting Plus Versi 18 sebagai berikut.

Pada jendela Command Centre bagian bawah, klik Reports, kemudian pilih Accounts. Setelah itu, akan muncul tampilan jendela Index to Report. Pilih Trial Balance pada menu Accounts, kemudian klik Customise.Isi kolom Selected Period dengan bulan November dan kolom Financial Year dengan This Year (FY 2015). Setelah itu klik Display. Akan muncul Neraca Saldo per 30 November 2015.

[contoh gambar tersedia pada lampiran]

2) Linked Account pada MYOB Accounting Plus Versi 18 terdiri atas 4 macam, yaitu

Accounts & Banking Accounts, yaitu linked account yang berhubungan dengan akun yang bertipe bank. Sales Accounts, yaitu linked account yang berhubungan dengan akun penjualan. Purchases Accounts, yaitu linked account yang berhubungan dengan akun pembelian. Payroll Accounts, yaitu linked account yang berhubungan dengan akun penggajian.

3) Langkah-langkah mengatur dan mengubah linked account pada Purchases Accounts yaitu sebagai berikut.

Klik Setup pada toolbar MYOB Accounting Plus Versi 18. Selanjutnya klik Linked Accounts dan pilih Purchases Accounts. Pada jendela Liability Account for Tracking Payables, klik ikon dan pilih Accounts Payable. Selanjutnya pada Bank Account for Paying Bills, pilih Cash in bank, kemudian klik OK.

29. Jelaskan langkah-langkah untuk mengentri bukti transaksi prive pada myob accounting plus versi 18


Jawaban:

Buka program MYOB Accounting Plus versi 18 dan login ke dalam akun perusahaan anda.Pilih menu "Accounts Command Centre" di bagian atas jendela program.Pilih "Record Journal Entry" dari menu drop-down yang muncul.Isi informasi header bukti transaksi, seperti tanggal bukti transaksi dan nomor bukti transaksi, pada bagian atas halaman.Isi informasi rincian transaksi pada bagian "Journal Entry" di bawah header. Pastikan untuk memilih akun yang sesuai untuk setiap entri transaksi yang anda masukkan.Jika anda perlu menambahkan catatan atau keterangan tambahan, gunakan bagian "Memo" yang tersedia.Jika sudah selesai, klik tombol "Record" untuk menyimpan bukti transaksi ke dalam program.
