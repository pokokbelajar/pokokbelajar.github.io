3 m³ = 3 × 1.000 dm³ = 3.000 dm³

3.000 dm³ = 3.000 × 1 liter = 3.000 liter

Jadi, 3 m³ = 3.000 dm³ = 3.000 liter = 3.000.000 ml

.... m³ = .... dm³ = 22.000 liter = .... ml

22.000 liter = 22.000 × 1.000 ml = 22.000.000 ml

22.000 liter = 22.000 ÷ 1 dm³ = 22.000 dm³

22.000 dm³ = 22.000 ÷ 1.000 m³ = 22 m³

Jadi, 22 m³ = 22.000 dm³ = 22.000 liter = 22.000.000 ml

.... m³ = .... dm³ = .... liter = 35.000 ml

35.000 ml = 35.000 ÷ 1.000 liter = 35 liter

35 liter = 35 ÷ 1 dm³ = 35 dm³

35 dm³ = 35 ÷ 1.000 m³ = 0,035 m³

Jadi, 0,035 m³ = 35 dm³ = 35 liter = 35.000 ml
