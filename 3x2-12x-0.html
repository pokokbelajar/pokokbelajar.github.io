Himpunan penyelesaian persamaan kuadrat dari 3x² - 12x = 0 adalah {0, 4}.

Hasil tersebut diperoleh dari memfaktorkan persamaan kuadrat.

Penjelasan dengan langkah-langkah

Diketahui :

Persamaan 3x² - 12x = 0

Ditanya :

Himpunan penyelesaian.

Jawab :

Menentukan akar persamaan kuadrat

3x² - 12x = 0

3x (x - 4) = 0

3x = 0

 x = 0/3

 x = 0

atau

x - 4 = 0

    x = 4

Jadi himpunan penyelesaian dari 3x² - 12x = 0 adalah {0, 4}.

α = p - 2

x = p - 2

3x² - 12x + 2 = 0

3 (p - 2)² - 12 (p - 2) + 2 = 0

3 (p² - 4p + 4) - 12p + 24 + 2 = 0

3p² - 12p + 12 - 12p + 26 = 0

3p² - 24p + 38 = 0
