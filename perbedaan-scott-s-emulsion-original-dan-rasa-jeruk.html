11. Ibu Dina memproduksi dua jenis keripik singkong, yaitu rasa original dan balado. Setiap kilogram keripik rasa original membutuhkan modal Rp. 10.000 sedangkan keripik rasa balado membutuhkan modal Rp. 15.000 per kilogram. Modal yang dimiliki ibu Dina Rp. 500.000. Setiap hari hanya bisa memproduksi keripik singkong paling banyak 40 kilogram. Keuntungan tiap kilogram keripik singkong rasa original adalah Rp. 2.500 dan rasa balado adalah Rp. 3.500. Tentukan keuntungan terbesar yang diperoleh ibu Dina ​


Jawaban:

Pembahasan :

Misal

x = keripik pisang rasa coklat

y = keripik pisang rasa keju

Modal :

keripik rasa coklat = Rp10.000,00

keripik rasa keju = Rp15.000,00

Modal ibu yang dimiliki = Rp500.000,00

10.000x + 15.000y ≤ 500.000

10x + 15y ≤ 500

2x + 3y ≤ 100 ........... (1)

(≤ : daerah diarsir ke bawah)

x = 0

=> 2(0) + 3y = 100

=> y = 100/3

=> (0, 100/3)

y = 0

=> 2x + 3(0) = 100

=> x = 50

=> (50, 0)

Tiap hari dapat memproduksi paling banyak 40 kg

x + y ≤ 40 ................ (2)

(≤ : daerah diarsir ke bawah)

x = 0

=> 0 + y = 40

=> y = 40

=> (0, 40)

y = 0

=> x + 0 = 40

=> x = 40

=> (40, 0)

x ≥ 0, y ≥ 0 .............. (3)

Fungsi Sasaran => Keuntungan

Keripik rasa coklat = Rp2.500,00

Keripik rasa keju = Rp3.000,00

f(x, y) = 2.500x + 3.000y

Ditanyakan :

Keuntungan terbesar = ... ?

Titik potong garis (1) dan (2)

2x + 3y = 100 |×1|

x + y = 40 |×2|

2x + 3y = 100

2x + 2y = 80

-------------------- -

....... y = 20

x + y = 40

x + 20 = 40

x = 20

(20, 20)

Berdasarkan gambar (lihat pada lampiran)

(0, 100/3), (20, 20) dan (40, 0)

Kita substitusikan ke fungsi sasaran

f(x, y) = 2.500x + 3.000y

f(0, 100/3) = 2.500(0) + 3.000(100/3) = 100.000

f(20, 20) = 2.500(20) + 3.000(20) = 110.000

f(40, 0) = 2.500(40) + 3.000(0) = 100.000

Jadi keuntungan maksimumnya adalah Rp110.000,00

12. Obat afdruk screen emulsion dan sensitizer merupakan cairan yang digunakan untuk


Obat afdruk screen emulsion dan sensitizer merupakan cairan yang digunakan untukPembahasan

Obat Afdruk digunakan untuk melapisi kain screen dan menghasilkan efek stencil (lubang bergambar) melalui proses photokimia.  Obat Afdruk biasanya terdiri dari dua komponen, cairan kental emulsion dan cairan warna orange yang sering disebut kalium /sensitizer (SR).

Sensitizer adalah cairan yang bersifat peka cahaya. Sensitizer berfungsi sebagai bahan campuran emulsi. Sensitizer digunakan untuk campuran photo emulsion obat afdruk, cara pakai menggunakan perbandingan 1 banding 8-12 (sesuai kebutuhan).

18. obat afdruk screen emulsion dan sensitizer merupakan cairan yang digunakan untuk


Obat Afdruk screen emulsion dan sensitizer ialah merupakan cairan yang digunakan untuk melapisi kain screen dan menghasilkan efek stencil atau lubang bergambar melalui sebuah proses photokimia, sedangkan sensitizer digunakan untuk menahan tinta diatas screem sehingga tidak merembes keluar.

Pembahasan  

Bahan emulsi atau yang biasa disebut dengan obat afdruk ialah merupakan cairan kental dari bahan kimia yang digunakan untuk melapisi screen printing. Dengan melalui proses photokimia, sebagian lubang pori screen printing yang dilapisi akan tertutup oleh emulsi dan sebagian lainnya dibiarkan terbuka sehingga bisa menghasilkan lubang bergambar. Melalui pori-pori yang berlubang inilah tinta akan mengalir ke media sablon. Obat Afdruk umumnya terdiri dari dua bahan cair yang harus dicampur, satunya berbentuk cairan kental atau disebut juga emulsion yang berfungsi sebagai penutup pori dan satunya berupa cairan encer yang disebut kalium atau sensitizer (SR) yang memiliki sifat peka terhadap cahaya. Jika kedua bahan ini dicampur maka akan terjadi reaksi kimia berupa pengerasan pada lapisan emulsi jika terkena cahaya. Oleh karena itulah proses pencampuran dan pelapisan emulsi pada screen printing harus dilakukan di tempat yang gelap.
