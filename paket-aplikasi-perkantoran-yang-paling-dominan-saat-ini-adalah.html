20. Jelaskan tentang icon menu pada paket aplikasi perangkat lunak perkantoran


perangkat lunak perkantoran yang biasa kita kenal dengan office suite. diantar aplikasi perkantoran yang umum digunakan oleh kita adalah microsoft office family.
icon menu diantaranya adalah 
1. file : (new, edit, save, dll) icon file digunakan untuk membuat dokumen baru, mengedit, mencetak, dan pengelolaan utama lainnya. 
2. edit :(copy, paste, cut, dll) icon edit digunakan untuk melakukan manipulasi konten dokumen seperti melakukan penjiplakan, menghapus secara masal, dan lain sebagainya.
3. view:(full screen, scale, web layout, dll) icon view digunakan untuk menyesuaikan tampilan dokumen yang sesuai dengan keinginan kita.
4. insert : (date and time, field, picture, dll) icon insert digunakan untuk menambahkan konten berupa keterangan-keterangan pendukung sebuah dokumen dan lain sebagainya
5. format : icon format digunakan untuk menentukan jenis huruf, tata letak paragraf, memberi bingkai, memberi footer dan header, mengatur spasi antar kata, spasi antar baris, dan sebagainya yang berkaitan dengan tata format sebuah dokumen
6. tools : icon tool digunakan untuk menyempurnakan ejaan sebuah dokumen, diantara fungsi yang sering dipakai dalam icon ini adalah menandai perubahan-perubahan yang dilakukan dalam dokumen dimana ketika mencatata setiap perubahan tersebut, kita dapat melihat kembali history yang sudah dilakukan.
7. tabel : icon tabel digunakan untuk mengatur tata letak tabel bila kita membutuhkan, segala sesuatu yang berkaitan dengan tabel, dimana ada kolom baris yang bisa kita pakai, untuk menyisipkan tabel dan sebagainya
