10. air yang sedang dimasak menunjukan suhu 90 derajat celcius. tentukan suhu pada termometer celcius fahrenheit dan kelvin


Reamur = 90 . 4/5 = 72°R

Farenheit = 90. 9/5 + 32 = 162 + 32 = 194°F

Kelvin = 90 + 273 = 363KMapel : Matematika
Bab : Satuan Suhu
Kelas : 6
Kode : [ Perbandingan - 2 - 6 - 06 ]

Jawaban :

Satuan suhu terbagi menjadi 4 bagian yaitu

1) Celcius ( C )
2) Reamur ( R )
3) Fahrenheit ( F )
4) Kelvin ( K ) => ( °C + 273 )

Jika pada satuan suhu kelvin tidak diberi satuan suhu atau derajat

Perhatikan perbandingan satuan suhu berikut ;

C : R : F = 5 : 4 : 9
Atau disingkat " Lats " yaitu

La = lima
t = empat
s = sembilan

Jika pada satuan suhu F jika akan diubah menjadi C atau R maka tambahkan ( + 32 ) jika satuan suhu C dan R diubah menjadi F maka kurangkan ( - 32 )

Soal :

Air yang sedang dimasak menunjukkan suhu 90°C . Tentukan suhu pada termometer Celcius , Fahrenheit , dan Kelvin

Jawaban :

Celcius = 90°

Reamur = R/C × 90°
= 4/5 × 90°
= 72°R

Fahrenheit = F/ C × 90°
= 9/5 × 90° + 32
= 162 + 32
= 194°F

Kelvin = °C + 273 = 90 + 273 = 363 K

11. ubahlah dari suhu 90 derajat kelvin kedalam skala Celcius Fahrenheit dan reamur​


Jawaban:

Pada saat Thermometer celcius menunjukkan skala 90°C , berapa skala pada thermometer Reamur, Fahrenheit, Kelvi

Pembahasan Soal :Diketahui :

t  = Suhu = 90°C

Ditanyakan :

t = Suhu pada Reamur = ..

t = Suhu pada Fahrenheit =...

t = Suhu pada Kelvin = ...

Penyelesaian :

C ⇒ R

4/5 x 90°R = 72°R                                                                                                

C ⇒ F

9/5 x 90 + 32 = 194°F

C ⇒ K

90 + 273 = 363°K

       

Penjelasan lanjutan :

Materi Konversi suhu  

Suhu benda yang diukur dengan mengunakan indra peraba menghasilkan ukuran suhu kualitatif yang tidak dapat dipakai sebagai acuan. suhu harus diukur secara kuantitatif dengan alat ukur suhu yang disebut Thermometer.  

Kita mengenal ada 4 skala thermometer yaitu :

1. Skala celcius :

Titik tetap atas =100°C

Titik tetap bawah = 0°C

skala celcius = 100°-0° = 100

2. Skala Reamur  

Titik tetap atas =80°R

Titik tetap bawah = 0°R

skala reamur = 80°- 0° = 80

3. Skala Fahrenheit

Titik tetap atas =212°F

Titik tetap bawah = 32°F

skala celcius = 212°- 32° = 180

4. Skala Kelvin

Titik tetap atas = 373°K

Titik tetap bawah = 273°C

skala celcius = 373°- 273° = 100

Perbandingan skala adalah : 5 : 4 : 9 : 5
