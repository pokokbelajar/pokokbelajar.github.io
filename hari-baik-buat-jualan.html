11. reza membeli sebuah sepeda motor dengan harga 9.400.000 motor tersebut di perbaiki 200.00 beberapa hari kemudian motor tersebut di jual 7.200.000 persentase kerugian reza adalah


penyelesaian :

harga beli = Rp. 9.400.000

biaya kerusakan = Rp. 200.000

harga jual = Rp. 7.200.000

kerugian = 9.400.000 - 200.000 - 7.200.000

               = 9.200.000 - 7.200.000

               = Rp. 2.000.000

persentase kerugian = 2.000.000 x 100/9.400.000

                                   = 200.000.000/9.400.000

                                   = 21,27%

                                   = 21,3%

Kerugian
= 9.400.000 - 7.200.000 + 200.000
= Rp. 2.400.000

Persentase Kerugian
= 2.400.000 / 9.400.000
= 0,25
= 25%

Materi : Aritmatika Sosial

HJ = Rp 728.000 ( 130% )

Biaya Perbaikan = Rp 105.000

Laba / Untung = 30%

Harga Beli [ HJ ] ( 100% )

HB = HJ × HB/HJ - Biaya Perbaikan

HB = Rp 728.000 × 100/130 - Rp 105.000

HB = Rp 728.000 × 10/13 - Rp 105.000

HB = Rp 56.000 × 10 - Rp 105.000

HB = Rp 560.000 - Rp 105.000

{ HB = Rp 455.000 }
