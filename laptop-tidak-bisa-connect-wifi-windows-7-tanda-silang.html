1. cara memasang wifi di laptop windows 7


Pertama, pastikan laptop kamu sudah memiliki wifi adapter di dalamnya, tenang saja, hampir semua laptop di zaman ini sudah memiliki wifi adapter di dalamnya.
Oke untuk cara memasang wifi di laptop, saya biasanya menggunakan salah satu cara, jadi menurut saya ada 2 cara:
Cara pertama :
Cari driver wifi kamu dengan cara mengetik di google "Download wifi driver for Merk laptop Tipe Laptop", contoh : "Download Wifi Driver for Asus A455L".
lalu masuk ke situs resminya dan download drivernya yang wifi.
Gunakan HP atau Laptop teman atau laptop keluarga untuk mendownloadnya.
Cara kedua.
Gunakan Driver Pack solution, cara ini cukup ampuh, tapi kamu harus mendownload file DPS (Driver pack solution) dahulu yang ukuranya relatif besar, sekitar 5-15 GB.
Kamu tinggal install, dan pasang saja drivernya secara otomatis sudah terinstall.

5. Bisa bantu?Bagaimana cara menyambungkan wifi yang menggunakan password ke laptop windows 7?​


Jawaban:

Windows 7 Buka Start Menu dan pilih Control Panel . Klik kategori Jaringan dan Internet kemudian pilih Pusat Jaringan dan Berbagi.Dari opsi di sisi kiri, pilih Ubah pengaturan adaptor . Klik kanan pada ikon untuk Koneksi Nirkabel dan klik aktifkan .

Penjelasan:

Bukanlah perkara pelik menemukan cara terbaik dan tercepat menyambung wifi ataupun hotspot ke laptop windows anda. Ini berlaku untuk Sistem operasi termasuk windows 7, 8 atau windows 10. Rasanya sudah hampir setiap perangkat komputer ataupun laptop sudah dibenamkan adapter Wi-fi didalamnya. Sama sekali tidak ribet dan hal ini bisa dilakukan dengan hanya beberapa step saja.

Setidaknya ada beberapa catatan yang perlu anda ketahui saat akan menyambungkan hotspot wifi dari perangkat lain ke laptop yang anda miliki. Jika sumber hotspotnya adalah dari perangkat anda sendiri (entah itu via modem atau Hp), sebaiknya gunakan password yang tidak mudah ditebak. Tujuannya apa? Hal ini bertujuan untuk menjaga jaringan koneksi internet anda bersifat pribadi. Jika anda membiarkannya terbuka alias tidak memakai password, maka setiap orang yang ter-cover dengan jaringan hotspot anda akan bisa dengan mudah memakai hotspot milik anda tsb. Efeknya apa? Bisa jadi paket data anda akan cepat habis dan koneksi yang anda gunakan akan mengalami penurunan dari segi kecepatan, karena telah dibagi ataupun dipakai secara bersama-sama.

Gunakanlah Wifi anda dengan bijak. Jangan biarkan wifi menyala secara terus menerus jika tidak sedang digunakan. Jika Wifi anda menyala secara kontinue, tentu saja baterai laptop anda akan cepat habis.

Apapun merk laptop anda, entah itu Acer, Dell, Toshiba, Axio atau merk lainnya, langkah yang diperlukan sama saja. Tutorial ini bisa pula diaplikasikan pada OS Windows mulai dari Windows 7, 8 atau versi Windows 10.

Perlu di ingat, tidak semua merk laptop menyediakan tombol khusus WiFi. Anda cukup beruntung jika laptop yang anda pakai ternyata menyediakan tombol tersebut, karena hal ini akan memudahkan anda dalam mengaktifkan wifi untuk hotspot di laptop anda. Cara melakukannya? Gampang, cukup menekan tombol yang ada lambang WiFinya tsb dalam beberapa detik, nanti akan ada indikator yang terlihat (biasanya berupa lampu yang menyala).

Via Tombol Fn + Function (Tombol F)

Coba perhatikan keyboard di laptop anda. Di barisan paling atas akan nampak deretan Function mulai dari F1 sampai yang terakhir F12, biasanya di salah satu tombol function tsb ada lambang Wifi-nya (berbentuk gambar signal). Letaknya memang berbeda-beda disetiap laptop. Khusus di laptop saya merek Acer (Aspire) ini, tombol yang ada lambang WiFi tersebut ada di F3.  Sedangkan tombol Wi-Fi yang ada di Laptop saya yang satunya merk Lenovo Thinkpad ada di F8.

Troubleshooting jika wifi bertanda X ( No wifi )

Paling awal, sebelum melakukan langkah lainnya. Dicoba cek keyboardnya apakah ada tombol yang terdapat wifi. ( Biasa di tombol F6 atau F2, tapi setiap laptop berbeda-beda ). Jika ada lakukan cara ini hanya jika terdapat tombol yang muncul tanda wifi ( Lihat lampiran )

Tekan tombol fn + ( tombol tersebut ), jika masih belum bisa langsung saja klik tombol tersebut

Jika cara diatas tidak bisa maka coba lakukan hal ini

Search aplikasi dalam windows yang namanya CONTROL PANELSetelah itu pilih NETWORK AND INTERNET ( pada beberapa komputer mungkin ini tidak ada, jika tidak ada langsung ke step selanjutnya )Pilih NETWORK AND SHARING CENTERPilih CHANGE ADAPTER SETTINGSMaka akan muncul seperti di gambar ( Lampiran 2 )Klik kanan pada WI-FI dan pilih ENABLEKlik kanan pada object lainnya ( Contohnya Ethernet ) dan pilih DISABLEJika sudah, silahkan menikmati akses wifi dengan cara yang biasa

** Ketika klik kanan pada opsi wifi, bisa pilih connect/disconnect untuk connect ke wifi terdekat

Jika masih tidak dapat dikoreksi harap menghubungi customer care / service center sesuai dengan tipe laptopnya
