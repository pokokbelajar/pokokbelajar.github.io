1. Di bawah ini yang benar tentang perbandingan skala termometer yaitu


Jawaban:

Perbandingan skala antara temometer Celcius, termometer Reaumur, termometer Fahrenheit, dan termometer Kelvin adalah

C : R : F : K = 100 : 80 : 180 : 100

C : R : F : K = 5 : 4 : 9 : 5

Pembahasan

Pada pembuatan termometer terlebih dahulu ditetapkan titik   tetap atas dan titik tetap bawah. Titik tetap termometer tersebut   diukur pada tekanan 1 atmosfer. Di antara kedua titik tetap tersebut   dibuat skala suhu. Penetapan titik tetap bawah adalah suhu ketika es   melebur dan penetapan titik tetap atas adalah suhu saat air mendidih.   Berikut ini adalah penetapan titik tetap pada skala termometer.

Termometer Celcius

Titik tetap bawah diberi angka 0 dan titik tetap atas diberi angka  100. Diantara titik tetap bawah dan titik tetap atas dibagi 100  skala.

Termometer Reaumur

Titik tetap bawah diberi angka 0 dan titik tetap atas diberi angka  80. Diantara titik tetap bawah dan titik tetap atas dibagi menjadi  80 skala.

Termometer Fahrenheit  

Titik tetap bawah diberi angka 32 dan titik tetap atas diberi   angka 212. Suhu es yang dicampur dengan garam ditetapkan  sebagai 0ºF. Di antara titik tetap bawah dan titik tetap atas  dibagi 180 skala

Termometer Kelvin

Pada termometer Kelvin, titik terbawah diberi angka nol. Titik   ini disebut suhu mutlak, yaitu suhu terkecil yang dimiliki benda   ketika energi total partikel benda tersebut nol. Kelvin menetapkan suhu es melebur dengan angka 273 dan suhu air mendidih  dengan angka 373. Rentang titik tetap bawah dan titik tetap   atas termometer Kelvin dibagi 100 skala.

Perbandingan skala antara temometer Celcius, termometer Reaumur, termometer Fahrenheit, dan termometer Kelvin adalah

C : R : F : K = 100 : 80 : 180 : 100

C : R : F : K = 5 : 4 : 9 : 5

17. menentukan skala termometer Tak berskala dengan membandingkan termometer berskala​


Untuk menentukan skala Termometer tak berskala bisa memakai rumus berikut ini:

(x - x₁) / (x₂ - x₁) = (y - y₁) / ( y₂ - y₁ )

dimana :

x₁ = titik tetap bawah termometer x

x₂ = titik tetap atas termometer x

y₁ = titik tetap bawah termometer y

y₂ = titik tetap atas termometer y

PEMBAHASAN

Untuk mengkonversi satuan suhu bisa di pakai perbandingan sebagai berikut ini :

°C : (°F-32) : °R : (K-273) = 5 : 9 : 4 : 5

[tex]\texttt{ }[/tex]

Untuk skala sembarang yang lain bisa di pakai perbandingan sebagai berikut ini :

(x - x₁) / (x₂ - x₁) = (y - y₁) / ( y₂ - y₁ )

dimana :

x₁ = titik tetap bawah termometer x

x₂ = titik tetap atas termometer x

y₁ = titik tetap bawah termometer y

y₂ = titik tetap atas termometer y

[tex]\texttt{ }[/tex]

Contoh Soal:

Termometer x dan y menunjukkan angka 30°X dan -20°Y saat mengukur air membeku , saat mengukur air mendidih masing-masing menunjukkan 150°X dan 250°Y. Jika suatu benda dapat diukur suhunya sebesar 60°X maka skala yang ditunjukkan pada termometer y adalah ?

Penyelesaian:

(x - x₁) / (x₂ - x₁) = (y - y₁) / ( y₂ - y₁ )

(60 - 30) / (150 - 30) = (y - (-20)) / ( 250 - (-20) )

(y + 20) = 67,5

y = 47,5

[tex]\texttt{ }[/tex]

Kesimpulan :

60°X = 47,5°Y

19. ubahlah skala termometer dibawah ini menjadi skala termometer yang di inginkan​


Matapelajaran : IPA (Fisika)

Kelas : VII ( 7 )

Materi :  Suhu

Kata kunci : Termometer

Kode soal :  6

Diketahui  :

x min = 0°

x rata = 20°

x max = 50°

y min = -10°

y max = 140°

Ditanya :

y rata = ...°?

Jawab :

untuk mencari rata - rata y maka kita gunakan rumus :

⇔ x - x min / x max - x min  =  y - y min / y max - y min

⇔ 20 - 0 / 50 - 20 = y - (-10)/ 140 - (-10)

⇔ 20/30 = 10y/ 150

⇔ 10y . 20 = 30 . 150

⇔ 200y  = 4500

⇔ y = 4500/200

⇔ y = 22,5°

jadi , rata - rata suhu y adalah 22,5°

27. Tentukan skala Termometer tak berskala dengan membandingkan dengan Termometer berskala #MohonDibantuYa


Cara menentukan skala thermometer tak berskala dengan membandingkan deng thermometer celcius.


Pembahasan.

Thermometer merupakan alat untuk mengukur temperatur .

Ada banyak sekali model themometer, namun yang sering kali kita ketahui diantaranya adalah thermometer berskala Celcius, Reamur, Fahreinhet dan Kelvin.

Temperatur titik terendah Celcius : 0°CReamur : 0°RFahreinhet : - 32°FKelvin : 273 KTemperatur titik tertinggiCelcius : 100°CReamur : 80°RFahreinhei : 212°RKelvin : 373KPerbandingan skala

Celcius :Reamur : Fahreinfet : Kelvin = 5 : 4 : 9 : 5

untuk menentukan konversi dari satu skala ke bentuk skala lainnya tersebut dapat digunakan persamaan umum :

[tex]\boxed{\frac{(X_{t}-X_{0} ) }{(X_{i} -X_{0}) }=\frac{(Y_{t}-Y_{0} ) }{(Y_{i} -Y_{0}) } }[/tex]


Keterangan :

Xt : suhu yang terukur pada thermometer X

Xi : titik didih pada thermometer X

Xo : titik beku pada Thermometer x

Yt : suhu yang terukur pada thermometer Y

Yi : titik didih pada thermometer Y

Yo : titik beku pada Thermometer Y


Jadi, jika ingin membandingkan themometer yang tak berskala dengan thermometer yang berskala, yaitu dengan cara :

Ukur titik beku thermometer yang yak berskalaUkur titik didik thermometer yang tidak berskalaBandingkan dengan thermometer celcius menggunakan persamaan diatas.

yaitu :

[tex]\boxed{\frac{(X_{t}-X_{0} ) }{(X_{i} -X_{0}) }=\frac{(C_{t}-C_{0} ) }{(C_{i} -C_{0}) } }[/tex]


Kesimpulan :

Cara menentukan skala thermometer tak berskala dengan membandingkan deng thermometer celcius.
