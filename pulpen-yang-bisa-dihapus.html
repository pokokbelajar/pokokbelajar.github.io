7. Harga sebuah pulpen adalah 3 kali harga penghapus. Jika harga 2 pulpen dan 2 penghapus tidak lebih dari Rp. 8.000. maka harga tertinggi penghapus adalah....​


Jawab:

Harga penghapus tertinggi adalah Rp1.000.

Penjelasan dengan langkah-langkah:

---- Menelaah informasi

Pulpen = x

Penghapus = y

Tidak lebih dari = ≤

2 pulpen + 2 penghapus ≤ 8000 = 2x + 2y ≤ 8000 ==> Persamaan yang akan dihitung

Harga pulpen = 3 kali harga penghapus ==> x = 3y

---- Perhitungan

2x + 2y ≤ 8000

2(3y) + 2y ≤ 8000

6y + 2y ≤ 8000

8y ≤ 8000

y ≤ 1000

Karena harga penghapus tidak lebih dari 1000 rupiah, maka harga tertingginya adalah Rp1.000.

9. Kuis [+50]: (Sori tadi salah soal) Harga 1 pulpen, 1 penggaris, 1 penghapus adalah Rp. 51.000,- Harga 11 pulpen, 22 penggaris, 33 penghapus adalah Rp. 891.000,- Harga 109 pulpen, 210 penggaris, 1.200 penghapus adalah Rp. 16.590.000,- Berapa harga 2.021 pulpen, 2.022 penggaris, 2.023 penghapus?


Penjelasan dengan langkah-langkah:

SPLTV

SPLTV singkatan dari sistem persamaan linear tiga variabelSPLTV memiliki tiga jenis variabel SPLTV juga mempunyai tiga Himpunan Penyelesaian atau HPCara mencari Himpunan Penyelesaian dari SPLTV bisa menggunakan cara Metode Eliminasi, Substitusi, Campuran, Determinan dan yang lainnya.

Maka,

Mencari model matematika nya :

misalkan :

a = pulpen b = penggaris c = penghapus

Sehingga, model matematika :

a + b + c = 51.000 → persamaan 1

11a + 22b + 33c = 891.000 → persamaan 2

109a + 210b + 1.200c = 16.590.000 → persamaan 3

Menggunakan cara Metode Eliminasi

a + b + c = 51.000

11a + 22b + 33c = 891.000 → bagi dengan (11)

a + 2b + 3c = 81.000

109a + 210b + 1.200c = 16.590.000

Eliminasi persamaan ke 1 dan 2

a + b + c = 51.000

a + 2b + 3c = 81.000

_________________ _

-b - 2c = -30.000

b + 2c = 30.000 → persamaan 4

Eliminasi persamaan 1 dan 3

a + b + c = 51.000 |×109|

109a + 210b + 1.200c = 16.590.000 |×1|

109a + 109b + 109c = 5.559.000

109a + 210b + 1.200c = 16.590.000

_____________________________ _

-101b - 1.091c = -11.031.000

101b + 1.091c = 11.031.000 → persamaan 5

Eliminasi persamaan 4 dan 5

b + 2c = 30.000 |×101|

101b + 1.091c = 11.031.000 |×1|

101b + 202c = 3.030.000

101b + 1.091c = 11.031.000

_____________________ _

-889c = -8.001.000

c = 8.001.000 / 889

c = 9.000

substitusikan nilai c = 9.000 ke :

b + 2c = 30.000

b + 2(9.000) = 30.000

b = 30.000 - 18.000

b = 12.000

Substitusikan nilai b dan c ke :

a + b + c = 51.000

a + 12.000 + 9.000 = 51.000

a = 51.000 - 12.000 - 9.000

a = 30.000

Jadi, harga persatuannya adalah

a = pulpen = Rp 30.000b = penggaris = Rp 12.000c = penghapus = Rp 9.000

Maka, Harga dari :

= 2.021 pulpen + 2.022 penggaris + 2.023 penghapus

= 2.021a + 2.022b + 2.023c

= 2.021 (30.000) + 2.022 (12.000) + 2.023 (9.000)

= 60.630.000 + 24.264.000 + 18.207.000

= Rp 103.101.000

Jadi, total keseluruhan nya adalah Rp 103.101.000,00.–
