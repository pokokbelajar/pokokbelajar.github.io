1. sebutkan 3 akibat tidak adanya persatuan dan kesatuan di masyarakat


Kategori Soal : Bab Akibat Tidak adanya Persatuan Dan Kesatuan di Masyarakat.
Kelas : 8 SMP/ VIII SMP

Pembahasan :

Akibat Akibat Karena Tidak Adanya Persatuan Dan Kesatuan di Masyarakat.
Yaitu :

-> Muncullah Adanya Adu-Domba { Adu Mulut }
-> Muncullah Konflik.
-> Masyarakat menjadi tidak rukun / harmonis 
-> Adanya Perselisihan antara Masyarakat.
-> Adanya Pertengkaran/pertentangan antara masyarakat. 
-> Kurang adanya Kebersamaan 
-> Kurang adanya Kerja sama 
-> Masyarakat tidak bisa menghargai dan menghormati orang lain.
