Langakah² Penyelesaian

Ingat faktorisasi pada persamaan kuadrat ax²+bx+c=0 dengan a=1 berikut.

x²+bx+c=0 maka (x+p)(x+q)=0 dengan p+q=b dan p×q=c.

Ingat juga:

a(b-c) = ab - ac

Diketahui persamaan kuadrat x²-5x-36=0 maka nilai x yang memenuhi persamaan kuadrat tersebut diperoleh sebagai berikut.

x²-5x-36=0

x²+4x-9x-36=0

x(x+4)-9(x+4)=0

(x+4)(x-9)=0

x+4=0 ∨ x-9=0

x=-4 ∨ x=9

Dengan demikian, himpunan penyelesaian dari x²-5x-36=0 adalah HP={x|x=-4 atau x=9, x∈ℝ}.

18. dengan cara melengkapi kuadrat sempurna tentukan akar akar dari penyelesaian dari bentuk a. x²+5x-36=0 b. -3x²-5x+2=0​


Akar-akar dari persamaan kuadrat berikut dengan cara melengkapi kuadrat sempurna:

a. x² + 5x – 36 = 0 adalah –9 atau 4.

b. –3x² – 5x + 2 = 0 adalah 2/3 atau 1.

♦ Pembahasan:

Persamaan Kuadrat (PK) adalah suatu persamaan yang variabelnya memiliki pangkat tertinggi yaitu sama dengan dua. Bentuk umum dari PK adalah sebagai berikut:

[tex]\boxed{\bold{ax^{2} + bx + c = 0}}[/tex] → di mana a, b, dan c ∈ bilangan bulat dan a ≠ 0

Akar-akar atau penyelesaian dari ax² + bx + c = 0 adalah nilai x yang memenuhi persamaan tersebut.

Cara menentukan akar persamaan kuadrat ada 3 cara, yaitu:

MemfaktorkanMelengkapi Kuadrat SempurnaRumus Kuadratik (ABC)

♦ Diketahui:

Persamaan kuadrat

a. x² + 5x – 36 = 0

b. –3x² – 5x + 2 = 0

♦ Ditanya:

Menentukan akar-akar penyelesaian persamaan kuadrat tersebut dengan menggunakan rumus melengkapi kuadrat sempurna

♦ Jawab:

Soal ( a )

x² + 5x – 36 = 0

x² + 5x = 36

[tex]\sf{x^{2} + 2 \times x \times \dfrac{5}{2} = 36}[/tex]

[tex]\sf{x^{2} + 2 \times x \times \dfrac{5}{2} + \bigg( \dfrac{5}{2} \bigg) ^{2} = 36}[/tex]

[tex]\sf{x^{2} + 5x + \dfrac{5^{2}}{2^{2}} = 36}[/tex]

[tex]\sf{x^{2} + 5x + \dfrac{25}{4} = 36}[/tex]

[tex]\sf{x^{2} + 5x + \dfrac{25}{4} = 36 + \dfrac{25}{4}}[/tex]

[tex]\sf{\bigg( x + \dfrac{5}{2} \bigg) ^{2} = 36 + \dfrac{25}{4}}[/tex]

[tex]\sf{\bigg( x + \dfrac{5}{2} \bigg) ^{2} = \dfrac{169}{4}}[/tex]

[tex]\sf{x + \dfrac{5}{2} = ± \dfrac{13}{2}}[/tex]

.

[tex]\sf{x + \dfrac{5}{2} = \dfrac{13}{2}}[/tex]

[tex]\sf{x = \dfrac{13}{2} - \dfrac{5}{2}}[/tex]

[tex]\sf{x = \dfrac{13 - 5}{2}}[/tex]

[tex]\sf{x = \dfrac{8}{2}}[/tex]

[tex]\sf{x = 4}[/tex]

= -4

a) x² - 100 = 0

( x + 10 ) ( x - 10 )

x1 = -10 dan x2 = +10

cara :

a² - b² = ( a + b ) ( a - b )

a² = x² , a = x

b² = 100 , b = 10

b) x² + 13 x + 40 = 0

( x + 8 ) ( x + 5 )

x1 = -8 dan x2 = -5

cara :

1. carilah bilangan faktor dari 40 yang bila dijumlah hasilnya +13

2. faktor 40 : 1 x 10 , 2 x 20, 4 x 10, 5 x 8

3. lihat angka yang bila dijumlah menghasilkan angka 13

4. yang menghasilkan angka 13 yaitu +5 dengan +8

c) x² + 5x - 36 = 0

( x + 9 ) ( x - 4 )

cara :
