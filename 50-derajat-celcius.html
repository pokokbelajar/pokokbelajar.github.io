30. 50 derajat Celcius ke derajat Fahrenheit​


50 °C di konversikan menjadi Fahrenheit​ adalah 122 F.

Pembahasan :

Diketahui : 50 °C

Ditanya : Konversi ke satuan suhu Fahrenheit ?

Penyelesaian soal :  

Suhu atau temperatur merupakan derajat panas atau dingin dari suatu benda. Beberapa skala satuan suhu, misalnya Celcius (C), Reamur (R), Fahrenheit (F), dan Kelvin (K).  

Mengubah satuan Celcius ke Fahreinheit dapat dengan menggunakan rumus berikut ini :  

50 °C = { [tex]\frac{9}{5}[/tex] 50 + 32 } F

         = { [tex]\frac{450}{5}[/tex] + 32 } F

         = { 90 + 32 } F

         = 122 F

⇒ Kesimpulan, maka 50 °C di konversikan menjadi Fahrenheit​ adalah 122 F.
