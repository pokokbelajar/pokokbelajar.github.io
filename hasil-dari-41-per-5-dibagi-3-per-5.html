(1.) (2x - 4)⁴
Pangkat 4, n = 4
===============================
CARI KOEFISIEN SUKU KE-4 (a = 4)
Suku ke-4,
r = a - 1
dimana a = suku keberapanya
r = 4-1,
r = 3
Gunakan kombinasi (ⁿCr)
ⁿCr = n!/(r!(n-r)!)
⁴C₃ = 4!/(3!(4-3)!)
⁴C₃ = 4!/(3!·1!)
⁴C₃ = 4!/(3!)
⁴C₃ = 4
------------------------------------------
Saat suku ke-a, suku kiri pangkat n+1 - a
Saat suku ke-4, suku kiri pangkat 4+1 - 4
(2x)⁴⁺¹⁻⁴ = (2x)⁵⁻⁴ = (2x)¹ = 2x
Saat suku ke-a, suku kanan pangkat a-1
Saat suku ke-4, suku kanan pangkat 4-1
dan (-4)⁴⁻¹ = (-4)³ = -64
------------------------------------------
Maka
Untuk mencari suku ke-4,
kombinasi · suku kiri · suku kanan =
⁴C₃ · (2x)¹(-4)³
= 4(2x)(-64)
= -512x
Koefisien suku ke-4 = -512
===============================
CARI KOEFISIEN SUKU KE-2 (a = 2)
Suku ke-2
r = a - 1
dimana a = suku keberapanya
r = 2-1
r = 1
Gunakan kombinasi (ⁿCr)
ⁿCr = n!/(r!(n-r)!)
⁴C₁ = 4!/(1!(4-1)!)
⁴C₁ = 4!/(1!·3!)
⁴C₁ = 4!/(3!)
⁴C₁ = 4
------------------------------------------
Saat suku ke-a, suku kiri pangkat n+1 - a
Saat suku ke-2, suku kiri pangkat 4+1 - 2
(2x)⁴⁺¹⁻² = (2x)⁵⁻² = (2x)³ = 8x³
Saat suku ke-a, suku kanan pangkat a-1
Saat suku ke-2, suku kanan pangkat 2-1
dan (-4)²⁻¹ = (-4)¹ = -4
------------------------------------------
Maka
Untuk mencari suku ke-2,
kombinasi · suku kiri · suku kanan =
⁴C₁ · (2x)³(-4)¹
= 4(8x³)(-4)
= -128x³
Koefisien suku ke-2 = -128
