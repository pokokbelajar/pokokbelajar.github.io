16. Tuliskan dan jelaskan fungsi yang terdapat pada tab home, tab page layout, tab references, tab mailings, tab review, tab view.​


Penjelasan:

1. Home

Tab ini memiliki fungsi untuk menyunting properti-properti tulisan seperti halnya mengganti font yang digunakan dalam lembar dokumen Microsoft Word 2007, ukuran font yang diinginkan atau bahkan hingga penempatan paragraf sehingga tulisan akan menjadi lebih rapi.

2. Insert

Tab Insert ini digunakan untuk mempercantik tampilan dokumen, untuk memasukkan gambar/foto ke dalam lembar kerja, Tab Insert ini bisa dibilang tab yang mendukung Tab Home sehingga lembar kerja yang kita buat menjadi lebih menarik dan rapi.

3. Page layout

Page Layout berbeda dengan Insert, Insert mengacu pada isi dokumen, sedangkan Page Layout mengacu pada lembar itu sendiri sehingga bisa dibuat semenarik mungkin agar lembar tidak terkesan polos.

4. Referances

Tab Referensi ini berfungsi sebagai tambahan tulisan/catatan penting pada sebuah lembar kerja, misal untuk menambahkan Daftar Isi, Catatan Penting di setiap lembar kerja, dan masih banyak lagi fungsi yang masih bisa digunakan.

5. Mailings

Tab Mailings ini dapat digunakn sebagai Mail Merger dan kebutuhan surat-menyurat lainnya sehingga akan lebih efisien jika menggunakan Mailings untuk membuat surat.

6. Review

Tab review ini sangat berguna untuk mereviewkan kembali seperti melihat ulang untuk mencari kesalahan penggunaan tata bahasa, memberikan informasi mengenai jumlah kata atau bahkan bisa lagi untuk memproteksi dokumen Anda.

7. View

Tab ini memberikan bantuan tampilan pada lembar kerja Microsoft Office Word, seperti perbesaran lembar kerja, memberikan tampilan buku, dan masih banyak lagi yang bisa digunakan untuk mencoba dokumen kita dalam berbagai bentuk tampilan.

25. jelaskanlah kegunaan dari tab home


kembali ke awal baris1.) Submenu Clipboard

Submenu Clipboard digunakan untuk menampung sementara file-file hasil copy, hasil pemotongan, serta format painter. Selain itu, submenu ini digunakan untuk menampilkan kembali hasil proses copy atau pemotongan yang telah dilakukan.

2) Submenu Font

Submenu Font berisi ikon-ikon untuk fungsi pengaturan model/jenis huruf, ukuran huruf, menebalkan huruf, mencetak huruf miring, membuat huruf turun, naik setengah karakter, menghapus huruf, memberi warna huruf, memberi background huruf, dan membuat huruf menjadi besar atau kecil satu karakter.

3) Submenu Paragraph

Kegunaan submenu Paragraph adalah untuk pengaturan dan pemberian bullet and numbering, pengaturan outline number, melakukan decrease dan increase indentasi paragraf, perataan teks atau Allignment (yang terdiri dari align left, right, dan justify), pengaturan jarak baris (spasi), pemberian color (fill color), pemberian bingkai (border), dan pengurutan data (sort).

4) Submenu Style

Submenu Style digunakan untuk mengatur penampilan teks (di antaranya dengan pilihan normal, tidak berspasi, serta heading).

5) Submenu Editing

Submenu Editing digunakan untuk proses pengeditan teks.

28. Mengidentifikan ikon ikon pada tab home dan tab page layout?


ikon-ikon pada tab home antara lain:
- Copy: icon untuk menyalin teks/ objek.
- Paste: icon untuk menempelkan teks/ objek yang sebelumnya disalin.
- Italic: icon untuk memberi efek miring pada teks.
- Bold: icon untuk memberi efek tebal pada teks.
- Align Text Right: icon untuk mengatur perataan teks di arah kanan.

ikon-ikon pada tab page layout antara lain:
- Margins: ikon untuk mengatur margin halaman.
- Orientation: ikon untuk mengatur orientasi halaman.
- Size: ikon untuk mengatur ukuran kertas.
- Columns: ikon untuk mengatur kolom koran pada halaman.
- Page Border: ikon untuk mengatur bingkai pada halaman.
