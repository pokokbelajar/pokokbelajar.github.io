**Cari nilai orde maksimum (m) :
d.sin θ = m.λ
3,33 x 10⁻⁶ . sin 90° = m . 6 x 10⁻⁷
m =3,33 x 10⁻⁶. (1)
          6 x 10⁻⁷
m = 5,56 ⇒ 6

Jadi nilai orde maksimumnya mendekati 6.

21. Seberkas cahaya kuning dengan panjang gelombang 589 nm. energi cahaya kuning tersebut dalam satuan ev adalah…


Jawaban:

Diketahui:

= 589 nm = 5,89 x 10^-7 m

n = 2 foton

1 eV = 1,6 x 10^-19 J

kecepatan cahaya, c = 3 x 10^8 m/s

Tetapan Planck, h = 6,6 x 10^-34 Js

Ditanyakan:

E?

Pembahasan:

Foton adalah partikel elementer dalam fenomena elektromagnetik. Biasanya foton dianggap sebagai pembawa radiasi elektromagnetik, seperti cahaya, gelombang radio, dan Sinar-X.

1. Mencari frekuensi.

f = c/

f = (3 x 10^8) / (5,89 x 10^-7)

f = 0,5 x 10^15 Hz

f = 5 x 10^14 Hz

2. Mencari energi cahaya.

E = n x h x f

E = 2 x (6,6 x 10^-34) x (5 x 10^14)

E = (13,2 x 10^-34) x (5 x 10^14)

E = 66 x 10^-20 J

E = (66 x 10^-20) / (1,6 x 10^-19) eV

E = 41,25 x 10^-1

E = 4,125 eV

Jadi, energi cahaya kuning tersebut adalah 4,125 eV.


22. Seberkas cahaya monokromatis dengan panjang gelombang 500 nm tegak lurus pada kisi difraksi jika kisi memiliki 400 garis tiap cm dan sudut deviasi sekitar 30 derajat banyaknya garis terang yang terjadi pada layar adalah


Mapel : Fisika

Kelas : 12

Bab : Optika Fisis
______________

Diketahui :
→ λ = 500 nm = 500 x 10^(-9)

→ N = 400 garis/cm = 40.000 garis/m

→ θ = 30°

Ditanya :
→ n =....?

Penyelesaian :
____________

⏩ Tentukan Jarak antar Celah kisi (d)

d = 1 / N
d = 1 / 40.000
d = 2,5 x 10^(-5) m

Sehingga, Jarak Celah Ke layar adalah :

n • λ = d • sin θ
n • 500 x 10^(-9) = 2,5 x 10^(-5) • sin 30°

[tex]n = \frac{2.5 \times {10}^{ - 5} \times \frac{1}{2} }{500 \times {10}^{ - 9} } \\ \\ n = \frac{2.5 \times {10}^{ - 5} \times \frac{1}{2} }{5 \times {10}^{ - 7} } \\ \\ n = 0.25 \times {10}^{7 - 5} \\ \\ n = 0.25 \times {10}^{2} \\ \\ n = 0.25 \times 100 \\ \\ n = 25[/tex]
Jadi, banyak Garis terang yang terjadi pada layar adalah 25 ✔
