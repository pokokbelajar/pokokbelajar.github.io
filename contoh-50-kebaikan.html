17. (+50) KuMat lagiKuis MatematikaSelisih dari kuadrat dua bilamgan sama dengan perkalian antara jumlah dan selisih kedua bilangan tersebut. Apakah pernyataan tersebut benar? Baik benar maupun tidak benar, buktikan secara aljabar, tanpa menggunakan contoh/sampel berupa angka. ​


Penjelasan dengan langkah-langkah:

a² - b² = (a - b)(a + b)

a² - b² = a(a + b) - b(a + b)

a² - b² = a² + ab - ab - b²

a² - b² = a² + 0 - b²

a² - b² = a² - b²

t e r b u k t i ....

Pembuktian:

misalkan kedua bilangan tersebut x dan y, maka diperoleh:

[tex]~[/tex]

x² - y² = (x + y)(x - y)

x² - y² = (x)(x) - (x)(y) + (y)(x) - (y)(y)

x² - y² = x² - xy + xy - y²

x² - y² = x² + (-1 + 1)xy - y²

x² - y² = x² + (0)xy - y²

x² - y² = x² + 0 - y²

x² - y² = x² - y² ------> terbukti benar
