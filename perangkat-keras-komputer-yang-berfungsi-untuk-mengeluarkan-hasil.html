Perangkat keras komputer yang berfungsi untuk mengeluarkan hasil pemrosesan dan pengolahan data disebut

A. output device

Pembahasan:

Sistem komputer secara umum terdiri dari software (perangkat lunak) dan hardware (perangkat keras).

Perangkat keras atau hardware adalah perangkat fisik, termasuk bagian-bagian atau komponen dari komputer. Perangkat keras dapat dilihat kasat mata dan merupakan perangkat elektronik.

Perangkat keras dapat dibagi menjadi:

A. Output device (perangkat keluaran), yang menghasilkan keluaran proses perngolahan data, berupa tampilan, suara atau cetakan. Misalnya adalah speaker, monitor, atau printer.

B. Storage device (perangkat penyimpanan), yang digunakan untuk menyimpan data yang digunakan. Misalnya adalah flash disk, hard disk, disket dan compact disk (CD).

C. Input device (perangkat masukan) yang menerima input dari pengguna, yang akan digunakan dalam pengolahan data. Misalnya adalah keyboard, mouse dan mikrofon.

D. Processing device (perangkat pengolah) yang melakukan pengolahan data dari input menjadi input. Misalnya adalah mikroprosesor dan prosesor grafis.

Pembahasan  Perangkat keras (hardware) adalah komponen fisik dari sebuah komputer seperti keyboard, CPU, mouse, RAM (Random Access Memory), Hard disk (HDD), dan lain-lain. Memory bukan termasuk perangkat keras. Namun, termasuk perangkat keras dengan nama khusus misalnya RAM. Perangkat keras komputer terbagi menjadi 4, yaitu perangkat input, perangkat output, perangkat processing, dan media penyimpan. Perangkat input berfungsi untuk memasukkan data dan program ke dalam komputer. Media penyimpan berfungsi untuk menyimpan data atau program komputer. Contoh: Hard disk, CD, DVD, USB Flash disk, dan lain-lain. Mouse berfungsi sebagai alat pointer untuk mengakses melalui layar monitor. Scanner untuk mengubah data hard copy menjadi soft copy yang dapat diolah di komputer. Joystick adalah tuas kendali yang dapat digerakkan ke segala arah sebagai pelengkap permainan video. Mouse dapat digunakan untuk mengoperasikan beberapa perintah seperti click, double klik, right click, dan drag. Drag dioperasikan dengan mengarahkan pointer mouse pada objek, kemudian menekan dan menahan tombol kiri mouse sambil digerakkan ke tempat yang diinginkan lalu dilepaskan. Keyboard berdasarkan fungsi dasarnya dibagi menjadi 4, yaitu Numeric key, function key, typewriter key dan special function key. Numeric key menampilkan angka, tanda baca titik dan operasi hitung. Perangkat output (output device) berfungsi untuk menyampaikan atau menampilkan informasi. Contoh: Monitor, speaker, dan printer.  ROM singkatan dari Read Only Memory merupakan salah satu memori yang ada dalam komputer yang bersifat permanen dan hanya bisa dibaca saja.
