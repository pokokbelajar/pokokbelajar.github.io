Prinsip dan cara kerja proxy server adalah sebagai perantara antara client dan server tujuan. Saat client menggunakan layanan proxy, permintaan file atau data yang dikirim oleh client akan diteruskan terlebih dahulu ke proxy server. Kemudian, proxy server akan meneruskan permintaan tersebut ke server tujuan.
