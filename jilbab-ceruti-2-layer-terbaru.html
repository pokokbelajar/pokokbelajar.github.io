6. 1) Jelaskan cara membuat file baru pada Photopea! 2) Jelaskan cara membuat layer baru pada Photopea! 3) Tuliskan cara memberikan warna pada sebuah layer (shape pada layer)!


Jawaban:

1. A. Kita dapat memilih menu File > New. Atau untuk mempermudah pengerjaan kita dapat pula menekan tombol sesuai dengan Shortkey yang tersedia (pada gambar ditandai dengan kotak Merah).

2. Kita akan masuk ke kotak option new sebelum masuk ke lembar kerja dokumen. Pilih Nama Dokumen sesuai selera. Secara default nama dokumen adalahUntitled-1.

3. Pada Options Preset, kita akan disuguhkan dengan berbagai jenis dan tipe ukuran kertas. Semisal jika kita ingin memilih ukuran kertas A4, kita dapat memilih International Paper.

4. Selanjutnya kita dapat memilih size document, menentukan panjang (Height) dan lebar (Widht) suatu lembar kerja sesuai keinginan kita, atau sering dikenal dengan istilah Custom.

5. Setelah resolution (tingkat ketajaman gambar) diatur, kita dapat pula (jika diperlukan) memilih mode warna dan warna background. Lalu klik OK.

2. Ada 3 cara yaitu

-. Cara Pertama : Panel Layer

untuk membuka panel layer ini sobat melalui menu Windows->Layer. Dan pada panel tersebut, silakan sobat cari ikon new layer.

-. Cara Kedua : Menu

menggunakan menu. Photoshop sendiri memiliki banyak menu yang memiliki fungsi nya masing-masing. Sedangkan menu yang digunakan untuk membuat layer baru sendiri ada di Layer->New->Layer

-. Cara ketiga Shortcut

Tekan Ctrl+Shift+N pada keyboard.

3. Dengan 2 cara yaitu

1. File->Open atau dengan Ctrl+O

2. Pilih warna foreground dahulu untuk memberi warna. Foreground tersedia pada toolbox, klik warnanya lalu akan muncul pop-up color picker yang berguna untuk mengatur warna, pilih warna sesuai keinginan kita.

3. Klik Paint Bucket Tool pada toolbox, dengan icon ember yang menumpahkan cat air.
