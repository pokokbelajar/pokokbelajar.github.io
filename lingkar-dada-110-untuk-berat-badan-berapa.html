25. Berat Ani ditambah berat Risma adalah 112 kg. Berat Ani ditambah berat Nana adalah 114 kg. Berat Risma ditambah berat Nana adalah 110 kg. Jumlah berat badan ketiga orang tersebut adalah....kg


Jumlah berat badan mereka yaitu 166 kg

Semoga membantu yaa^^Mata Pelajaran : Matematika
Kelas : XI
Kategori : Sistem Persamaan dan Sistem Pertidaksamaan
Kata Kunci : Sistem Persamaan Linear Tiga Variabel

Penyelesaian Sistem Persamaan Linear Tiga Variabel ada beberapa metode
1) eliminasi
2) substitusi
3) Gabungan antara eliminasi dan substitusi
4) Determinan matriks
5) Invers matriks
6) Eliminasi Gauss Jordan

Pada kasus diatas, misalkan umur Ani adalah a tahun, umur Risma adalah r tahun, dan umur Nana adalah n tahun. Maka diperoleh sistem persamaan linear tiga variabel, yaitu
(1) a + r = 112
(2) a + n = 114
(3) r + n = 110
Pada kesempatan ini, saya berikan dua cara.
Cara 1
Eliminasi a dari persamaan (1) dan (2), diperoleh r - n = - 2
sebut saja persamaan hasil ini sebagai persamaan (4), yaitu r - n = - 2
Eliminasi r dari persamaan (3) dan (4)
(3) r + n = 110
(4) r - n = - 2
diperoleh 2r = 108, maka r = 54
Substitusikan r = 54 ke persamaan (3) dieproleh 
54 + n = 110, maka n = 56
Substitusikan r = 54 ke persamaan (1) dieproleh
a + 54 = 112, maka a = 58
Jadi umur Ani adalah 58 tahun, umur Risma adalah 54 tahun, dan umur Nana adalah 56 tahun.
Jumlah umur mereka adalah 58 + 54 + 56 = 168 tahun

Cara 2
Jumlahkan ketiga persamaan tersebut
(1) a + r = 112
(2) a + n = 114
(3) r + n = 110
dieproleh 2a + 2r + 2n = 336
dengan membagi 2 diperoleh
a + r + n = 168
Jadi, jumlah umur mereka adalah 168 tahun.
