2. Cara merubah settingan windows shortcut pada desktop di Wd 10​


Windows Key + Arrow kiri – untuk meletakkan windows di kiri layar

Windows Key + Arrow kanan – untuk meletakkan windows di kanan layar.

Windows Key + Arrow atas – untuk meletakkan windows di bagian atas layar.

Windows Key + Arrow bawah – untuk meletakkan windows di bawah layar.

Hal ini sebenarnya juga bisa Anda lakukan dengan mouse, Anda tinggal menggeser window yang ingin Anda letakkan ke salah satu sisi, maka window akan menyesuaikan ukuran sesuai dengan Anda meletakkanya. Namun, dengan menggunakan shortcut, hal itu akan terasa lebih mudah.

Menyesuaikan Tema WordPress dengan 3 Cara

menyesuaikan tema WordPress dengan:

1. Kustomisasi Tema WordPress dengan Penyesuai Tema

>Akses Dasbor Admin WordPress Anda;

>Arahkan ke Appearance dan pilih Customize dari menu dropdown;

>Akses penyesuai tema WordPress melalui dasbor admin WordPress

>Anda akan diarahkan ke Penyesuai Tema tempat Anda dapat membuat perubahan;

>Sesuaikan tema Anda melalui Penyesuai Tema WordPress

Klik Simpan & Terbitkan untuk mengonfirmasi perubahan.

>Catatan: Opsi penyesuaian (misalnya, membuat perubahan pada latar belakang, font & gaya, header, footer, sidebar, dll.) bervariasi tergantung pada tema.

2.Kustomisasi Tema WordPress dengan Visual Composer

Visual Composer adalah salah satu pembuat situs web paling canggih, karena menawarkan respons otomatis ke semua perangkat, kode bersih yang menghasilkan pemuatan yang lebih cepat, dan kinerja keseluruhan yang lebih baik untuk situs web Anda! Namun demikian, editor front-end kompatibel dengan semua tema dan plugin WordPress paling populer, seperti Yoast dan WooCommerce.

3.Sesuaikan Tema WordPress dengan Editor Tema

Untuk mengakses Editor Tema:

>Buka Dasbor Admin WordPress Anda;

>Arahkan ke Appearance dan pilih Theme Editor dari menu dropdown;

>Dari sini Anda dapat mengedit file style.css (tampilan visual tema) dan functions.php (fitur tema) ;

>Jika Anda ingin menambahkan CSS khusus, Anda dapat mengedit dan melihat langsung perubahan CSS di Editor CSS bawaan.Menyesuaikan tema WordPress Anda melalui Editor Tema

>Catatan: Jika Anda membuat perubahan pada file tema asli, Anda berisiko tinggi kehilangannya setelah pembaruan (penting untuk menjaga keamanan) yang merupakan alasan lain untuk menggunakan tema anak untuk menerapkan perubahan apa pun pada file tema Anda.

Tampilan

Beberapa tampilan di Windows 10 akan jauh berbeda dengan tampilan di Windows 7.


Karena Windows 10 dibuat untuk mengakomodir tampilan di PC dan tablet sedangkan Windows 7 hanya untuk pc maka ada beberapa tampilan yang dimaksimalkan.


Beberapa perbedaan tampilan tersebut yaitu di Start Menu, File Explorer, dan tambahan Notifications Area


Tampilan Start Menu

di Windows 10 terlihat lebih modern dengan icon yang mengotak yang disebut sebagai tampilan metro.


Sedangkan di Windows 7 tampilan start menu lebih sederhana dengan menampilkan daftar aplikasi dengan beberapa folder atau file yang dapat diakses langsung dari Start Menu.


Tampilan metro yang lebih menarik juga akan membuat konsumsi ram akan bertambah karena memungkinkan pembaruan informasi dari aplikasi yang kita atur tampil di Start Menu.


Perubahan tampilan Start Menu yang sudah familiar di Windows 7, membuat pengguna Windows 10 harus membiasakan diri menggunakan tampilan yang baru.


Tampilan File Explorer

Penambahan pilihan Quick Access dan One Drive pada tampilan di File Explorer dan menghilangnya tambahan informasi pada pada bagian bawah saat akses ke folder atau file menjadi perbedaan dengan tampilan File Explorer di Windows 7.


Quick Access akan menampilkan file atau folder yang sering kita akses atau terakhir kali diakses.


Hadirnya Quick Access memang akan memudahkan untuk mengakses kembali file atau folder yang sering kita buka, tapi juga akan menggangu privasi.


Untuk menonaktfikan Quick Access menampilankan file atau folder yang sering kita akses,

dari segi tampilan windows 10 lebih ungul dari windows 7 dalam segi penampilan dekstop yang lebih berwarna,simple, dan elegan
3. Lebih Responsif di banding windows 7
banyak orang yang bilang windows 10 lebih berat dari pada windows yang sebelum nya, dengan ram 2gb dan core duo sistem bisa berjalan dengan baik,lebih lancar membuka aplikasi di bandingakan win 7 ,dan responsif di bandingkan win 7
4. Segi keamanan win10 vs win7
dari segi kemanan windows 10 lebih unggul dari pada win 7 di karenakan pada windows 10 mendapatkan sistem keamanan bawaan berupa antivirus yang bernama "windows defender" yang sudah terpasang otomatis saat os di pasang ke pc, jadi tidak perlu proteksi tambahan yang lain nya bila anda sudah merasa aman dengan antivirus bawaan windows 10

Shortcut Command Prompt

Untuk yang aktif menggunakan command prompt, shortcut keyboard ini akan mempercepat pekerjaan Anda.

Ctrl + F: Membuka dialog Find untuk mencari output Command Prompt.

Alt + F4: Menutup jendela Command Prompt.
