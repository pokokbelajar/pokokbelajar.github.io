Pembahasan

Bilangan dapat dibagi menjadi dua yaitu:

Bilangan ganjil atau bisa dedefinisikan sebagai bilangan yang tidak habis dibagi dua atau meskipun dibagi dua pun, hasilnya akan berupa pecahan. Contoh bilangan ganjil adalah 1, 3, 5, dan seterusnya dimana jika bilangan tersebut dibagi dua makan akan menjadi 0,5; 1,5; dan 2,5. Bilangan genap atau bisa didefinisikan sebagai bilangan yang bisa habis dibagi dua atau meskipun tidak habis dibagi dua pun, hasilnya setelah dibagi dua bukan berupa pecahan. Contohnya adalah 2, 4, dan 6. Jika angka-angka tersebut dibagi dua, maka akan menjadi 1, 2, dan 3. Selanjutnya jika dibagi dua lagi akan menjadi 0,5; 1; 1,5 atau bisa dibilang tidak habis dibagi dua.

Menurut saya, untuk memudahkan membedakan bilangan ganjil dan genap, yaitu dengan membagi angka tersebut satu kali saja dengan angka dua. Jika pada pembagian pertama hasilnya merupakan pecahan, maka bilangan tersebut adalah bilangan ganjil. Sebaliknya jika pada pembagian pertama hasilnya berupa bilangan bulat, maka bilangan tersebut merupakan bilangan genap.

Maka dari itu, bilangan-bilangan yang saya sebutkan di atas merupakan bilangan ganjil.

Pedoman penjumlahan bilangan ganjil dan genap:

Bilangan ganjil + bilangan ganjil = bilangan genapBilangan ganjil + bilangan genap = bilangan ganjilBilangan genap + bilangan ganjil = bilangan ganilBilangan genap + bilangan genap = bilangan genap

Pertama, kita perlu membaca input suatu angka. Gunakan syntax if-else untuk menentukan sebuah kondisi.

Kondisi 1 : Jika angka tersebut habis dibagi 2, maka program akan mencetak bilangan genap antara angka 2 dan 50 inklusif. Untuk mempermudah proses pencetakan, kita gunakan syntax for dengan nilai awal adalah 4, nilai maksimum adalah 50, dan perubahan nya adalah ditambah 2.Kondisi 2 : Jika angka tersebut menyisakan 1 jika dibagi 2, maka program akan mencetak bilangan ganjil antara angka 1 dan 100. Untuk mempermudah proses pencetakan, kita gunakan syntax for dengan nilai awal adalah 3, nilai maksimum adalah 99, dan perubahannya adalah ditambah 2.

Pada C++, Penulisan If-else adalah

if ( <kondisi> ) {

<perintah 1>;

<perintah 2 >;

...

} else {

<perintah a>;

<perintah b>;

...

}

<perintah 1>, <perintah 2>, ... dalam if akan dijalankan jika kondisi bernilai true, sebaliknya jika kondisi bernilai false, maka <perintah a>, <perintah b>, ... akan dijalankan

Pada C++, penulisan for adalah

for( <inisiasi awal>; <inisiasi akhir>; <perubahan> ){

<perintah 1>;

<perintah 2>;

...

}

selama nilai dari inisasi awal belum sama dengan inisiasi akhir <perintah 1>, <perintah 2>, ... akan terus dijalankan sampai nilai dari inisiasi akhir sama dengan nilai dari inisiasi awal.

2.Jika angka pada bilangan 100100100100100...diteruskan dengan pola yang sama tentukan

a. angka ke 100 → jawab angka 1

b. angka ke 1000 → jawab angka  1

c. angka ke 3000 → jawab angka 0

d. angka ke 2016 → jawab angka 0

e. Banyak angka 1 hingga ke 50 → jawab angka 1

f. banyak angka 0 hingga angka ke 100 → jawab angka 0

g. banyak angka 1 hingga ke 300 → jawab angka 1

h. banyak angka 0 sampai angka ke 1000 → jawab angka 0

Pembahasan

Pada bilangan 100100100100... pola selalu berulang setelah 3 angka

1-0-0

Angka pertama 1

Angka kedua 0

Angka ketiga 0

Setiap kelompok selalu terdiri dari 3 angka itu

a. Angka ke 100

Kita bagi 100 dengan 3

100 : 3 = 33 sisa 1

Karena sisa satu maka angka ke 100 sama dengan angka pertama pada tiap kelompok, yaitu angka 1

b. Angka ke 1000

Kita bagi 1000 dengan 3

1000 : 3 = 333 sisa 1

Karena sisa 1 maka angka ke 1000 sama dengan angka pertama pada tiap kelompok, yaitu angka 1

c. Angka ke 3000

Kita bagi angka 3000 dengan 3

3000/3 = 1000 tanpa sisa

Karena tanpa sisa, maka angka ke 1000 adalah angka terahir pada tiap kelompok, yaitu angka 0

d. Angka ke 2016

Kita bagi 2016 dengan 3

2016 : 3 = 672 tanpa sisa

Karena tanpa sisa, maka angka ke 2016 adalah angka terahir pada tiap kelompok, yaitu angka 0

e. Banyak angka 1 hingga ke 50

Kita bagi 50 dengan 3

50 : 3 = 16 sisa 2

Sehingga jumlah angka 1 sampai angka ke 50 ada 16 x 1 + 1 = 17 angka 1

f. Banyak angka 0 hingga angka ke 100

Kita bagi 100 dengan 3

100 : 3 = 33 sisa 1

Tiap kelompok ada 2 angka 0,

Sehingga jumlah angka 0 sampai angka ke 100 adalah 2 x 33 = 66 angka 0

g. Banyak angka 1 hingga ke 300

Kita bagi 300 dengan 3

300 : 3 = 100 tanpa sisa

Tiap kelompok ada 1 angka 1, sehingga banyak angka 1 sampai angka ke 300 ada 100 angka 1

h. Banyak angka 0 sampai angka ke 1000

Kita bagi 1000 dengan 3

1000 : 3 = 333 sisa 1

Tiap kelompok ada 2 angka 0, dan sisa 1 adalah angka pertama yaitu angka 1 bukan angka 0, sehingga banyak angka 0 sampai angka ke 1000 adalah 2 x 333 = 666 angka 0
