8. Dalam satu paket soal ujian terdiri dari paket A sebanyak 5 soal dan paket B sebanyak 6 soal.banyak cara menjawab 5 soal,jika 3 soal dari paket A dan 2 soal dari paket B adalah...cara


Menggunakan kombinasi berdasarkan variasi soal adalah sama apabila berbeda urutan.
Berlaku:
n = Soal A x Soal B
n = C(5,3) x C(6,2)
n = 5!/(2!3!) x 6!/(4!2!)
n = 5.4/2 x 6.5/2
n = 10 x 15
n = 150 cara

26. 5. Sebuah toko menjual 3 macam paket makanan sepertiberikut!PAKET 14 Donat dan 2 BakpaoRp26.000,00PAKET 23 Donat dan 4 BakpaoRp27.000,00PAKET 32 Donat dan 5 BakpaoRp....Jika Om Ali membeli paket 3, maka harus membayarsebesar....​


Sebuah toko menjual 3 jenis paket makanan. Paket 1 terdiri dari 4 donat dan 2 bakpao dengan harga Rp. 26.000. Paket 2 terdiri dari 3 Donat dan 4 bakpao dengan harga Rp. 27.000. Paket 3 terdiri dari 2 donat dan 5 bakpao. Maka harga dari paket 3 adalah Rp. 25.000.

Penyelesaian Soal :

Misalkan Harga donat = x

               Harga bakpao = y

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Paket 1 terdiri dari 4 donat dan 2 bakpao dengan harga Rp. 26.000" yaitu :

4x + 2y = 26.000   ...(Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Paket 2 terdiri dari 3 Donat dan 4 bakpao dengan harga Rp. 27.000" yaitu :

3x + 4y = 27.000   ...(Persamaan 2)

LANGKAH KETIGA (III)

Eliminasi persamaan 1 dan persamaan 2 sehingga didapat nilai x dengan menggunakan cara sebagai berikut :

4x + 2y = 26.000    ║×2║    8x + 4y = 52.000

3x + 4y = 27.000    ║×1 ║    3x + 4y = 27.000

____________________________________ -

                                                      5x = 25.000

                                                        x = [tex]\frac{25.000}{5}[/tex]

                                                        x = 5.000

LANGKAH KEEMPAT (IV)

Subtitusikan nilai x pada persamaan 1 untuk mendapatkan nilai y dengan cara :

4x + 2y = 26.000

4 (5.000) + 2y = 26.000

20.000 + 2y = 26.000

2y = 26.000 - 20.000

2y = 6.000

y = [tex]\frac{6.000}{2}[/tex]

y = 3.000

LANGKAH KELIMA (V)

Hitung harga dari paket 3 yang terdiri dari 2 donat dan 5 bakpao dengan cara :

2x + 5y = 2 (5.000) + 5 (3.000)

             = 10.000 + 15.000

             = 25.000

∴ Kesimpulan harga dari paket 3 yang terdiri dari 2 donat dan 5 bakpao Rp. 25.000.
