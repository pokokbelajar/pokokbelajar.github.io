2. Mengapa setelah dimasukkan ke dalam flashdisk video tidak bisa dibuka?


Jawaban:

Kemungkinan pertama: flashdisk tersebut ada virusnya. Sehingga file video mengalami kerusakan dan tidak bisa dibuka (diputar).

Kemungkinan kedua: file video tidak dikenali formatnya. Misalnya format videonya adalah FLV, sedangkan di komputer hanya ada program AVI player dan MKV player.

Kemungkinan ketiga: tidak ada program pemutar video sama sekali di komputer yang akan digunakan untuk membuka file video dalam flashdisk.

Penjelasan:

Solusi kemungkinan pertama adalah melakukan scan pada flashdisk dengan antivirus.

Solusi kemungkinan kedua yaitu mengkonversi format video dengan format yang dikenali (misalnya file video FLV dikonversi menjadi file video AVI).

Solusi kemungkinan ketiga ialah menginstal program pemutar video di komputer yang akan dipakai untuk membuka file video dalam flashdisk (contohnya program ALL Player).


3. Bagaimana cara mengembalikan file yang tidak bisa dibuka pada flashdisk?


Tahan tombol SHIFT di komputer/laptop dan masukkan flashdisk. Menekan dan menahan tombol SHIFT adalah untuk pengamanan saja, memastikan agar tidak ada program di flashdisk yang langsung dijalankan tanpa sepengetahuan pengguna (autorun.inf)Setelah itu hapus semua file dan folder yang merupakan shortcut (silahkan melihat kolom TYPE di windows Explorer). Pastikan yang dihapus adalah shortcut, dan jangan sampai salah melakukan klik dua kali (membuka shortcut) tersebut.Untuk mengembalikan semua data yang disembunyikan virus, bisa menggunakan program buatan saya Hidden File Tool atau langsung melalui Command Prompt. Silahkan memilih salah satu atau jika dengan Hidden File Tool ada yang tidak berhasil ditampilkan (misalnya nama file menggunakan huruf unicode: Arabic, China, Jepang dll) bisa ditambahkan menggunakan Command Prompt. Menggunakan Hidden File Tool, klik tombol Browse dan pilih drive flashdisk. Selanjutnya klik tombol -=SEARCH=-. Setelah hasilnya ditampilkan, pilih semua semuanya (klik file yang paling atas, tekan dan tahan SHIFT dan klik file yang terakhir). Selanjutnya klik tombol APPLY, sehingga semua file dan folder di flashdisk akan muncul semuaMenggunakan Command Prompt. Buka Command Prompt melalui Start Menu > Run dan tulis atau ketikkan:CMD lalu Enter. Atau melalui Start Menu > All Programs > Accessories > Command Prompt. Selanjutnya ketikkan huruf yang merupakan drive flashdisknya. Misalnya flashdisk ada di drive E, maka ketikkan E: (E diikuti tanda titik dua) dan tekan Enter. Setelah berhasil tampil E:\> di layar Command prompt, ketikkan perintah berikut: ATTRIB -S -H -R *.* /S /D kemudian tekan Enter.

sementara b = a + 1.500

untuk b = a+1.500 dimasukkan ke pers 1
6a + 4(a+1.500) = 41.000
6a+4a+6.000 = 41.000
10a + 6.000 = 41.000
10a = 41.000 - 6.000
10a = 35.000
a = 35.000 : 10
a = 3.500

untuk a = 3.500 ==> b = a+1.500 maka b = 3.500+1.500 = 5.000

jadi yang harus dibayar untuk 15a+10b
= 15(3.500) + 10(5.000)
= 52.500 + 50.000
= 102.500

Beberapa hal yang patut dicurigai ketika UFD (USB Flash Disk) yang kita miliki tidak terdeteksi saat dicolok ke PC, antara lain :

– Port USB Rusak

– Setting bermasalah ( Bios, Windows)

– Driver bermasalah

– Rusak secara Fisik

– dll

untuk mencoba mengatasinya maka dapat dilakukan beberapa hal berikut ini :

1. Coba colok USB kita pada Port USB yang lain /pindah-pindah, jika belum berhasil coba colokan ke PC lain, apabila berhasil maka USB kita tidak bermasalah dan kecurigaan 100% kepada PC.

2. Atau dilakukan sebaliknya, coba colokan USB lain di PC kita (jika ada)/ atau perangkat lain yang menggunakan usb port. Dengan asumsi, jika alat yang kita hubungkan terdeteksi, maka port USB tidak bermasalah.

3. Cek di settingan BIOS barangkali disable sengaja atau tidak, hal ini bisa saja terjadi perubahan setting dengan tidak kita sadari manakala baterai MotherBoard kita udah soak / lemah.

4. Kemungkinan yang lainnya adalah PC terjangkit Virus, maka harus dibersihkan dari virus dengan menggunakan anti virus

5.Jika mencurigai sebuah software yang baru diinstall sebagai penyebabnya maka coba uninstall dahulu software tersebut.

6. Coba gunakan software utiliti untuk find dan mount partisi ( mudah2an bisa).

23. sebuah toko komputer mempunyai persediaan flashdisk ukuran 2GB sebanyak 30buah ,flashdisk ukuran 4GB sebanyak 45 buah dan flashdisk ukuran 8GB sebanyak 27 buah,tentukan A.Perbandingan ketiga jenis flashdisk dalam bentuk paling sederhana B.Perbandingan ketiga ketiga jenis flashdisk jika telah terjual 5 flashdisk ukurn 2GB dan 7 flashdisk ukuran 8GB


Sebuah toko komputer mempunyai persediaan flashdisk ukuran 2 GB sebanyak 30 buah, flashdisk ukuran 4 GB sebanyak 45 buah dan flashdisk ukuran 8 GB sebanyak 27 buah. Perbandingan adalah bentuk sederhana dari beberapa nilai dari besaran yang sejenis untuk membandingkan nilai-nilai besaran tersebut.


Pembahasan

Diketahui

jumlah flashdisk ukuran 2 GB = 30 buah jumlah flashdisk ukuran 4 GB = 45 buah  jumlah flashdisk ukuran 8 GB = 27 buah

Ditanyakan

a. Perbandingan ketiga jenis flashdisk dalam bentuk paling sederhana  

b. Perbandingan ketiga ketiga jenis flashdisk jika telah terjual 5 flashdisk ukuran 2 GB dan 7 flashdisk ukuran 8 GB


Jawab

a. Perbandingan ketiga jenis flashdisk dalam bentuk paling sederhana

= flasdisk 2 GB : flasdisk 4 GB : flasdisk 8 GB

= 30 : 45 : 27

= (30 ÷ 3) : (45 ÷ 3) : (27 ÷ 3)

= 10 : 15 : 9


b. Perbandingan ketiga ketiga jenis flashdisk jika telah terjual 5 flashdisk ukuran 2 GB dan 7 flashdisk ukuran 8 GB

= flasdisk 2 GB : flasdisk 4 GB : flasdisk 8 GB

= (30 – 5) : 45 : (27 – 7)

= 25 : 45 : 20

= (25 ÷ 5) : (45 ÷ 5) : (20 ÷ 5)

= 5 : 9 : 4

30. Apa perbedaan boot flashdisk dengan multiboot flashdisk


boot flashdisk berarti menyediakan fasilitas booting pada flashdisk dan jumlahnya mungkin hanya 1 saja (misalnya mini windows xp)

multiboot flashdisk berarti menyediakan fasilitas boot pada flashdisk dengan jumlah lebih dari 1 (misalnya: bisa boot mini windows xp dan boot mini windows 7)
