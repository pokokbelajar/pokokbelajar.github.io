9. Sebutron 4 bagian dari shape tool?​


Sebutron:v


Rectangle Tool : Merupakan Tool untuk membuat objek shape dengan bentuk Persegi atau persegi panjang.
Rounded Rectangle Tool : Merupakan tool untuk membuat objek shape berbentu persegi dan persegi panjang, sama dengan Rectangle Tool, tetapi dengan sudut-sudut yang Melengkung atau Melingkar, yang Lengkungannya (radius) bisa kita sesuaikan dengan keinginan kita.
Ellipse Tool : Merupakan tool untuk membuat objek shape dengan bentuk Lingkaran atau pun Ellipse (lonjong).
Polygon : Merupakan tool untuk membuat objek shape yang bersudut lebih dari 4 atau Polygon. Dengan Polygon tool kita juga bisa membuat bentuk Bintang menggunakan Photoshop.

30. Jelaskan perbedaan Pick Tool, Shape tool, Freehand Tool, Zoom Tool dan Text Tool !


Perbedaan tersebut dapat kita lihat melalui fungsinya.

1. Pick Tool
Pick Tool ini memiliki ikon seperti cursor mouse pada setting defaultnya. Berfungsi untuk menyeleksi atau memilih objek.

2. Shape Tool
Shape Tool ini berfungsi untuk mengedit garis atau objek dengan manipulasi titik.

3. Freehand Tool
Freehand Tool berfungsi untuk menggambar garis kurva atau garis lengkung dan juga garis lurus.

4. Zoom Tool
Zoom Tool ini berfungsi untuk mengubah tingkat pembesaran window document Corel yang sedang kita buka. Kita bisa mengedit objek yang terlalu besar atau objek yang terlalu kecil sehingga nantinya tammpilan tersebut akan sesuai di mata kita.

5. Text Tool
Text Tool ini berfungsi untuk menambahkan tulisan pada canvas Corel kita. Kita juga bisa mengatur jenis font, besar font, dan juga memperbesar atau memperkecil tulisan yang ada.
