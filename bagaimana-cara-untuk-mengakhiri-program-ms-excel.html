2. Cara yang tepat unutk mengaktifkan ms- excel adalah…Start – program – ms office – ms wordProgram – ms excel – start – ms officeMs excel – start – program – ms officeStar – program – ms office – ms excel​


Jawaban: Start > Program >> MS Office >>> MS Word

Penjelasan: Pada windows kelas bawah (Windows 95, 98, ME, XP) Kita harus menekan Start, lalu arahkan ke Programs (Jangan di klik, hover saja), Arahkan lagi ke MS Office, lalu kamu bisa klik di MS Word.

Ada cara lain untuk membuka MS Word yang beragam. Untuk Windows Vista, 7, 8, 8.1, 10   Bisa klik start, Ketik Word, Nanti akan muncul program Word dan tinggal klik saja.
