21. Harga sebuah spidol Rp.2.000,00 kurangnya dari 3 kali harga sebuah pensil. Jika harga 2 pensil dan 1spidol adalah Rp 18.000,00. Jika harga spidol dimisalkan a dan pensil b, maka harga masing-masingpensil dan spidol tersebut adalah ...​


Diketahui :

SPIDOL = 3 Pensil - 2.000

2 PENSIL + SPIDOL = 18.000

Kita misalkan spidol = a dan penisl = b

A = 3B - 2000

2B + A = 18.000

Untuk menyelesaikan masalah ini mudahnya pakai eliminasi sehingga tersisa 1 Variabel, baru di substitusi ke satu variabel sisanya.

A + 2B = 18.000

A - 3B = -2.000

------------------------- -

5B = 20.000

B = 4.000

Ketemu harga pensil (b) yaitu 4.000 maka substitusikan 4.000 ini ke persamaan lain, disini saya pakai A + 2B = 18.000

A + 2(4000) = 18.000

A + 8.000 = 18.000

A = 18.000 - 8.000

A = 10.000

Jadi harga Spidol (A) = 10.000 dan harga Pensil (B) = 4.000

27. harga sebuah pensil lebih murah rp2.500 dari harga sebuah spidol harga 4 buah pensil dan 2 buah spidol adalah Rp50.000 Tentukan harga pensil dan spidol​


Materi : SPLDV

( pensil = x ) dan ( spidol = y )

\______________________/

[ x = y - Rp 2.500 ]

4x + 2y = Rp 50.000

\_____/ Substitusi ( x => y )

4( y - Rp 2.500 ) + 2y = Rp 50.000

4y - Rp 10.000 + 2y = Rp 50.000

6y = Rp 60.000

[ y = Rp 10.000 ]

x = y - Rp 2.500

x = Rp 10.000 - Rp 2.500

[ x = Rp 7.500 ]

\__________________________/

Result / Hasil :

( x = Rp 7.500 dan y = Rp 10.000 )
