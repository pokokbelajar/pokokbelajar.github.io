1. untuk menjalankan atau menghentikan daemon samba, smbd dan nmbd dapat diatur secara otomatis melalui skrip


/etc/rc.d/init.d/smbUntuk menjalankan atau menghentikan daemon samba, smbd dan nmbd dapat diatur secara otomatis dengan menambahkan skrip berikut di /etc/rc.local:

menjalankan:
/etc/init.d/samba start
/etc/init.d/smbd start
/etc/init.d/nmbd start

menghentikan:
/etc/init.d/samba stop
/etc/init.d/smbd stop
/etc/init.d/nmbd stop

Samba
Samba Server merupakan sebuah protokol yang dikembangkan di Sistem Operasi Linux untuk melayani permintaan pertukaran data antara mesin Ms. Windows dan Linux. Disamping untuk melayani file sharing antara Windows dan Linux, Samba juga merupakan salah satu protokol yang digunakan di Sistem Operasi Linux untuk melayani pemakaian data secara bersama-sama.

FTP
File Transfer Protocol (FTP) adalah suatu protokol yang berfungsi untuk tukar-menukar file dalam suatu network yang menggunakan protokol TCP. Dua hal yang penting dalam FTP adalah FTP Server dan FTP Client. FTP server adalah suatu server yang menjalankan software yang berfungsi untuk memberikan layanan tukar menukar file dimana server tersebut selalu siap memberikan layanan FTP apabila mendapat permintaan (request) dari FTP client.
