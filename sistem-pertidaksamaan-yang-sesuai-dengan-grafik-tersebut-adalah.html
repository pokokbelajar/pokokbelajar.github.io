Persamaan garis yg melalui titik (–2 , 0) dan titik (0 , 1) adalah :

[tex]\begin{array}{rcl}\frac{y-0}{1-0}&=&\frac{x-(-2)}{0-(-2)}\\~\\y&=&\frac{x+2}{2}\\~\\2y&=&x+2\\~\\x-2y&=&-2\end{array}[/tex]

Salah satu titik yg berada di dalam DHP (daerah himpunan penyelesaian) adalah titik (1 , 3), maka gunakan titik ini untuk mengetahui tanda ketidaksamaannya.

[tex]\begin{array}{rcl}x-2y&...&-2\\~\\x=1~;~y=3\to 1-2.(3)&...&-2\\~\\1-6&...&-2\\~\\-5&\boxed{\leqslant}&-2\end{array}[/tex]

Maka, pertidaksamaan yg memenuhi adalah : [tex]\boxed{x-2y\leqslant -2}[/tex]

[tex]\\[/tex]

[tex]ii~)[/tex]

Persamaan garis yg melalui titik (7 , 0) dan titik (0 , 6) adalah :

[tex]\begin{array}{rcl}\frac{y-0}{6-0}&=&\frac{x-7}{0-7}\\~\\\frac{y}{6}&=&\frac{x-7}{-7}\\~\\-7y&=&6.(x-7\\~\\-7y&=&6x-42\\~\\6x+7y&=&42\end{array}[/tex]

Gunakan titik (1 , 3) untuk mengetahui tanda ketidaksamaannya.

[tex]\begin{array}{rcl}6x+7y&...&42\\~\\x=1~;~y=3\to 6.(1)+7.(3)&...&42\\~\\6+21&...&42\\~\\27&\boxed{\leqslant}&42\end{array}[/tex]

Maka, pertidaksamaan yg memenuhi adalah : [tex]\boxed{6x+7y\leqslant 42}[/tex]

[tex]\\[/tex]

[tex]iii~)[/tex]

DHP berada di atas sumbu-X : [tex]\boxed{x\geqslant 0}[/tex]

[tex]\\[/tex]

Jadi, sistem pertidaksamaan linier yang memenuhi adalah : [tex]\boxed{\boxed{\begin{cases}x\geqslant 0\\~\\y\geqslant 0\\~\\x-2y\leqslant -2\\~\\6x+7y\leqslant 42\end{cases}}}[/tex]

11. perhatikan grafik berikut!! sistem pertidaksamaan linear yang sesuai dengan penyelesaian pada daerah diraster adalah...​


Jawaban:

Sistem pertidaksamaan dari daerah penyelesaian (daerah yang diarsir) pada grafik berikut adalah

a. 3x + y ≥ 21; x + 3y ≤ 21; y ≥ 0

b. 3x + y ≤ 3; x + y ≤ 2; x ≥ 0; y ≥ 0

Persamaan garis yang melalui titik (0, a) dan (b, 0) adalah

ax + by = ab

Jika koefisien y adalah bilangan positif, maka

ax + by ≥ ab, daerah himpunan penyelesaiannya diarsir ke atas

ax + by ≤ ab, daerah himpunan penyelesaiannya diarsir ke bawah

Pembahasan

a) Persamaan garis yang melalui titik (0, 21) dan (7, 0) adalah

21x + 7y = 21(7)

==> kedua ruas dibagi 7 <==

3x + y = 21

Karena daerah arsirnya di atas garis 3x + y = 21, maka pertidaksamaannya adalah

3x + y ≥ 21

Persamaan garis yang melalui titik (0, 7) dan (21, 0) adalah

7x + 21y = 7(21)

==> kedua ruas dibagi 7 <==

x + 3y = 21

Karena daerah arsirnya di bawah garis x + 3y = 21, maka pertidaksamaannya adalah

x + 3y ≤ 21

Daerah arsirnya di atas sumbu x maka pertidaksamaannya adalah

y ≥ 0

Jadi sistem pertidaksamaan dari grafik tersebut adalah

3x + y ≥ 21

x + 3y ≤ 21

y ≥ 0

b) Persamaan garis yang melalui titik (0, 3) dan (1, 0) adalah

3x + 1y = 3(1)

3x + y = 3

Karena daerah arsirnya di bawah garis 3x + y = 3, maka pertidaksamaannya adalah

3x + y ≤ 3

Persamaan garis yang melalui titik (0, 2) dan (2, 0) adalah

2x + 2y = 2(2)

==> kedua ruas dibagi 2 <==

x + y = 2

Karena daerah arsirnya di bawah garis x + y = 2, maka pertidaksamaannya adalah

x + y ≤ 2

Daerah arsirnya di atas sumbu x maka pertidaksamaannya adalah

y ≥ 0

Daerah arsirnya di sebelah kanan sumbu y maka pertidaksamaannya adalah

x ≥ 0

Jadi sistem pertidaksamaan dari grafik tersebut adalah

3x + y ≤ 3

x + y ≤ 2

x ≥ 0

y ≥ 0

y/5 = ( x + 6 )/8

8(y) = 5( x + 6 )

5x - 8y + 30 ≥ 0

x² + 2x - 3 ≤ 0

persamaan garis lurus A(-6,0) dan B(2,5)

x₁ = -6, y₁ = 0, x₂ = 2, y₂ = 5

y - y₁ / y₂ - y₁ = x - x₁ / x₂ - x₁

y - 0 / 5 - 0 = x - (-6) / 2 - (-6)

y / 5 = x + 6 / 2 + 6

y/5 = x + 6 / 8

5x - 8y = 30 ---> karena daerah yang diarsir di bawah garis maka tanda pertidaksamaan adalah ≤

sehingga pertidaksamaan yang berlaku 5x - 8y ≤ 30

menentukan fungsi kuadrat

fungsi memotong sumbu X di titik (-3,0), (1,0), dan melalui titik (0,-3)

x₁ = -3, x₂ = 1, x = 0, y = -3

y = a(x - x₁)(x - x₂)

-3 = a(0 - (-3))(0 - 1)

-3 = a(0 + 3)(- 1)

-3 = a(3)(-1)

-3 = -3a

 a = 1

Fungsi kuadrat:

y = a(x - x₁)(x - x₂)

y = 1(x - (-3)(x - 1)

y = (x + 3)(x - 1)

y = x² - x + 3x - 3

y = x² + 2x - 3

Untuk menentukan tanda pertidaksamaan pada fungsi kuadrat, ambil sembarang titik di daerah arsiran misal (-1,0)

x = -1, y = 0

Substitusikan ke fungsi kuadrat sehingga

0 = (-1)² + 2(-1) - 3

0 = 1 - 2 - 3

0 ... -4

---> beri tanda pertidaksamaan agar titik (-1,0) termasuk di daerah penyelesaian. Tanda yang tepat adalah 0 ≥ -4

sehingga pertidaksamaan untuk fungsi kuadrat adalah y ≥ x² + 2x - 3

Jadi, sistem pertidaksamaan yang sesuai grafik adalah 5x - 8y ≤ 30 dan y ≥ x² + 2x - 3.
