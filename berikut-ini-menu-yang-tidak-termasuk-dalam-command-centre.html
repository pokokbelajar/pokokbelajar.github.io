23. Fungsi menu command adalah.. Tolong ba tu jawab ya!!


Menu home Mc Office Power Point :


Slides : terdiri dari New Slide, Reset, Layout, Section.
1. New Slide digunakan untuk menambahkan slide baru ( Ctrl M )
2. Reset digunakan untuk mengembalikan ke ukuran, posisi dan format slide ke settingan default (semula)
3. Section digunakan untuk membagi slide menjadi beberapa Section / bagian. Gunanya akan sangat terasa jika membuat presentasi dengan jumlah slide yang banyak.

4. Layout digunakan untuk membuat sebuah layout pada slide dengan pilihan yang telah disediakan.
Font : Terdiri dari Italic, Bold, Strikethrough, Underline, Superscript, Subscript, text shadows, Font Size, Font, Decrease Font Size, Increase Font Size, Change Case, Character Spacing, dan Clear All Formating.


Bagian Paragraph juga lebih kurang sama dengan Word. Tapi ada tambahan yang perlu dijelaskan yaitu :
1. Text Direction : Fitur ini digunakan untuk mengatur text Effects. Akan tersedia beberapa pilihan seperti Text Outline, Text Fill, shadows, Outline Style, 3D Format, Reflection, Glow and Soft edges, 3D Rotation dan Text Box
2. Align Text : Fungsinya sama dengan Align test left dan right. Akan tetapi fitur ini arahnya Vertikal. Jadi terdiri dari Middle, Align text Top, dan bottom.
3. Convert to SmartArt : digunakan untuk mengubah text biasa kedalam bentuk SmartArt.

Drawing: terdiri dari Arrange Shape, Shape Fills, Quick Styles, Shape Outlines dan Shape effects
1. Arrange. Memiliki banyak fungsi yang bercabang, jadi alangkah baiknya dijelaskan pada pembahasan selanjutnya.
2. Shape. digunakan untuk memberi warna pada beground, namun bukan beground utama.
3. Shape Fills. Digunakan untuk mengubah warna pada shape
4. Quick Style. Digunakan untuk membuat style pada shape dengan cepat karena sudah tersedia ragam pilihan style yang bagus
5. Shape Outline. berfungsi untuk memilih tipe garis, warna garis dan ketebalan
6. Shape Effects. digunakan untuk memberi Efek pada Shape

Editing: terdiri dari Find, Select, Replace
1. Find berfungsi untuk menemukan kata atau kalimat yang berada didalam dokumen (Ctrl F)
2. Replace, berfungsi untuk mengubah text yang ada pada dokumen ( Ctrl H)
3. Select, cukup jelas.
Crop : berfungsi untuk mengcrop sebuah gambar. Hanya akan bisa difungsikan jika ada gambar yang di Insert.

29. Grup dari tab menu bars yang di dalamnya terdapat command button, disebut dengan


Grup dari tab menu bars yang di dalamnya terdapat command button, disebut dengan ribbon. Ribbon ini berisikan beberapa command dalam office.

Pembahasan

Ribbon merupakan berisi semua perintah yang Anda perlukan untuk melakukan tugas umum di Word. Pita memiliki beberapa tab, masing-masing dengan beberapa grup perintah. Grup Perintah: Setiap grup berisi serangkaian perintah yang berbeda. Secara umum, pita adalah kumpulan tombol perintah yang dapat digunakan untuk melakukan fungsi tertentu sesuai permintaan. Pita terdiri dari tab menu, grup perintah, dan tombol perintah. Setiap menu terdiri dari grup perintah, yang merupakan tombol perintah yang dikelompokkan. Tombol perintah mewakili berbagai kegunaan dan fungsi di Microsoft Excel, dan pita digunakan untuk mengumpulkan dan menampilkan perintah dan fungsi milik aplikasi Microsoft Office.
